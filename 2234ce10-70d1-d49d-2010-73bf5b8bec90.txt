　　当时间到达1点钟的时候，伴随着‘砰’的一声金属撞击声，这一场另白队无数粉丝忐忑的比赛终于是正式开始。
<br/>　　这一场比赛白队这边在蓝色方，而OG战队则是在紫色方，上来白队倒是并没有针对ban人，而是ban着当前版本比较热门的英雄。
<br/>　　而反观OG战队那边，在知道白队这边上场的是一个从来没有打过比赛的小女孩后，他们倒是也没有将自己的针对位放到中单，毕竟对方擅长什么英雄他们都还不知道。
<br/>　　而且打心里他们其实并不认为杨小艺到底有多强的，所以他们更多的则是将目光放到了香蕉和夜雪狂歌两人的身上，针对他们两人ban人。
<br/>　　“其实如果这一场比赛加入到教练战的话，那么说不定白队还有些希望取胜，但是现在这种情况，按照你说的小艺10点才休息，这种情况她的发挥肯定是要受到影响的，而且对方的中单可是大师组300名的选手呢，也是一个顶尖好手。”米粒开口说道。
<br/>　　“其实小艺的实力还是非常的强的，只不过她昨天确实没有好好休息，这一点倒是让人比较担心，如果正常状态下的小艺，她对阵OG的中单问题应该不是很大的。”林雅楠跟着开口说道。
<br/>　　对于林雅楠的话，大家倒是并不认为她说的是真的，只不过是以为她和荣耀战队几人的关系好，在那里帮着杨小艺甩锅呢。
<br/>　　场上双方轮流ban选，很快最终阵容便是确定了下来。
<br/>　　白队这边选择了：上单鳄鱼、打野蜘蛛、中路妖姬、下路则是女警和扇子妈这对强势组合。
<br/>　　而OG战队这边则是选择了：上单大树、打野狮子狗、中单沙漠皇帝、下路则是圣枪游侠和布隆这对组合。
<br/>　　单看阵容的话，无论是前期还是后期OG战队这边都是占据了很大的优势。
<br/>　　所以在双方阵容确定的一瞬间，尤其是在众多粉丝在查询到杨小艺的这个账号才打到洲区白金1的时候，大家对于这场白队的比赛更加的不看好了。
<br/>　　“哎，这一局估计gg了，这个中单才洲区白金1，这根本不可能和世界区大师组的打啊。”
<br/>　　“这上去存属是去找虐的，我现在只希望这一局白队输的不要太惨，然后下一场大魔王早点上场，那样的话咱们白队还能保持着不败的纪录。”
<br/>　　“也只能如此了，希望这一把输的不要太过难看，虽然大魔王有心培养新选手，但是他的步子还是有些急了，有些东西是要一步一步来的，起码让这个小艺打到世界区钻1的时候在让她上场啊。”
<br/>　　直播间里的众多粉丝纷纷开口议论着，偶尔有一两个说杨小艺胜率超级高的人，也是被那些弹幕瞬间给挤掉了。
<br/>　　大部分人都是匆匆看了一眼杨小艺的段位，随后便是直接退出了那个查询界面，并没有看她的那个具体战绩。
<br/>　　而那几个观察到杨小艺战绩的人，此刻则被那一页页绿色的战绩给震撼着，因为杨小艺打到白金1的时候，是以98的胜率上去的，中间只是偶尔输上那么一两场。
<br/>　　而杨小艺在和柳清月两人打到国服大师组的时候便是转战到了洲区，而且还重新换了一个白队的专属id，所以倒是并没有几个人知道她。
<br/>　　至于杨小艺她们这么早就转战洲区，更主要的原因还是她和柳清月两人想要快点进世界区。
<br/>　　按照官方给出的信息，只需要在国服打到大师组，那么进入洲区定位十连胜后，每一场排位的胜利就会有10胜点，所以两人自然是没有心思在等下去了。
<br/>　　而且国服那个王者第一的头衔对于她们的吸引力也确实不算太大，当然了最重要的一点要说的就是，杨小艺以这个胜率打到白金1，是在带着柳清月上分的情况下。
<br/>　　也就是说杨小艺硬生生的将一个原本只有钻4水准的柳清月，以98的胜率带到了洲区白金1。
<br/>　　当知道这个消息的时候，就是连叶帝还有秦霄他们都吃了一大惊，虽然他们也可以，但是杨小艺以如此年纪就打出这样强势的战绩，还是让几人很震撼的。
<br/>　　“这一场比赛白队这边的中单小艺居然选择了妖姬这个英雄，这个英雄之前可是经历过一次很大的削弱的。”看着杨小艺的选人，米粒开口说道。
<br/>　　“虽然如此，但是这个英雄还是可以玩的，虽然妖姬的沉默没了，但是她的爆发还在的，只要手速够快，她还是可以轻松的秒人的。”林雅楠笑着解说道，既然杨小艺已经拿出了这个英雄，她自然也就没有必要在帮着隐瞒这个英雄的厉害之处了。
<br/>　　事实上连林雅楠也没有想到杨小艺这么快就拿出这样一个算是大招的英雄。
<br/>　　毕竟之前已经有着无数的职业选手已经说了，妖姬这个英雄被削成这样，接下来已经没法玩了，甚至连这个英雄的使用率在版本更新之后，都是直线下降了很多。
<br/>　　只不过叶帝那边却是没有多想，叶帝知道这些选手虽然可能会被一时蒙蔽，但是用不了多久这些人肯定能够反应过来的，等他们再次使用妖姬，了解到这个英雄的爆发之后，肯定会继续拿出来玩的。
<br/>　　之前他们选取这个英雄对线的时候，主要还是因为大家打的都是高端局，在没有沉默之后，对方很容易利用位移技能逃跑。
<br/>　　但是当彻底熟练了一个没有沉默的妖姬之后，叶帝相信，这些人对于这个英雄肯定还是会非常的喜爱的。
<br/>　　片刻后，双方便是正式的进入到了游戏之中。
<br/>　　白队这边的粉丝心中都是万分的忐忑，而反观OG战队那边的粉丝，这个时候心里则都是有了一丝期待。
<br/>　　这一局比赛说不定他们真的能够赢下一局白队呢，虽然之后大魔王上场他们OG战队还是会输。
<br/>　　不过这些都没关系了，只要他们拿到这个击败过白队的荣誉就够了，国内这么多战队，到现在也就只有蓝队击败过一次白队而已。
<br/>　　白队除了输给蓝队那一场之外，在其他的比赛中甚至连一小局的对战都没有输过呢。
