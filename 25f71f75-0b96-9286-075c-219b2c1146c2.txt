　　“九魔城？”
<br/>　　“这葬天，是七大宗门的人？”
<br/>　　“这种实力，出身自七大宗门，也并非没有可能。”
<br/>　　听到葬天的话，众人低声议论起来。
<br/>　　葬天的来历，一直都是神秘无比，他并不属于北域之中的其他宗门，如今终于是公布了他的身份。
<br/>　　七大宗门，九魔城。
<br/>　　地魔门，便是曾经九魔城中的分支之一，来到北域壮大起来都能够成为北域的超级势力。
<br/>　　这九魔城的强大，不需要任何的言语。
<br/>　　葬天既然是九魔城的人，也怪不得有着如此的实力。
<br/>　　“小子，你曾经破解过我的杀心魔决，不过现在的我，可是跟以前截然不同了，现在我倒是要看看，在我的大心魔决面前，你还能不能跟我抗衡！”
<br/>　　葬天冷喝一声，猛然向前一步。
<br/>　　在他的身上，滔天魔意，展现出来。
<br/>　　这滔天魔意，直接便是化为了一道道剑锋，这每一道剑锋，都是蕴含这无数心魔。
<br/>　　虽然看似这剑锋是斩向了楚风眠的身躯，但是实际上这剑锋的目标却是楚风眠的心神，蕴含心魔的一剑斩向楚风眠，要将楚风眠的心神都给毁灭。
<br/>　　一旦是在楚风眠的心中，中下心魔，那楚风眠可就没有任何的反抗之力，只能够被这葬天随意击败。
<br/>　　这心魔之剑，那是一种极高的境界。
<br/>　　能够控制心魔，凝聚成剑，这葬天在大心魔决上的造诣，极其高明。
<br/>　　“凝聚心魔为剑，这葬天到底的什么人物？九魔城的大心魔决，他居然修行到了这一步？”
<br/>　　下方的一些强者，也是认出了葬天所用的功法，正是大心魔决。
<br/>　　这大心魔决，在九魔城中都属于极为高深的功法，名声在外，但是真正可以修炼到极高境界的却是少之又少。
<br/>　　而现在葬天，已经是达到了凝心魔为剑的地步，他的实力，绝对是达到了一种极为恐怖的地步。
<br/>　　这只要是其中一道剑锋，斩在楚风眠的身上。
<br/>　　楚风眠便是就会被心魔控制，倒时候楚风眠的性命，也就全部会掌控在这葬天的手中。
<br/>　　“大心魔决？你的大心魔决，对我没有任何意义，因为没有人可以影响到我！”
<br/>　　面对这心魔之剑，楚风眠冷笑一声，他的大手一挥。
<br/>　　猛然之间，这面前的心魔之剑，全然破碎，楚风眠任由其中的碎片，斩在他的身上，但是楚风眠的心中，却是没有任何一丝的波动。
<br/>　　这心魔，在楚风眠的心中刚刚被引动，便是就被楚风眠直接斩杀。
<br/>　　“什么？”
<br/>　　众人看向楚风眠，看见楚风眠遇到这大心魔决的一刻，居然是没有任何的变化，无数人的脸色都是震惊到了几点。
<br/>　　这大心魔决的强大，他们可都是清楚的，之前对付寒月漓的时候，这葬天催动大心魔决，顿时便是将寒月漓彻底击败。
<br/>　　而且若非是因为楚风眠从寒月漓的身上，布置下了禁制，可能寒月漓都是会直接被葬天轰杀。
<br/>　　可现在，同样是这大心魔决，楚风眠却是甚至丝毫不理会，没有收到任何的影响。
<br/>　　这意志，要是坚韧到什么程度，才足以做到这一点？
<br/>　　心魔，人人都有心魔。
<br/>　　这大心魔决最为可怕的一刻，便是引动武者心神中的心魔，几乎是防不胜防，因为在强大的武者心中，都会存在心魔。
<br/>　　这大心魔决，引动心魔，就算是拥有再强大的实力，在意志上不够坚定的话，便是都会受到影响。
<br/>　　远古时代，便是有这不少的强者，虽然实力强大，但是却心智不够坚韧，直接被心魔控制，一下子疯狂。
<br/>　　这样的事，不在少数。
<br/>　　所以这大心魔决，几乎足以称得上是整个北域之中最难以对付的功法之一，极为棘手。
<br/>　　可现在，居然是在楚风眠的面前，没有任何的意义，被楚风眠这样轻而易举的给抵挡下来。
<br/>　　“怎么可能！”
<br/>　　葬天的脸色，都是微微变化着。
<br/>　　他的出手，可没有任何的保留，一出手便是真正的全力，以大心魔决，凝聚心魔之剑，想要一击必杀，彻底斩杀楚风眠。
<br/>　　可现在，居然是没有丝毫影响到楚风眠，这也大大超出他的意料。
<br/>　　不过很快葬天的脸色，便是平静了，看向楚风眠冷冷的开口道。
<br/>　　“不错，看来你的实力，还没有让我失望，要是一下子就败在了我的手中，未免也有些太过无趣了。”
<br/>　　“你的心智在坚韧如何？在力量上，你也不可能是我的对手！”
<br/>　　擂台之上。
<br/>　　葬天一声怒喝。
<br/>　　他的身躯猛然一动，直接冲天而起，猛然一拳，已经是向着楚风眠轰击过去。
<br/>　　这葬天之前击败敌人，所用的，都是他的一身力量，他的力量，也是雄厚到了极点。
<br/>　　这一拳的力量，楚风眠感觉到，几乎是已经要超越了生死境的范畴。
<br/>　　这一拳的速度，十分之快，几乎是瞬间，这一拳已经是向着楚风眠的头顶，轰击过去。
<br/>　　单单是凭借身体的力量，这葬天，便是已经站在了这一届武斗大会的顶端，更何况他还拥有这极为难以对付的大心魔决。
<br/>　　寒月漓败在他的手中，倒是不冤。
<br/>　　这力量，在无数众人看来，都甚至有些惊世骇俗了，一个神力境武者，居然就将力量达到了这一步。
<br/>　　那若是葬天未来在度过生死大劫，实力暴涨，那只怕会变得更加可怕。
<br/>　　不过在楚风眠的眼中，这力量，却是算不得什么。
<br/>　　一直以来，单纯凭借力量，楚风眠都是同境界无敌的存在，甚至是足以碾压更加强大境界的武者。
<br/>　　在力量交锋上，楚风眠还没有怕过谁，他可是有着战龙血脉，远古龙王身。
<br/>　　如同一条人形战龙一般。
<br/>　　面对这葬天的一拳轰击过来，楚风眠站在原地，不闪不避，他的右手猛然凝聚灵力，化为一拳，猛然向着空中轰击了过去。
<br/>　　硬碰硬。
<br/>　　面对这样的力量，楚风眠也是丝毫没退，反而是直接选择上去，跟着葬天硬碰硬。
<br/>　　众人而已知晓，楚风眠的力量也是强大到了惊人的地步，但是这个时候还选择硬碰硬，还是让许多人没有想到的。
