　　第632章
<br/>　　“侯爷说得在理，可宋伯康诬陷我们勾结戎族，我们就算死在此地，也一样要背负千古骂名。”
<br/>　　“我们做的事我们自己知道，我是帝国的兵，如今也只能求个问心无愧。如果能破掉此局，我们就去长安，向朝廷讨一个公道。”
<br/>　　百里胜其实无所谓对朝廷的忠心，如果有可能的话，他也想趁着这乱世谋求自己的利益，争一争天下。就算没机会触摸到那至高无上的帝位，也能求个裂土封王。实在是这些年过去，李氏皇族将人心丢得一干二净。
<br/>　　不过，不忠于朝廷未必代表不忠于帝国，让他去投靠戎族，帮着戎族打帝国他是不屑为之的。
<br/>　　通过一席话，百里胜也确定了此战的目的。求援是不可能的，附近也不可能有人会支援他们粮草。百里胜的目的只有一个，突破重围一路打到长安，向朝廷讨一个公道。
<br/>　　庙堂之上的勾心斗角，孙鸣之这些人不明白，百里胜是明白的。自己是将军府的人，变相也相当于是李乐天的人。但宋伯康敢向自己动手，定然是已做好万全准备。要么是相府和将军府已达成某种交易，也或许是帝国皇帝已经无法控制相府的势力扩张。
<br/>　　无论如何，整个帝国已经无人在乎自己的生死，棋盘上的棋子被下棋者抛弃后，只能靠自己拼一条活路。
<br/>　　下定决心后，百里胜打开城门，带着三千兵丁出城，在宋伯康的大营前叫阵。
<br/>　　宋伯康得到消息后，来到辕门前，看到百里胜骑着一匹乌骓马，手持两杆钢鞭，在那里耀武扬威的骂阵。
<br/>　　“听说百里侯爷也有些武艺在身，只不过在长安一直没机会见到他出手，也不知武艺如何。”宋伯康道。
<br/>　　“王上，他在营前叫阵，咱们打或不打？”戴白问道，他在凉州城负责军事。
<br/>　　“差人出去，会一会百里侯爷的武艺。”如今的宋伯康胜券在握，说话也自信许多。
<br/>　　一匹快马冲出辕门，奔向百里胜，快行到他面前时勒马停住。
<br/>　　“本侯爷手下不死无名之鬼，来将报上名来？”百里胜喝道。
<br/>　　“本将军乃是武威城先锋官吕亮。”马上将官扛一柄大刀，口中喝道：“逆贼，见到本将官还不快快纳命来。”
<br/>　　百里胜仰天哈哈大笑：“逆贼，逆贼……本侯爷倒是成了逆贼，好好好，你来让侯爷我斩了你的人头。”
<br/>　　吕亮拍马攻来，一柄大刀抡开斩的是百里胜脑袋。两匹马交错而过，兵器在空中相撞，却是谁也没占到便宜。
<br/>　　拨转马头再战，二人纠缠在一起，忽然百里胜卖了个破绽，引得吕亮的招式用老，正是旧力未尽，新力未生之时，两柄钢鞭照着吕亮脑门砸下来。
<br/>　　只听咔嚓一声，吕亮的天灵盖被砸得稀巴烂，一头跌落马下，流出红白之物。
<br/>　　狼山关一方齐声叫好，挥舞着兵器口中发出阵阵怪叫，一时间士气大振。
<br/>　　而戴白却是倒抽一口凉气，他也曾听说过这位来自长安的侯爷，花天酒地，眠花宿柳，却是位风流人物。但不曾想到，竟然有如此硬的身手，却是不得不对他刮目相看。
<br/>　　宋伯康倒是对此不以为然，战场上的胜负并不是由大将之间的交锋来决定，如果真是如此的话，那么战争就来得太简单了。
<br/>　　现在百里胜已经陷入绝地，所以他才不得不拼命，然而宋伯康占有全部优势，所以能够从容不迫。
<br/>　　“百里侯爷倒是好武艺。”宋伯康道：“谁有把握去胜他？”
<br/>　　“金城杨武请令，末将愿去会他。”
<br/>　　杨武来自金城，也有不错的身手，宋伯康挥挥手算是许他出阵。
<br/>　　杨武拍马出阵与百里胜战在一起，双方倒是棋逢对手将遇良才，最后还是百里胜更凶狠些，一鞭击落杨武，他被手下抢回营中。
<br/>　　百里胜连败两将，己方士气大振，宋伯康一方难免垂头丧气。此刻，就连宋伯康也皱起眉头，这百里胜竟强得出乎想象。如果无人能够胜得了他，是对士气的巨大打击。
<br/>　　“戴白，还有谁能出阵，去击败百里胜？”
<br/>　　戴白面有难色，这暴露了宋伯康一个软肋。他手下兵强马壮，武器精良，粮草充裕。凉州骑兵在整个帝国，战斗力也算是强的。然而，宋伯康手中并没有拿得出手的大将。面对今日敌将讨战的局面，倒是不好应对。
<br/>　　戴白沉思片刻，道：“王上，属下保举一人，定能胜过百里胜？”
<br/>　　“谁？”
<br/>　　“赵将军。”
<br/>　　宋伯康恍然大悟，立刻道：“快去请赵将军。”
<br/>　　赵子龙就在左近，能成为战场上的不败将军，他对战争还是蛮感兴趣的。被叫到辕门前，宋伯康道：“赵将军，你可看到那贼人耀武扬威，在阵前叫骂，气焰十分嚣张，还请赵将军出战，灭一灭他的威风。”
<br/>　　赵子龙远远望了一眼，表情有些尴尬。他若出阵，自信胜过百里胜不难，可他本来就是过来摸鱼的，不愿为宋伯康打仗。而假装败给百里胜，他不败将军的记录，可并不希望为了宋伯康而打破。
<br/>　　“那个……抱歉，我病了，怕是无法出阵。”赵子龙本非圆滑之人，扯句谎也是死板得很，没有任何说服力。
<br/>　　“病了，什么病？”宋伯康一楞。
<br/>　　“风寒。”
<br/>　　看着赵子龙红润的脸庞，实在不像生病的样子。而宋伯康此刻也没有与赵子龙撕破脸的决心，如果撕破脸，让程大雷和百里胜联起手来，则凉州危矣。
<br/>　　如此只能强压下火气，道：“赵将军病了就好好休息，养好身体要紧。”
<br/>　　回过头来，却是满脸杀气，口中喝道：“谁可出战！”
<br/>　　“王上，末将愿往。”
<br/>　　此人乃是宋伯康手下的大将，名唤徐童，使一柄大锤，双臂有千钧之力。
<br/>　　宋伯康一声令下，徐童跨着枣红马，出阵与百里胜战在一起。
<br/>　　赵子龙装病不出战，但此刻并未离去，同样在辕门观战。远远看着打斗的两人，他嘴角抿起，发出一道笑声。
<br/>　　宋伯康一激灵，你这声笑是几个意思。
<br/>(本章完)
