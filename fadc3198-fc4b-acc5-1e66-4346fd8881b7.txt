        &lt;h3&gt;第三卷第二百五十八章  陈宁合作&lt;/h3&gt;
<br/>
<br/>        宁雅还是在凌辰那甜蜜的情话里沉沦了，她遵从自己的内心，不想再作无畏的挣扎和试探，就让自己大大方方无所畏惧的爱一次又何妨？
<br/>
<br/>        她对凌辰说：“我要主宰我自己的命运，所以，你不用担心我，你应该为我感到庆幸。”
<br/>
<br/>        凌辰心里大喜，他以为宁雅要和他在一起：“你想好了？你确定？”
<br/>
<br/>        “是啊，我确定，我已经重新开始了，等我这次回去，陈子昂的耗材推向市场之后，我就辞职去干我自己的事业，不过这个我家里人不知道，我目前不想让他们知道，我怕他们会阻扰我，毕竟，我还是会动用一部分家里给我的资源。”
<br/>
<br/>        “啊，你想干什么事情？我怎么不知道？”
<br/>
<br/>        “嗯，美容，先在北京开一家，后期如果可以的话，就开连锁的，我要将我自己的美容医院开到各个地方，哈，也可以让你美美的。”宁雅满怀希望，心情一下子就好起来了。
<br/>
<br/>        “肯定是陈子昂给你出的主意。”
<br/>
<br/>        “也没有，只是我接受了她的建议，她给了我自信，我希望能够像她一样自强自立。”
<br/>
<br/>        “女人太好强了也不幸福，总是........”
<br/>
<br/>        “咦，你这是什么意思？嫌弃我？不支持我？”
<br/>
<br/>        “不不不，我家雅儿干什么都是香的，我家雅儿是世界上最棒的，雅儿，雅儿，我可以入股吗？雅儿求你让我入一股吧，求你给我一个赚钱的机会吧。”
<br/>
<br/>        “渣男本渣，渣男属性暴露无遗。”宁雅笑着说。
<br/>
<br/>        “不，此生此世，我只渣宁雅一人。雅儿，我说的是真的，让我入一股吧。”凌辰认真的看着宁雅。
<br/>
<br/>        “回头再说，该说晚安了，我明天要回宁家公司谈判。”宁雅挥手和凌辰道别。
<br/>
<br/>        .......
<br/>
<br/>        宁家公司
<br/>
<br/>        宁雅带着律师走进了宁家会议室，这里是她曾经呆过的地方，她在这里主持了很多次会议，做了很多次抉择，如今，她竟然是以另外一个公司谈判者的身份来到这里。
<br/>
<br/>        当宁雅走进会议室的那一瞬间，办公室里寂静无声。
<br/>
<br/>        宁玥坐在首位，旁边坐着几个部门领导，而这几个人看到宁雅的时候，脸上是吃惊的表情。
<br/>
<br/>        宁雅看着大家吃惊的样子，冲大家点点头，这里面有几个都是她曾经的手下，律师替她拉开凳子，宁雅轻轻的坐下。
<br/>
<br/>        宁玥看着宁雅自己坐下来，大家都不说一句话，心里感觉有一丝丝的快慰，可是想到昨晚凌辰和宁雅说的话，心里越看宁雅越不顺眼。
<br/>
<br/>        “大家认识某公司的代表吗？宁氏的嫡长女，现在竟然代表别人的公司回来和自己的家族谈判。真是千古一谈啊。”宁玥开口了。
<br/>
<br/>        “我现在是某公司的代表，而不是宁氏的嫡长女，宁总请注意说话方式。”宁雅冷冷的开口，既然坐在了这个谈判桌上，那真的是要开诚布公的好好谈谈了。
<br/>
<br/>        “你以为你傍上高枝就是凤凰？可惜，你的那个公司和宁家还差的很远。”宁玥也冷笑着开口。
<br/>
<br/>        “差不差得远不重要，重要的是能不能合作。重要的是宁家能不能得到我公司给你们公司的进账。”宁雅今天的角色就是代表公司来谈判的人，能不能合作取决双方面的态度，她希望能谈下来这次合作，无论是从陈子昂的角度来说还是从宁家的角度来说，对双方都是百利而无一害的。
<br/>
<br/>        只是不知道宁玥今天会不会想要谈成功？
<br/>
<br/>        “你......”宁玥没有想到宁雅毫不犹豫的就将身份摆好了，完全就是一个对立的状态。
<br/>
<br/>        “宁总，这次合作我希望我们都在一个公平公正的状态下进行，我更希望这次合作对我们双方都能起到促进作用，我们是抱着诚心来谈合作的，希望宁总能抛去杂念，认真的对待这件事情。”宁雅不再给宁玥斗嘴的机会，她没有必要把精力浪费在和自己的妹妹争宠上，况且从小到大她都是让着她的。
<br/>
<br/>        但是这一次，她不会再让着了，他们会在谈判桌上见分晓。
<br/>
<br/>        宁雅和律师俩个人，律师只是见证合同的签订以及落地盖章的程序，可以说宁雅今天是一对六，而且还是面对着曾经的部下和自己的妹妹。
<br/>
<br/>        虽然说心里已经很坚定了，但是人的心毕竟还是柔软的，在价格上宁雅差一点就退让了。
<br/>
<br/>        律师全程都不说话，在没有触及到法律上面的事务他是不会说话的，陈子昂让宁雅前来是绝对相信宁雅的，即使宁雅有所偏颇他也不会吭气的。
<br/>
<br/>        宁雅看了一眼律师，他正低着头，专心的看着他面前的商务条款，似乎对她的谈判一点也不在意。
<br/>
<br/>        宁雅叹了口气：“对于价格，我不希望大家有过多的纠结，我不求你们给我太多特殊，但是也不要给我太高，既然是合作，当然要有诚意，如果没有诚意。不谈也罢。”宁雅站起来准备起身离开。
<br/>
<br/>        宁玥见宁雅要走，心里有点慌，爸爸可是说一定要签下这个合作项目。
<br/>
<br/>        她看了一眼宁雅：“价格方面我需要请示一下，你能不能等一会。”
<br/>
<br/>        宁雅微微一笑抬手看了一下腕表：“我们的时间都是有价值的，十分钟之后我再过来，王律师，我们出去一下。”
<br/>
<br/>        “好！”王律师将手中的文件也拿了起来，随着宁雅一起走出了会议室。
<br/>
<br/>        “如果他们的价格不降低，你是不是就不签这份合作书了？”
<br/>
<br/>        “商务问题你要是觉得可以我们就签，反正宁总裁是不会放弃这个项目的，如果让大家知道他的女儿回来和他谈判都谈崩了，那他以后的形象会崩塌的。价格问题我也是知道的，在大家都有利的情况下，我相信他还是会答应的。”宁雅淡定的说。
<br/>
<br/>        知己知彼，宁家她怎么能不知道情况？陈子昂啊陈子昂！
<br/>
<br/>        “商务问题我会注意的，我只是担心你。”
<br/>
<br/>        “谢谢你，王律师。”宁雅心里划过一丝温暖，衷心的说着感谢。
<br/>
<br/>        不到十分钟，宁玥安排人到休息室找他们。
<br/>
<br/>        看着王律师在商讨商务条款，宁雅拿起手机给陈子昂发了一个信息：“搞定！”
<br/>
<br/>        是啊，宁雅虽然站在了宁氏立场对立面，但是他们完成了一次合作，他们现在又站在了同一条战线上。
<br/>
<br/>        合作，其实是在同等的条件下，相互扶持，相互鼓励，共同前进。宁雅其实是心里还是很在乎宁家的。
