　　韦斯利驾驶着租来的汽车，朝着酒店返回，坐在后排的布谷已经通过手机预订了三张返回法兰西国的机票。
<br/>　　韦斯利、布谷、斯黛拉三个人抵达酒店的时候，黑桃J和黑桃10还没有从酒店中出来，坐在黑色宝马轿车中负责接引的黑桃9看到三个人走进酒店，立刻拨通了黑桃10的电话，说道：“他们返回酒店了，你们不要离开，我立刻上来增援你们，我们必须要把韦斯利解决掉！”
<br/>　　黑桃9拉开储物格，从里面拿出一把使用了加长弹匣的格洛克18全自动手枪，退出弹匣，看了眼弹匣中的子弹后，将两个备用弹匣塞进衣服口袋中，把手中格洛克18子弹上膛后，把枪藏在衣服中，跟在韦斯利、布谷、斯黛拉三个人走进了酒店。
<br/>　　看着韦斯利、布谷、斯黛拉三个人走进电梯后，黑桃9才按下旁边那部电梯，刷卡按下楼层数字键。
<br/>　　黑桃J和黑桃10已经安装好了炸弹，从韦斯利的房间中撤了出来，并且藏到了他们斜对面的空房间中，通过房门上的猫眼观察着外面的情况，两个人手里各拿着一把MP5K微型冲锋枪，随时准备冲出房间进行补枪。
<br/>　　韦斯利、布谷、斯黛拉三个人走出电梯，各自返回自己的房间去收拾行李，然后准备前往机场。只要能顺利搭乘飞机返回到法兰西国，韦斯利、布谷也是有了足够了底气，“龙牙”的人随时可以过来接应，就算那些试图暗杀韦斯利的人有三头六臂，也不会有任何的胜算。
<br/>　　“五分钟后，底楼服务台碰头。”韦斯利对布谷说道。
<br/>　　布谷点了点头，拿出自己的房卡朝着自己的房间走去；韦斯利这边，斯黛拉已经是用房卡刷开了房间门，并且按下了房门把手，把门推开。
<br/>　　就在斯黛拉推开门的时候，布谷听到了从房门背后传来的急促的“嘀嘀”声响。
<br/>　　布谷脸色顿时剧变，对着韦斯利大喊道：“小心！”
<br/>　　轰！
<br/>　　布谷的话音刚落，斯黛拉面前的房门背后就发生了爆炸，一团火球从房间里蹿了出来，直接把斯黛拉和韦斯利两个人给炸飞出去，就连布谷都是被气浪掀翻在地，在倒地的瞬间，布谷已经是取下了他挂在牛仔裤口袋上的战术折刀，按下了锁定钮，弹出了刀刃。
<br/>　　炸弹刚被引爆，黑桃10和黑桃J所躲藏的房间门就被打开，黑桃10和黑桃J拿着MP5K微型冲锋枪从房间里冲了出来。
<br/>　　布谷看到突然冲到走廊中的两个握着MP5K冲锋枪的男子，甩手就把手中的战术折刀甩向了站在靠前位置的黑桃10。
<br/>　　噗嗤！
<br/>　　黑桃10还没来得及扣下扳机，就被布谷甩出的战术折刀刺穿了喉咙，黑桃10瞪大着眼睛，捂着自己不断冒出鲜血的脖子仰头倒在地上。
<br/>　　旁边位置的黑桃J看到这突如其来的一幕，愣了愣，他没想到会发生这样的情况，他跟黑桃10两个人只是刚露头，黑桃10就被对方甩出的一把战术折刀给秒杀了！
<br/>　　黑桃J发愣的时间并不长，大概也就1秒钟的左右。
<br/>　　这“滴答”一声的一秒钟对于一个普通人来说可能做不了什么事情，但是对于有些人来说就能完成很多事情。对于一个职业杀手，他可以完成一次必杀；对于一个职业电竞选手，他可以在团战中因为这一秒钟的控制时间，完成对对方玩家的击杀，奠定一场比赛的胜局；对于布谷来说，这一秒钟的时间，让有了继续反杀的机会！
<br/>　　布谷在走廊上狠狠蹬踢，身体弹射而出，直奔黑桃J而去；等到黑桃J举起他手中的MP5K微型冲锋枪，对着布谷扣动扳机，扫出子弹的时候，布谷已经抢在他之前完成了战术规避，避开了子弹的射击轨迹。
<br/>　　倒在地上的黑桃10虽然被布谷用飞刀刺穿了喉咙，但是在这么短的时间内他还没有彻底断气，布谷翻滚到黑桃10身边，反手握着插在他脖子上的战术折刀，将战术折刀从他的脖子中拔了出来，在将战术折刀拔出的时候，又顺势在黑桃的颈部大动脉上划了一刀，这一刀彻底收割了黑桃10的生命。
<br/>　　等到黑桃J反应过来的时候，布谷已经是到了他的身前。
<br/>　　唰！唰！
<br/>　　布谷对着黑桃J划出两刀，战术折刀的刀锋带起两道刀芒，这两刀攻向黑桃J的下盘。
<br/>　　布谷因为打过黑拳的关系，本身就有一些格斗基础，加上夜鹰对他长达数年的训练，布谷的格斗技巧早已不是当年那个拳台上黑拳少年的水平，布谷出刀即快又准，这两刀直接切断了黑桃J的跟腱。
<br/>　　跟腱断裂，黑桃J身体一个踉跄立刻摔倒在地上，他抬起手中的MP5K冲锋枪想要杀死布谷；布谷根本不会给他这个机会，手中的战术折刀直接一刀插在了他的太阳穴上，将黑桃J解决掉。
<br/>　　从布谷甩出战术折刀反杀黑桃10开始，到他一刀扎在黑桃J的太阳穴上，解决掉黑桃J前后的时间间隔不到十秒钟的时间，就是这短短几秒钟的时间，布谷独自一人解决掉了两个杀手，动作干净利落，没有丝毫的拖泥带水。
<br/>　　黑桃9这个时候也从电梯中冲了出来，他藏在衣服中的格洛克18全自动手枪也握在了他的右手中；可是当他走到走廊中的时候，他却看到了令他震惊的一幕，韦斯利和斯黛拉的确是已经倒在了地上，但黑桃10和黑桃J也是倒在了血泊中，黑桃9立刻意识到了不妙，对着布谷的方向扫出一串子弹，转身就朝着电梯的方向跑去。
<br/>　　布谷用黑桃J的尸体挡住了黑桃9的子弹，抓起黑桃J和黑桃10掉落在地上的MP5K冲锋枪，双手持枪朝着黑桃9冲去。
<br/>　　哒哒哒哒哒……
<br/>　　哒哒哒哒哒……
<br/>　　两把MP5K冲锋枪喷出两道火舌，子弹朝着黑桃9扫去。黑桃9虽然闪躲的很快，但是他的手臂上还是被子弹擦出了一道口子，鲜血立刻就涌了出来。
