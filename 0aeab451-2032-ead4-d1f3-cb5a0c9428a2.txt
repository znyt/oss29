　　无数乌云，笼罩在楚风眠的上方。
<br/>　　楚风眠猛然抬起头，他自然清楚，这乌云，正是冲着他来的。
<br/>　　生死大劫。
<br/>　　一直以来楚风眠足以压制住生死大劫，便是他的实力还缺少这一点，在这四十九道无上剑气，全部都被楚风眠吞噬炼化的一刻，终于是将楚风眠的实力，达到了最为极致的地步。
<br/>　　现在的楚风眠，便是已经达到了神力境最为极限的地步，再也无法压制住。
<br/>　　就当楚风眠将这四十九道无上剑气，全部炼化的一刻，这生死大劫，便是直接降临了。
<br/>　　这乌云，简直是要将整个沙漠，都给笼罩，一眼望去，甚至似乎都是有些望不到尽头。
<br/>　　其中密密麻麻的劫雷，根本便是数不过来，但是一眼望去，要比起楚风眠当初在御风境突破神力境的时刻，出现的劫雷，还要多得多，多出无数倍来。
<br/>　　曾经楚风眠在御风境突破到神力境的时候，都是将生死大劫，引了出来，这简直就是一件骇人听闻的事情了。
<br/>　　只有在九域中那些上天都为之嫉妒的天才，才会在突破神力境的时候，都引来一次雷劫，但是最多引来的，都是一些平凡的雷劫。
<br/>　　像是楚风眠这样，突破神力境引来的雷劫，居然是万雷帝劫的，简直是在九域的历史之中，都从来没有出现过。
<br/>　　当时一些名震九域的大人物，超级天才，在突破神力境的时候，最多引来的都是千雷皇劫。
<br/>　　就连楚风眠，其实也是他重生为人，有着两世的积累，才足以做到的这一点。
<br/>　　在当初突破神力境的时候，楚风眠便是就将这雷劫之中最为恐怖的万雷帝劫给引了过来，如今楚风眠现在真正是达到了神力境巅峰的地步，要引来生死大劫。
<br/>　　这到底是会出现什么样的雷劫，楚风眠都不敢想象。
<br/>　　但是眼前，这笼罩这整个沙漠上空的劫云，楚风眠一眼望过去，便是足以感觉得到，这劫云的面积，比起当初万雷帝劫的时刻，还要庞大无数倍。
<br/>　　简直是从九域的历史之中，都从来没有出现过的，如此恐怖的生死大劫，如今正是降临在了楚风眠的头上。
<br/>　　“九域第一人，如今我就要度过这九域历史之中都从来没有出现过的雷劫，成就真正的九域第一人！”
<br/>　　楚风眠抬起头，他的脸上没有任何的慌张，反而是极为期待。
<br/>　　这样的天劫，正是对他实力最为完美的认可。
<br/>　　越是强大的雷劫，度过之后说得到的好处，也就越多。
<br/>　　面对这雷劫的袭来，楚风眠甚至都变得有些迫不及待起来，他的身形一动，直接便是向着空中，冲了上去。
<br/>　　他不要等待这雷劫，而是直接冲着雷劫，猛然冲了过去。
<br/>　　“轰隆！轰隆！”
<br/>　　无数劫云之中，都是一阵阵雷声，突然响起，这劫云，似乎都没有想到，楚风眠居然是丝毫不惧怕，反而是直接冲了上来。
<br/>　　一道道劫雷，便是从这劫云之中，直接便是落了下来，这些劫雷，一眼望去，都是足足是有着上万道之多。
<br/>　　这一次楚风眠的生死大劫，单单是一个开头，所落下的，便是整整也给万雷帝劫。
<br/>　　要是让其他人观看这一幕，简直是都要疯狂，在九域之中，无数天才拼劲一切，想要引来的万雷帝劫，如今在楚风眠的生死大劫之中，居然都只能算是一个开头，前奏。
<br/>　　一道道雷劫，狰狞这，融为一体，化身成为了一头巨大无比的猛虎，猛然之间向着楚风眠扑了过来。
<br/>　　“劫雷，只能是化为我的力量！”
<br/>　　楚风眠看着天空之上，他猛然开口，狂吼一声，整个人便是变化成为了真正远古战龙的样子，冲天而上。
<br/>　　他的大口长着，一口直接冲这那猛虎，吞了过去。
<br/>　　在这一口之下，居然是将这万道劫雷，凝聚出来的猛虎，都是一口吞了下去。
<br/>　　就在楚风眠的身躯之中，无数劫雷，在疯狂的翻腾这，想要将楚风眠的身躯撕的粉碎。
<br/>　　可是这一道道劫雷，却是无法伤及到楚风眠身躯中的一点一滴，反倒是被全部镇压，一道道恐怖的劫雷，反而是化为了给楚风眠炼体的雷霆。
<br/>　　令如今楚风眠化身的远古战龙看起来，整个身躯，都是变得坚韧了许多。
<br/>　　人人都惧怕的生死大劫，但是在现在楚风眠的面前，却是被他当做了一种炼体的手段。
<br/>　　“太强横了！这到底是一个什么人物！”
<br/>　　从楚风眠的身躯之中，一直沉睡的星玄，如今都是突然苏醒过来，正是看着眼前这不可置信的一幕。
<br/>　　星玄在九域，已经是不知道纵横过多久，他听说过无数天才，绝世妖孽，甚至是那些好久万年难遇的超级天才，度过生死大劫的样子。
<br/>　　但是没有任何一人，可以跟楚风眠的样子来媲美。
<br/>　　这哪里是在渡劫，这劫雷，反倒是被楚风眠利用，成为了楚风眠炼体的手段，来淬炼这他的身躯。
<br/>　　“九域第一人，这样下去，真的可能是九域第一人了，九域中，居然又出现了这样一位人物。”
<br/>　　星玄看着楚风眠，似乎是在自言自语一声，不知道想到了些什么。
<br/>　　曾经的楚风眠，在很弱小的时候，便是立下誓言，要踏平七大宗门，成为九域第一人，当时的星玄，还是觉得，那是楚风眠的天真，狂妄。
<br/>　　但是如今这引来的雷劫，真的足以称得上是九域第一人了，只要是楚风眠将来成长起来，九域第一人，也许并非是妄谈。
<br/>　　“纵横古今，我为第一！”
<br/>　　楚风眠心神默念一声，轰的一声，在他身躯之中里的所有劫雷，全部都被吞噬，融入到了楚风眠的血肉之中。
<br/>　　雷霆，本就是淬体最好东西。
<br/>　　劫雷，更是最为精粹的雷霆，以劫雷淬体，绝对是天下间淬体最好的手段。
<br/>　　可是真正足以做到这一幕的人，却是寥寥无几，生死大劫，寻常武者看到都是极为惧怕，都不管去招惹。
<br/>　　能够度过，都已经不易，又有谁敢去淬体，就算是有，也是取之一二。
<br/>　　像是楚风眠这样完整吞噬万道劫雷，融入身躯之中淬体的，九域古今，都从未有过。
