
北方十二月底的天，已经是进入冬季该飘起雪花的时候了，响雷基本是听不到的了，所以今晚京城上空的雷声滚滚着实让不少市民都吓了一跳。<br/>　　<br/>　　十二月的惊雷，来的太不寻常了！<br/>　　<br/>　　不过好在这雷声响了片刻后雨到是还没有下就散了，不过京城上空却仍旧乌云笼罩，压得人心头有点发慌。<br/>　　<br/>　　这种状况持续了很久，到了隔天早上太阳应该照常升起的时候，天边的阴云都没有散去，本该七点左右该大亮的天，依旧灰蒙蒙的，天空中这时也飘起了雪花，纷纷扬扬落到地上后，随即就化开了。<br/>　　<br/>　　雪一直下，没有任何停歇的迹象！<br/>　　<br/>　　于此同时，东四环的那片地皮里，玄门命字脉的人正在紧锣密鼓的改着龙脉，他们务求要在一天的时间里，让这天降异象尽快消失，这才有托词好和上面交代。<br/>　　<br/>　　自古到今，天降异象的事偶有发生，并不稀奇，而且多数都是和这龙脉有关，也就此种状况才会变了天，其实每一次的改朝换代基本都是如此，只不过普罗大众并不知晓是啥状况罢了。<br/>　　<br/>　　这一天，除了京城普降雪花以外，南方开始下起了连绵细雨，这本来在天气预报里并不存在的天气，来的属实有点太突然了些，让预报部门妥妥的尴尬了一把，不少市民都在网上发帖一阵批评，甚至有人都把电话打到了气象局去，一接通就破口大骂起来。<br/>　　<br/>　　官方对此的解释就是，一阵冷空气从西伯利亚突然吹了过来，然后就没了。<br/>　　<br/>　　因为气象局也是一脸懵逼的！<br/>　　<br/>　　午后，天气依旧灰蒙蒙的，雪化飘啊飘的太放纵了，没有一点停止的意思，韩观山这时也接到了来自某某办公室的电话，就是常人所不了解的那种办公桌上的红色电话机打过来的。<br/>　　<br/>　　韩观山朝着胡胡望了一眼，对方点了点头，两人无声的对接了下意思，前者是问你到底把握不把握，后者是告诉他一切尽管放心，于是韩观山走到一旁接通了电话。<br/>　　<br/>　　办公室打过来的电话到还真不是因为天气的原因，不过就是北方下雪南方下雨而已，还达不到让上面兴师动众的地步，主要是因为有位姓向的同志通过层层关系，把一个消息给捅了上来，而这位向同志的师傅叫祝淳刚，则是上面挂了号的，所以向同志汇报的情况，就不得不让上面重视了。<br/>　　<br/>　　王惊蛰的这一手开起了五黑的序幕，他借用向缺来过问此事，给上头吹起了耳边风，先埋下了坑人的引子，只等最后再来个王炸了。<br/>　　<br/>　　韩观山走到一旁，礼敬的说道：“您好，刘秘书……”<br/>　　<br/>　　“韩主任，昨夜到今天的异象，是怎么回事？”<br/>　　<br/>　　韩观山深吸了一口气，缓缓解释道：“刘秘书我们也察觉到了，不过似乎问题不大，大概明日应该就都能一切正常了，现在我们已经调集了玄门的人开始彻查和解决，那边汇报来的状况是偶然之变，今晚到明天就能雪过天晴了，还请您和领导们放心就是了”<br/>　　<br/>　　“能解决？”电话中的声音沉稳的问道。<br/>　　<br/>　　韩观山笑了笑，语态轻松的说道：“给您立个军令状都行……解决不了，我就卷铺盖回家了，随时接受处分！”<br/>　　<br/>　　韩观山和刘秘书就寥寥的聊了几句，随后就把电话给挂了，他来到胡胡身旁问道：“上面已经过问了，我如实交代说是明天差不多就一切正常了，你这里没问题吧？”<br/>　　<br/>　　胡胡笑道：“放心，必须稳妥！”<br/>　　<br/>　　两人随后就轻松，惬意的聊了起来，尽管语气之中还有着那么一丝担忧和紧张，但他们并不认为会出什么大事。<br/>　　<br/>　　时间一晃而过，到了下午的时候，这下雪天果然慢慢变小，随后就停了，天边的阴云也有着散开的迹象了，同时南方的连绵小雨也悄然而止了。<br/>　　<br/>　　一切都是按照计划来的，没有任何的疏漏。<br/>　　<br/>　　东四环的地皮中油毡布也在晚间给撤了下去，祭龙台也被扒了，玄门命字脉的老者跟胡胡和韩观山讲，龙脉已经改完了。<br/>　　<br/>　　王子文激动的伸出手，跟胡胡和韩观山握了握：“地方已经选好了，咱们这就庆功去吧？你们都完事了，剩下的就交给我了，明年开春破土动工，然后楼盘预售，公司里的策划和销售团队已经拿出方案来了，保守估计我们的盈利应该是在百分之三百左右，可能还会更多，因为后期的物业维护费用同样也很可观……”<br/>　　<br/>　　晚间，天已晴，星空高照月光闪耀。<br/>　　<br/>　　胡胡，王子文和韩观山三人找个地方开庆功会去了，这块地皮开启了他们三方合作的开端，眼看着大把的钞票已经开始向他们在招手了。<br/>　　<br/>　　王子文端着酒杯，沉吟片刻后，看着他俩缓缓的说道：“最近一段时间，我这心里一直有个念头在盘算着，之前呢没有漏出来，是觉得心里不太托底，但东四环这片地皮操作成了后，我就把心放了下来，你们听听看？”<br/>　　<br/>　　“行，你说！”<br/>　　<br/>　　王子文一仰而尽，笑道：“我觉得这个模式还可以复制下去，毕竟已经有前车之鉴了么，我的意思是等这块地操作完结束，我们可以再去别的地方找块地皮也如此复制，别管是闹鬼还是风水不好，总之让地出问题不就行了，对不对？随后，我通过官方介入，等到接盘了以后，咱们再重新操作，低买高卖，如此以来我们的成本可就能降到很低，而利润则就可观了”<br/>　　<br/>　　“唰”胡胡和韩观山顿时对视一眼，两人稍一思量，就点头说道：“人才啊，王公子！”<br/>　　<br/>　　王子文淡笑道：“我不懂技术，但懂商业操作，咱们是三剑合璧啊，互相把钱赚吧”
