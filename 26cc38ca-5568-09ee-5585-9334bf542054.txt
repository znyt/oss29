　　第288章
<br/>　　在程大雷开始治兵计划半个多月后，山寨再一次遇到缺粮危机。
<br/>　　突然加入的五百人，成为主要耗粮大户。当然，程大雷已不是初出茅庐的时候，对付这样的事情他也有了经验。这几日，一直派人去各地购粮，用的自然是莫鸣米留下的银子，撑到现在，终于是撑不下去了。
<br/>　　“现在的粮食还够吃几天？”程大雷。
<br/>　　李行哉默默算了一下，道：“最多三天。”
<br/>　　程大雷身边一直却一个懂经济的人才，以前是靠黄三元，但如今黄三元和高飞虎去了南方。现在的山寨，是靠刘悲撑着，只不过也是赶鸭子上架，至于徐神机……他还是充当一个吉祥物更好。
<br/>　　而李行哉却是这方面难得的人才，毕竟是帝国六王子，从小接受精英教育，虽然没有考过文凭（科举），但很显然是山寨文化程度最高的。只是，人家堂堂王子殿下，怎么会陪着自己落草，虽然偶尔也帮一些忙，但正如他所说，还是考察学习的成分多些。
<br/>　　“只能撑三天了么，得想办法从何处搞点粮食。”
<br/>　　琴川的环境和青牛山不同，青牛山穷山恶水，周围村庄比山寨还穷，而相对来说，琴川这里还算富裕些。
<br/>　　所以，程大雷一开始就没有种田的打算。
<br/>　　如窦竹仝等十八家势力，不就是靠收保护费过活么。
<br/>　　程大雷眼睛一亮，对呀，自己怎么忘了他。
<br/>　　程大雷和李行哉、福德勒走进了地牢。
<br/>　　云中龙还被关在地牢内，进攻这些时间，甚至胖了一圈，不得不说，程大雷优待俘虏的政策执行得不错。
<br/>　　一见程大雷，云中龙就来了精神：“程老大，您可算来了，您什么时候放我走。”
<br/>　　程大雷叹口气，道：“本来么，我已经给你的人传信，只要他们把银子拿来，我就打算把你放了。可是，你的人不仅没把银子送来，还与人合起伙来攻打我，哎，这该让我如何是好。”
<br/>　　“那帮兔崽子，程老大，你放我回去，我保证三天内把银子送来。”
<br/>　　“现在，你被关在这里，他们还会听你的么，会不会已经把你当成死人了？”
<br/>　　“这个程老大放心，没有我，那帮兔崽子根本玩不开，只要我回去，还是他们的老大。”
<br/>　　“好，我就信你一次。”程大雷道：“这次你可以回去了。”
<br/>　　程大雷当真是将云中龙放了，由赵子龙率领他的小队押着，直奔灰羊原。临走之前，云中龙对程大雷感激涕零，依依惜别之情溢于言表。
<br/>　　“程老大，我们等着你起来呢，你若能杀了莫鸣米，我们维你程大雷马首是瞻。”
<br/>　　一队人去了，高飞豹和张肥凑过来，道：“大当家，你这可有些不够意思呐，为啥派子龙去，我们也可以去呐。”
<br/>　　分队之后，各个小队之间也有竞争，如此各人的积极性也被调动起来。程大雷这次第一个想到赵子龙，很明显是觉得大家不如赵子龙么。
<br/>　　这，的确是程大雷内心的真实想法。
<br/>　　程大雷把张肥、秦蛮、关鱼、高飞豹四人聚到聚义厅。
<br/>　　“大家已经训练很久了，究竟有没有用，有多大用，还是要通过实战检验。我决定展开为期十天的实战比武，通过战绩定出输赢。”
<br/>　　“怎么个实战比武？”高飞豹。
<br/>　　“抢劫。”
<br/>　　一听这两个字，众人都精神起来。毕竟，繁重的训练确实有些枯燥，大家也愿意去外面活动活动筋骨。
<br/>　　“早就该这样了，在岛上都闲出鸟来。”
<br/>　　“大当家，什么时候开始，是现在么？”
<br/>　　几个人都快跃起来，看上去怎么摁都摁不住。
<br/>　　“立刻，马上。”程大雷道：“这次你们几个分头行动，怎么行动我不管，但是抢谁我要定一下。”
<br/>　　“抢谁？”
<br/>　　“前些日子谁来抢了我们，我们就去抢谁。”
<br/>　　四人点点头，都明白了程大雷的意思。十八家头领在此地作威作福这么多年，积累了多少财富，如果说吃富户，还有比他们更富的么。更何况，双方已经交过手，对方的战斗力根本不值一提。
<br/>　　在得到程大雷命令后，四人离开开始着手行动。鱼字队，张字队，蛮字队，豹字队四支队伍集结。人人都是雄赳赳气昂昂，想要在这次比武中取胜。
<br/>　　一个指挥者，往往能决定一支队伍的精气神。关鱼生性傲慢，向来不知道输字怎么写，而张肥，高飞豹又是平素好争强斗胜的主儿。相比之下，秦蛮的胜负心却是不太强，于是他的蛮字队也是稳扎稳打。
<br/>　　只不过，都是血气方刚的汉子，谁也不想屈居人下，心里都有较量的意思。
<br/>　　程大雷这次是放手不管，随他们去折腾，反正十八家头领的战斗力都不太强，出不来大事。
<br/>　　四队人马各显神通，都出去打劫了，程大雷等人在岛上窝着。这个时候，山寨忽然有人来投奔。
<br/>　　最近时不时有人来投奔，程大雷也不可能每个人都过问，山寨自然有一套考核程序。只不过今天山寨比较空，程大雷也是闲得没事，就帮着徐神机一起考核。
<br/>　　“姓名？”程大雷随口问道，他面前站着一个身高八尺的汉子，五官相貌没有什么出奇之处。
<br/>　　“俺姓张，张强。”汉子笨口拙舌，表情有些憨痴。
<br/>　　程大雷眯起眼睛，多看了这汉子两眼。
<br/>　　“大王，怎么了？”汉子似乎觉察到程大雷的眼神有些异样。
<br/>　　“哦，没什么，干嘛的？”
<br/>　　“在村里是铁匠。”
<br/>　　“喔，铁匠么。”程大雷抿起嘴角，而在他眼里，汉子的信息看得清清楚楚。
<br/>　　姓名：牧长庚（默默无闻的顶级武将）
<br/>　　年龄：32
<br/>　　技能：铁环刀
<br/>　　隐藏属性：无
<br/>　　武将为何要冒充铁匠，姓穆为何要说是姓张……隐姓藏名，却又是为了那般，于是，程大雷的笑容闲得有些意味深长了。
<br/>　　“大王，是哪里不对么？”牧长庚憨憨的挠挠后脑勺。
<br/>　　“喔，原来是张强张兄弟啊，我可是把你盼到了。”
<br/>　　“这个，大王认识我？”
<br/>　　“嘿，以后大家都是兄弟，还有什么不认识的。”
<br/>(本章完)
