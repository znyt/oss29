　　诸磊慌了，彻底慌了。
<br/>　　他的父亲是军人，他的叔叔是军人，他的舅舅是军人，他们家绝大多数人都是在军队里供职，他将来也一定会进入军队，成为其中的一员。而且他还是从红旗学校毕业的，其中所代表的意义非同一般。
<br/>　　按照剧本发展，诸磊绝对可以拥有一个绝对光明的未来；可是现在剧本被方牧南毁了！
<br/>　　诸磊这个时候再也控制不住自己的情绪，冲着方牧南吼道：“你这个老头子，有什么资格开除我！？”
<br/>　　诸磊冲着方牧南这一吼，让办公室内包括诸钟伟在内的所有人，脸上都是露出了失望的神色。
<br/>　　方牧南面对诸磊的质问，只是淡然的一笑，随后说道：“我只是一个腿脚不便的老头子，就是你，也可以一拳把我打倒在地上，我不是红旗学校里的人，我确实没有资格开除你；但是，他可以！”方牧南指了指沈琛的方向。
<br/>　　的确，方牧南没有权利开除诸磊，可是沈琛有啊，沈琛有绝对的权利决定诸磊的去留。
<br/>　　“方老，您看在他还是一个孩子的份上，放他一次。”诸钟伟终归是不忍就这么看着诸磊被这么开除出红旗学校，向方牧南求情道：“他如果被开除了，他就回不了家了。他的父母是绝对不会让他回去的，这辈子就毁了啊。”
<br/>　　方牧南扭头看向站在诸钟伟身边的诸磊，方牧南从诸磊的眼睛中看到了怨恨、愤怒和戾气，看到诸磊的眼睛，方牧南想到了一个人——萧克。
<br/>　　当年赵天宇通过对罗昊和萧克两个人的测试，决定将罗昊作为自己接班人的时候，萧克也露出过跟诸磊一样的眼神。萧克就这么带着对赵天宇的怨恨，对罗昊的仇视，一步一步踏入深渊。
<br/>　　诸磊跟萧克的情况还不一样，诸磊今年已经十二岁了，他在红旗学校里的这几年，经过了最严苛最全面的军事训练，他会近身格斗、会各种轻武器操作，会利用日常生活中所能接触到的东西制作爆炸物，会制作陷阱，熟悉侦察和反侦察……
<br/>　　诸磊的年纪虽然不大，可是他已经是个掌握了各种军事技能的职业军人，毫不夸张的说，如果诸磊走上邪途的话，他所造成的危害绝对要超过萧克，因为萧克并没有诸磊这么全面的军事技能。
<br/>　　方牧南几乎已经可以预见，如果把诸磊开除出红旗学校后，会发生什么样的事情，他制止了正在敲打着电脑键盘的沈琛，站起身来看了眼诸磊，随后对诸钟伟说道：“他可以继续留在红旗学校。”
<br/>　　听到方牧南改变了主意，诸钟伟和诸磊顿时一喜，沈琛、罗昊、齐凡这些人则是露出不解之色。
<br/>　　“他可以继续留在红旗学校，但是他的狙击课程永远不会及格。”方牧南继续说道。
<br/>　　方牧南后续这句话，让罗昊、齐凡、沈琛明白了方牧南的用意。要想从红旗学校毕业，不说每门课程都必须要达到优秀，但是每门课程都必须要及格，只要有一门课程不及格，这个学院就无法从红旗学校毕业。
<br/>　　不说有方牧南在，诸磊的狙击课程不会及格，蓝若依所教授的心理学课程他也不会及格；甚至只需要沈琛的一句话，可以让诸磊任何一门课程不及格，永远都无法从红旗学校毕业离开。
<br/>　　如果说诸磊是一头已经暴露出兽性，随时都有可能露出獠牙，造成危险的野兽，那么红旗学校，就是困住他这头野兽的兽笼！离不开兽笼的野兽，也就不具备威胁。
<br/>　　诸磊的脸刹那间就白了，如果不能从红旗学校毕业，那么继续留在红旗学校还有什么意义，毫无意义，他这辈子只能在红旗学校混吃等死，他甚至不能像自己的叔叔这样成为红旗学校的教官，因为沈琛不会给他这样的机会，等到沈琛从红旗学校校长这个位置上退下来之后，后面继任的校长也同样不会给他成为教官的机会！
<br/>　　诸磊明白，他这辈子完了！
<br/>　　“我不要留在红旗学校！”诸磊大声叫喊着，想要跑出校长办公室，“我不要留在红旗学校！我要离开这个地方！”
<br/>　　齐凡一把抓住诸磊的肩膀，向后一带，把他摔倒在地板上。红旗学校这个地方，进来困难，要想出去也困难！没有沈琛的同意，诸磊是绝对不可能从红旗学校离开的。
<br/>　　如果他想着要从红旗学校逃离，那么他必须要有本事避开红旗学校所有的监控探头、明哨、暗哨、还有数量未知的红外线警报器。
<br/>　　一旦逃跑失败被发现，事情的性质会变得非常严重，因为诸磊会被当成探查红旗学校机密的间谍处理，被送上军事法庭。
<br/>　　从红旗学校创办至今，还没有一个学员能在不被任何人发觉的情况下从红旗学校逃出去；只有蛰龙，在避开了学校所有监控探头、没有被学校明哨、暗哨发现，没有触发学校警报的情况下离开过红旗学校，但蛰龙那次不是从学校逃跑，而是为了帮方牧南出去偷酒！
<br/>　　“把他带走吧。”沈琛对诸钟伟说道：“另外，你暂时不用继续教课了，我会找人替代你，什么时候想清楚了，再来找我。”
<br/>　　诸钟伟看了眼被齐凡后带，摔在地上，摔得七荤八素，还没缓过劲来的诸磊，叹了口气。如果自己当时阻止了诸磊，现在这一切或许根本就不会发生！
<br/>　　现在即便是后悔，一切都已经晚了，根本不可能挽回！
<br/>　　云悠松开了云熙的脖子，充满希翼的看着她，脆生生的说道：“妈妈，我不想要留在这里了，我们回家好不好？”
<br/>　　云悠的话，出乎了在场所有人的意料，也包括云熙在内！其他孩子想要进入红旗学校都是困难重重，而云悠则是被老校长夏洋亲自带进红旗学校的，却想着要从红旗学校离开。
<br/>　　“告诉爷爷，为什么想要离开这里啊？”方牧南笑眯眯地看着云悠，问道。
