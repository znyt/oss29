　　这几名古帝跪在地上，就差磕头了，全然没有古帝的尊严了。
<br/>　　“战魔尊主，我等也没有办法啊。”
<br/>　　“那小子的出手太快，我们赶来时，少宗主就已经陨落了。”
<br/>　　几名古帝急忙的讨饶道。
<br/>　　“不用废话了，你们几个，都去给阴绝陪葬吧！”
<br/>　　战魔尊主冷喝一声，一掌轰击而下，这几名跪在地上古帝，当即被轰杀，一团团血雾，都被战魔尊主，当即吞噬。

<br/>　　在杀了那几名古帝后，战魔尊主的愤怒压抑下来了一点，他死死的盯着玉符之中楚风眠的身影。

<br/>　　在战魔尊主的身旁，一名战魔宗的长老，轻声的询问道。

<br/>　　“上一届九域天才战？楚风眠，尊主是说，当初击败陛下的那名少年？”
<br/>　　这几名战魔宗的长老，许多之前都是六道魔门的新人。
<br/>　　如今这战魔尊主，以前也只是战魔宗的一位长老，后来上一任战魔尊主，正是死在了楚风眠的手中。

<br/>　　在九域剧变的时候，许多六道魔门弟子得到了巨大的好处，实力暴涨，一步登天，都成为了战魔宗的长老。
<br/>　　所以他们都没有见过楚风眠，可是上一届九域天才战，他们可是听说过。
<br/>　　魔帝亲自去参加，败在了一个楚风眠的少年手中。
<br/>　　这件事也是六道魔门的一个禁忌，无人敢提，但是私下里，还是有人议论过。
<br/>　　“这，这楚风眠不是死了吗？”
<br/>　　又一名战魔宗的长老突然开口道。
<br/>　　后来楚风眠去攻打六道魔门，最终楚风眠三人，是跟当时的天魔尊主一起，应该是同归于尽了。
<br/>　　可现在楚风眠还活着？
<br/>　　“这件事，交给陛下去定夺！不过此人，敢杀我的儿子，我一定要杀了他！”
<br/>　　战魔尊主盯着玉符之中的楚风眠，眼神之中杀机涌现道。
<br/>　　“这小子来归墟城，多半是为了去归墟天！通知归墟天之中所有六道魔门的弟子，碰到此子，立刻来通知我，我要亲手去杀了他！”
<br/>　　这战魔尊主也看的出来，楚风眠绝非是一位古帝，真正境界应该就是准帝境界。
<br/>　　但是不知道是有什么惊天的奇遇，才有这如此实力，但是他只要是能抓到楚风眠，杀了楚风眠，并非难事。
<br/>　　“是！”
<br/>　　几名六道魔门长老回应道。
<br/>　　而战魔尊主，则是化为了一道遁光，直接向着归墟天追了过去。
<br/>　　另一边。
<br/>　　归墟城内发生的事，楚风眠不清楚，不过楚风眠也足以猜测的出，战魔尊主的震怒了。
<br/>　　这一次楚风眠可是杀了一名战魔宗的副宗主，还杀了战魔尊主的独子纣阴绝。
<br/>　　只怕是在战魔尊主的心中，已经是不死不休的仇敌了。
<br/>　　在杀了战狠长老的时候，楚风眠还特意将他的记忆搜查了一下，了解了关于六道魔门的事。
<br/>　　六道魔门，如今的六位尊主，都是九劫古帝的境界。
<br/>　　六位副宗主，都是八劫古帝，还有几名八劫古帝实力的长老，而七劫古帝级别的长老，足足一共是有着上百人。
<br/>　　上百位古帝，六名九劫古帝。
<br/>　　这样的实力，简直是比起圣龙秘境，都逊色不了多少了。
<br/>　　当然虽然现在还不如圣龙秘境，可六道魔门才崛起这么短的时间，如果在发展下去，未来绝对比起圣龙秘境，还要可怕。
<br/>　　这让楚风眠也产生了几分压力，他杀了战天冥等人，只怕是他的身份，瞒不过魔帝等人了。
<br/>　　魔帝对楚风眠恨之入骨，一旦是知道楚风眠还活着，定然会来追杀楚风眠的。
<br/>　　不过楚风眠现在担心九劫古帝，可是他一旦是踏入了帝尊境，楚风眠有自信足以跟九劫古帝一战，倒时候面对整个六道魔门的追杀，楚风眠也不担心。
<br/>　　当务之急，还是踏入帝尊境！
<br/>　　楚风眠的遁光，已经是来到了归墟天外，现在的归墟天，已经是跟之前，大不相同了。
<br/>　　以前归墟天的裂痕，每三年才开启一次，一次只有一个月的时间，可现在整个归墟天，已经是门户敞开。
<br/>　　这裂痕之大，已经是足以让任何实力的强者，都踏入其中了，不过在归墟天外，却是有着一群六道魔门的弟子，在把手了。

<br/>　　楚风眠遁光靠近过去的一刻，几名六道魔门弟子已经围了上来。
<br/>　　“滚！”
<br/>　　楚风眠冷哼一声。
<br/>　　这几名普通的六道魔门弟子，楚风眠还懒得跟他们计较。

<br/>　　几名六道魔门弟子，听到楚风眠的话脸色大变，没有丝毫犹豫，运转起杀招来，对着楚风眠轰杀而来。

<br/>　　“这一股灵力要带我去哪里？”
<br/>　　楚风眠虽然来到过归墟天无数次了，但是许多归墟天的隐蔽之地，楚风眠还没有去过。
