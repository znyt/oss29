他带着小心翼翼的试探来问她，夏夕绾就觉得心里骤疼，像是一根细长的针扎进了她的心房里，泛起密密麻麻的疼痛感。
<br/>
<br/>        这三天他不来找她的理由，她已经全猜到了。
<br/>
<br/>        夏夕绾看着他，“不用来找我了…”
<br/>
<br/>        不用来找我了…
<br/>
<br/>        陆寒霆拽着的大拳瞬间松开，很快他将薄唇勾出了一道自嘲而黯淡的弧线，“知道了，我不会去骚扰你的…奶奶那里不用担心，我会解释的，奶奶很喜欢你，以后我不在的时间你去看看她，她会很开心的…我们的婚姻本来就是…有名无实，一场和平协议罢了，后续的事情我会交给律师处理…”
<br/>
<br/>        夏夕绾打断了他的话，“陆先生，我很好奇你是怎么定义有名无实的，难道那个晚上你没有爽到？”
<br/>
<br/>        陆寒霆迅速抿了一下薄唇，声线低哑道，“如果你不想我去骚扰你的话，就不要跟一个成年男人聊这种话题。”
<br/>
<br/>        夏夕绾，“…”
<br/>
<br/>        “你毕竟是我的陆太太一场，我会给予你最好的物质补偿的。”
<br/>
<br/>        “你想怎么补偿我啊？”
<br/>
<br/>        “你喜欢什么，珠宝钻石，别墅，游轮，飞机…我都可以给你买，给你的那张黑金卡，我不会收回，那就是给你的，你可以刷一辈子。”
<br/>
<br/>        那张烫着金色“陆”字的黑金卡可以动用他名下所有的财产，他不打算要回了。
<br/>
<br/>        夏夕绾澄亮的翦瞳隔着一扇落地窗望着他，“陆先生还真是大方，这是分手了还打算养我一辈子啊，为了夏妍妍壕掷十二个亿，跟我分手又要养着我，以后陆先生再多找两个女人会不会破产了啊？”
<br/>
<br/>        “管这么多干什么，有我在，你一辈子有钱花不就行了？”
<br/>
<br/>        夏夕绾被他这一句话给堵得哑口无言，他还真是…一如既往的霸道嚣张。
<br/>
<br/>        “陆先生，不用来找我了，因为我已经来了，找找看，我现在在哪里？”夏夕绾不打算逗他了。
<br/>
<br/>        陆寒霆高大英挺的身躯倏然一僵，她说什么，她已经来了？
<br/>
<br/>        她在哪里？
<br/>
<br/>        陆寒霆捏着手机转过身，一双深邃的狭眸四处眺望，想寻找她那道纤柔玲珑的少女身影。
<br/>
<br/>        在场的高层们都懵了，他们不知道自家总裁在找些什么人，他们还是第一次看到这位身处高位，深沉杀伐的男人露出这样焦急而紧张的神色。
<br/>
<br/>        “你在哪里，我没有看到你。”
<br/>
<br/>        夏夕绾看着他，看着他在满世界的找她，她轻勾红唇，“陆先生，你真笨，抬头，向前看。”
<br/>
<br/>        陆寒霆抬头，一下子就看到了陆氏大厅里亭亭玉立的那道纤柔身影，夏夕绾就在那里，在他的陆氏里，俏生生的，迎着一双黑漉漉而水汪汪的眸子盈盈的望着他。
<br/>
<br/>        她的瞳仁里，满满都是他的倒影。
<br/>
<br/>        陆寒霆挂了电话，迈着稳健的步伐迅速走了进去，他高大的身躯停在女孩面前，“你怎么来了？”
<br/>
<br/>        夏夕绾仰着绝色的小脸看着他，“笨蛋，我不一定在后面，也许我在前面停下脚步等着你呢。”
<br/>
<br/>        刚跑出来的前台瞬间僵住了，这是什么情况，难道这个女孩一直在跟总裁通电话？天哪！

<br/>        陆寒霆眼眶发红狠戾的将夏夕绾盯住，“你来干什么，考虑过后果么？”
