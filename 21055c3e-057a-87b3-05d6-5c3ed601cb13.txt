夜老曾经说过，他绝对不会让夏夕绾进科学院的大门的，没想到今天夏夕绾光明正大的走了进来。

<br/>        夜老就觉得，这个夏夕绾是给他上了一课，什么叫话不能说太满！

<br/>        夜老看着夏夕绾，“你就是邱远派来的人？你跟邱远是什么关系，邱远是T大校长，而你就读的是A大。”
<br/>
<br/>        围观的人群也是惊掉了下巴，他们都认识夏夕绾的，最近夏夕绾名声太响了，再加上她成为了陆子羡的未婚妻，简直是万千少女的…噩梦！
<br/>
<br/>        夏夕绾怎么来了？

<br/>        我还以为来的是X，没想到来的只是夏夕绾。

<br/>        是啊，邱院士怎么派夏夕绾来了，他跟夏夕绾是南辕北辙的两个人吧，他们怎么有交集的？
<br/>
<br/>        先不要管这些好吗，她竟然是陆院士的未婚妻了，啊，我都心痛的快死掉了！
<br/>
<br/>        夏夕绾没有受这些议论声的影响，她挺直了纤柔的美背，气场淡定从容，大家风范，她看着夜老，“夜老，不管是T大还是A大，我都是一名学生，我也是邱院士的学生。”
<br/>
<br/>        夜老冷哼了一声，这个夏夕绾还真是牙尖嘴利，她没有正面回答问题，而是用学生和老师的关系来解答她和邱远的关系，偏偏这个回答让人抓不到一点的错处。

<br/>
<br/>        范恬看着夜老离开的身影，久久没有收回目光。
<br/>
<br/>        这时范恬就感觉一道目光落在了她的脸上，她迅速警觉，扭头一看，直接撞上了夏夕绾那双澄亮剔透的水眸。
<br/>
<br/>        夏夕绾正看着她，已经将她刚才盯着夜老看的那一幕都尽收眼底了。
<br/>
<br/>        范恬瞬间就有了一种很不好的预感，这个夏夕绾是不是已经知道些什么了？

<br/>
<br/>        夏夕绾勾起了红唇，一双翦瞳盈盈发亮，“厉嫣然，我发现你越来越喜欢废话了，不如说点有用的吧。”
<br/>
<br/>        厉嫣然一张漂亮的小脸都冷了下来，“看来你是不见棺材不掉泪了，走吧，我们一起去电台大楼见台长，你还不知道吧，台长就是我妈咪，奶奶想认你当孙女，我妈咪早就想见一见你了。”
<br/>
<br/>        夏夕绾知道了，电台就是一个雷，她踩到雷了。
<br/>
<br/>        厉嫣然的妈咪就是夜老的亲女儿夜滢，不过夏夕绾听说厉嫣然的爹地厉君墨和夜滢的感情并不好，很多年前就开始分居了。
<br/>
<br/>        厉君墨这些年一心致力慈善，是慈善大亨，行踪神秘，而夜滢管理电台，是一个女强人。
<br/>
<br/>        夏夕绾来到了科学院的电台大楼，当看到富丽堂皇，高入云端的电台塔，她再一次感叹资本的力量，听说这个电台是厉君墨捐赠的，甚至二十年前夜家在林水瑶的重创下差点覆灭，都是厉君墨出手稳住了夜家，更在这些年间将夜家扶植成了帝都四大豪门之一。
