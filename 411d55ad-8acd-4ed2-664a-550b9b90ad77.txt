　　“咕。”
<br/>　　萧逸不禁咽了口口水。
<br/>　　还好他刚才反应快，连忙发动了大阵。
<br/>　　否则，就刚才八龙焚火炉爆发的这四道火焰冲击，怕是瞬间就能将他轰成重伤。
<br/>　　他现今实力，毕竟是连第二道封印都奈何不得。
<br/>　　这第四道封印解开所爆发的火焰，自远远不是他能抵挡的。
<br/>　　“还好冰尊者的冰玄封界阵足够强。”萧逸自笑一声。
<br/>　　随后，感知放出，进入八龙焚火炉内。
<br/>　　果然，八龙焚火炉内的那片火焰空间，再次增大了。
<br/>　　算起来，对比之前第一道封印解开时的空间，如今整整大了4倍。
<br/>　　而同时，又是一不知名的黝黑兽手，出现于火焰空间内。
<br/>　　“出。”萧逸低喝一声。
<br/>　　嗖…一股赤红色光芒激射而来。
<br/>　　这一次，光芒未聚集于萧逸手中，而是聚集于左脚之上。
<br/>　　萧逸看了眼，光芒正环绕着他的膝盖部分。
<br/>　　又是一阵难受的炙热感，他的左脚，已与妖兽之脚无异。
<br/>　　之前与金刚白猿战斗之时，他的衣裳已打得破破烂烂了，自然一眼看到脚上情况。
<br/>　　“原来是兽腿。”萧逸自语一声。
<br/>　　之前在他的感知中，还以为又是兽手，没想到竟是兽腿。
<br/>　　“嗯？不对。”萧逸蓦地反应了过来。
<br/>　　从解开八龙焚火炉内封印至今，先是左手，后是右手，现在又是左脚。
<br/>　　萧逸面露惊色，按这般规律下去，第五道封印解开的话，岂非会出现右脚？
<br/>　　难道说…
<br/>　　萧逸心头冒出一个惊骇的想法。
<br/>　　“难道说，这八道封印之内，便是一头强大妖兽的完整尸骸？”
<br/>　　“先是左手，后是右手，现今是左脚，之后右脚、身躯、头颅…”
<br/>　　若真是如此的话，这头不知名的强大妖兽，竟需要以分尸的形式，再特别给出各道封印，封于八龙焚火炉内。
<br/>　　那这头妖兽生前，到底强悍到何等地步？
<br/>　　毕竟现今才第四道封印的火焰冲击，已有如此威力。
<br/>　　后面的各道封印，又该有何等威力？
<br/>　　萧逸蓦地想起来剑姬前辈的话，八龙焚火炉内，有八道龙炎吐息，乃是最强火属性重宝。
<br/>　　或者，剑姬前辈的话是真的。
<br/>　　八龙焚火炉内，确有八道龙炎吐息。
<br/>　　不过，八龙焚火炉的封印，并非是为了存放这八道龙炎。
<br/>　　而是为了存放这不知名的妖兽尸骸。
<br/>　　八道龙炎吐息的存在，乃是辅助封印，镇压这些尸骸。
<br/>　　据萧逸所知，在一些异常强悍的上古大阵中，确有不少大阵会以强大的火焰去增强大阵或者封印的威力。
<br/>　　拿他自己的阵法来说，有时也会拿冷焱剑等强大之物来增加镇压威力。
<br/>　　“这妖兽，死了还要分尸封印镇压？”萧逸眉头紧紧地皱了起来。
<br/>　　他向来谨慎，这些问题自然是要多谨慎几分。
<br/>　　萧逸双眼眯了眯，半晌，低喝一声，“散。”
<br/>　　双手、左脚，霎时恢复了正常。
<br/>　　两只兽手，以及一只兽脚，瞬间回到八龙焚火炉的那片火焰空间内。
<br/>　　“出。”萧逸低喝一声。
<br/>　　两只兽手，一只兽脚，再次覆盖出现于自己身上。
<br/>　　“散。”
<br/>　　“出。”
<br/>　　萧逸一连低喝了几声，尝试了数次。
<br/>　　很显然，这两只兽手与兽脚，都在八龙焚火炉的绝对控制之下。
<br/>　　而也是八龙焚火炉，这些妖兽的尸骸，才能以覆盖的方式，被自己所借用。
<br/>　　这些兽手、兽脚，是真正的妖兽尸骸。
<br/>　　不过因为八龙焚火炉的缘故，却幻化成赤红色光芒的存在，随后才能覆盖自己身上。
<br/>　　只要自己绝对掌控着八龙焚火炉，便也能绝对掌控这些尸骸。
<br/>　　当然，八龙焚火炉到底有着什么秘密，萧逸也不知晓。
<br/>　　“若是…”萧逸皱眉思索着。
<br/>　　边思索，也边看了眼漂浮于前方的八龙焚火炉。
<br/>　　此时，八龙焚火炉仍旧在源源不断地涌入金光力量。
<br/>　　他的思索，也忽然停下。
<br/>　　一切，还是等八龙焚火炉被金光力量冲破多少道封印再说。
<br/>　　趁着八龙焚火炉还在涌入金光力量，距离填满还有一段时间，萧逸再度试了试两只兽手与兽脚的威力。
<br/>　　咔咔咔…
<br/>　　萧逸握了握拳头，手臂以及拳头上蕴含的仿佛无穷无尽的力量，让他心惊。
<br/>　　鳞甲上夹着着的点点红芒，亦散发着无比炙热的气息。
<br/>　　嘭…
<br/>　　萧逸手中一凝，一股澎湃的火焰，瞬间出现。
<br/>　　“好强。”萧逸脸色一变。
<br/>　　他现今凝聚的，不过是普通火焰。
<br/>　　嘭…又是一股火焰凝聚而出，火焰，呈紫色。
<br/>　　现今凝聚的，乃是紫晶灵炎。
<br/>　　几乎是紫炎凝聚的瞬间，瞬成火海。
<br/>　　火海，漂浮于半空。
<br/>　　萧逸握了握拳头，明显感知到其中蕴含的绝对掌控。
<br/>　　这偌大片火海，在他手中，简直成了如臂指使，随意操控之物，毫无压力可言。
<br/>　　要知道，这么一大片紫炎火海，蕴含力量之浑厚，可绝不是能轻易掌控的。
<br/>　　可现今，在这兽爪之下，却是轻如空气。
<br/>　　“好恐怖的控火能力。”萧逸倒吸一口凉气。
<br/>　　这头不知名的妖兽，生前，比是一头恐怖的控火妖兽。
<br/>　　“喝。”萧逸低喝一声，一拳打出。
<br/>　　半空中的紫炎火海已然消失。
<br/>　　萧逸凌空打出的一拳，破空声震耳欲聋。
<br/>　　拳出，一股火焰的爆发性，蕴含其中。
<br/>　　前方，大阵之外的火海，瞬间被冲散。
<br/>　　“好强的爆发力。”萧逸再次吃了一惊。
<br/>　　下一秒，脚步一瞪，瞬间跃起。
<br/>　　借力瞪起的，正是此刻被妖兽鳞甲覆盖的左脚。
<br/>　　左脚之上，黝黑鳞甲内的红芒点点，霎时爆发。
<br/>　　脚中，惊人的爆发力陡然产生一股气爆声。
<br/>　　几乎只是一个呼吸间，萧逸已化作一道火红色流光，冲天而起，凌驾云端之上。
<br/>　　嗖…萧逸脚步一瞪，猛踩空气。
<br/>　　一个闪身间，脚中爆发，横跨万米。
<br/>　　“好快。”萧逸脸上惊色，愈发浓郁。
<br/>　　嗖…
<br/>　　萧逸再次脚步一瞪，从高空落下，稳稳的回到大阵之内。
<br/>　　此刻的萧逸，脸上写满了喜意。
<br/>　　这双兽手，以及兽脚，坚逾金石，完全可当护身铠甲或者武器所用。
<br/>　　且，其内明显拥有极强的控火能力。
<br/>　　完全可以作为控火重宝。
<br/>　　就他刚才的一番尝试来看，控火能力以及增幅之强，甚至远在中品圣器星幻手套之上。
