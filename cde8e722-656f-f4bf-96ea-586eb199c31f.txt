　　第225章
<br/>　　戎族这次派出的谈判团人数超过三百人，由草原上数百个部落组成。总负责人是来自羌部的王子呼延力，另外北蛮部的金问道也在其中。除了谈判，他们还有采购的任务，把帝国的纸张，茶叶，铁器之类的带回去。这些东西在戎族都是天价，一口锅换一个女人并不夸张。
<br/>　　帝国这边的负责人是大王子李乐天，谈判代表由礼部，兵部等老奸巨猾的官员组成。
<br/>　　谈判进行得不太顺利，或者说，对帝国方面很不顺利。帝国说话的特点是客套，总是：哈，今天天气不错，去年天气可没这么好，呵呵，前年天气也不错……东拉西扯，每句话里都打着机锋，没有一定学识根本听不懂。
<br/>　　而戎族都是直爽的汉子，根本受不了这一套，确实也听不懂这一套，动不动就拔出刀：你们要是不给，我们就抢了。
<br/>　　弱国无外交。
<br/>　　其实在去年，戎族便逼着帝国签下城下之盟，如今只是确定一些执行细节。割地，赔款，还有和亲。
<br/>　　经过一系列讨价还价，已确定大部分事情，如今要聊得也只是和亲一项。
<br/>　　其实也并不复杂，无非临时找个宫女封为公主，至于帝国真正的公主，哪怕是亲王的女儿是不舍得嫁到戎族那样的蛮夷之地。
<br/>　　羌部的首领名叫呼延跋，他还有一个称号：戎族王，整个戎族名义上的统治者。据说此人虽然已年过半百，但依旧体力惊人，能夜御数女面不改色。
<br/>　　而戎族那样的蛮夷之地，帝国人心里还都是鄙视的。
<br/>　　今日谈判又一次陷入僵局，似乎是呼延力对帝国选出来的『公主』不太满意。
<br/>　　“我看今天不如就如此，择日再谈。”李乐天道。反正这样的僵局不是第一次遇到，慢慢磨总会有解决的办法。
<br/>　　“我们是长生天的儿子，你们休想拿一个平民女子糊弄。”呼延力。
<br/>　　“怎么会，怎么会。”李乐天笑着：“幽州王恰好也在京城，诸位要不要见上一见。”
<br/>　　言外之意就是，我们也有打败你们的人，别太放肆，没什么用。
<br/>　　“幽州王？”呼延力目光环视身边的族人，发现大家都摇摇头：“见什么见，又没听说过。”
<br/>　　“就是杨龙停，是……”李乐天想说：就是把你们打得好惨好惨的人，你们怎么能没听说过。
<br/>　　“不认识，不见。”呼延力直接道。
<br/>　　李乐天摇摇头，心道：戎族还真是没气度，面对曾经战胜他们的人连见一面的勇气都没有，好像是故意不提起此事，自己这句话应该恰好击中他们痛处了。
<br/>　　“如果有个人在长安，我倒是想见一见。”金问道突然发声。
<br/>　　“不知王子想见谁，若离得近的话，应该是可以见到的。”
<br/>　　“他……”金问道脑海中浮现不好的回忆：“他自称蛤蟆大王，应该叫……程大雷。”
<br/>　　“程大雷……”李乐天怔住，目光环视周围的朝廷官员，大家和刚才戎族的表情一样：不认识，没听说过。
<br/>　　可为何金问道别的不见，偏偏要见这程大雷，他究竟是谁。
<br/>　　“程将军击败我们，屠戮我族很多勇士，身为北蛮部的王子我应该恨他，但作为军人，我尊敬他。”金问道。
<br/>　　李乐天一脸懵逼：谁呀，究竟谁呀。
<br/>　　“立下这么大功，帝国不会不封赏他，他现在应当也在长安吧。”金问道。
<br/>　　“哈，是是，我尽快让他来见诸位。”李乐天。
<br/>　　众人一起往大殿外走，李乐天还在琢磨金问道刚才的话，一时有些失神。正此时一个小太监从旁边的夹道跑过来，不偏不倚撞在李乐天身上。
<br/>　　李乐天看到她模样，就是一惊，你往日胡闹也就算了，今天是什么场合，也敢蹦出来乱转。
<br/>　　“快滚！”
<br/>　　李婉儿跑得气喘吁吁，两腮泛红，看到这一幕也吓了一跳。
<br/>　　“是是。”
<br/>　　呼延力饶有兴致的盯着这小太监，道：“大王子，这位是……”
<br/>　　“一个不懂事的小太监，打扰了诸位。”
<br/>　　李婉儿低着头往外走，或许是太紧张的缘故，踩到台阶上，一下扑倒在地上。
<br/>　　李乐天想捂脸：太丢人了。
<br/>　　李婉儿抬起头，冠帽跌落，如瀑长发垂落，一张脸紧张的看着后面，却见所有人的目光都看着自己，此时她额头泛起汗珠，心情是和李乐天一样：太丢人了。
<br/>　　“明玉公主，明玉公主。”
<br/>　　两个宫女急忙忙跑过来，搀起李婉儿，快步离开。
<br/>　　李乐天干笑两声，道：“哈，手下没有规矩，回头我一定狠狠惩罚他们。”
<br/>　　呼延力仍注视着李婉儿的背影：“没什么，没什么。”
<br/>　　再将戎族送到住处后，李乐天来到未央宫，先恭恭敬敬的向明帝行了一礼。
<br/>　　“谈完了？”明帝。
<br/>　　“已将他们送回去，如今尚未确认的只有和亲一项，但很快就会有结果。”
<br/>　　明帝轻轻点点头，看着面前跪倒的李乐天，道：“委屈你了。”
<br/>　　割地，赔款，和亲这都不是露脸的事情，任何一个和戎族签下这些的人，都要承受被天下人戳脊梁骨的结果，甚至可以说遗臭万年。李乐天肯负责这件事，在很多人看来，他的政治生涯已经毁了，未来与皇位无缘。
<br/>　　但李乐天心里清楚，决定皇位的不是天下人，是明帝。毕竟战败了，总要有人承受骂名，这骂名是替父王背的，父王能不念自己好么。
<br/>　　“替父王分忧，本是儿臣该做的。”李乐天又深施一礼。
<br/>　　“起来说话吧，尝尝御膳房送来的点心。”
<br/>　　“是是。”李乐天激动得不行，这样的关心可是以前从未有过的。
<br/>　　“若戎族再得寸进尺，下次谈判就让杨龙停去，也让他们懂点事。”
<br/>　　“估计……没什么用。”李乐天顿了顿：“他们好像不是很乐意见到杨龙停，反而这次提到一个叫程大雷的人。”
<br/>　　“谁？”明帝也是一脸懵逼。
<br/>　　“我也不清楚，但看戎族方面的意思，这程大雷好像在戎族很受尊敬。”
<br/>　　明帝皱起眉头，摆摆手道：“查一查。”
<br/>　　
<br/>(本章完)
