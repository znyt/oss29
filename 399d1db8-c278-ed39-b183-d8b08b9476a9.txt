<br/>    燕山之上，灵药飘香，陈明享受着卓青瑶小手的按压，一缕缕剑气渗入陈明的体内，在骨节之间游走，别说，还真是相当的舒服！
<br/>    
<br/>        黑蟒前来禀报道：“山主，那些妖王已经被彻底震撼到了，昨天晚上，我敢保证，他们一夜都没睡。”
<br/>    
<br/>        陈明点点头，然后道：“很好，既然已经镇住了他们，那接下来的事情就好办了。再次将他们召集在一起吧。”
<br/>    
<br/>        众多妖王等待着陈明的接见，不知道将会面对什么。陈明走到大厅之内，不过刚刚来到，所有妖王连忙起身，他们经过昨天的事之后，是真的怕了，他们甚至不敢猜测陈明这具身体之中蕴含着多大的能量。
<br/>    
<br/>        陈明坐在自己的位置上，然后挥手示意众多妖王坐下，然后发表了整个燕山应该共同发展的讲话，“昨天的炼丹房，想必诸位也已经看到了。”
<br/>    
<br/>        众多妖王连连点头，“我等已经见过了燕山的鬼斧神工了！”
<br/>    
<br/>        “这实在是太可怕了！”
<br/>    
<br/>        “原来我等只是萤火之光，竟然在与燕山争锋，真的是在找死！”
<br/>    
<br/>        陈明继续道：“既然你们已经愿意归属我燕山，那么，从今往后，都要在燕山的规矩中来。而燕山的第一条规矩，便是除了燕山之外，不能有任何武力存在，所有灵兵，收归燕山所有，燕山会根据灵兵的级别相应给予灵石。收集所有功法，所有燕山境内的修行功法，术法，阵法，丹方，全部上交燕山。”
<br/>    
<br/>        陈明一语道出，众多妖王纷纷大乱，“这是要拔掉我们的爪牙！”
<br/>    
<br/>        “山主难道现在还不相信我们已经臣服了吗？”
<br/>    
<br/>        “我们只是需要部分妖军保护自己，绝对不会对燕山有任何非分之想！”
<br/>    
<br/>        陈明根本不顾这些妖王的反对，继续道：“燕山境内，所有修行者，每十年都必须在燕山建立新的身份文牒，若是没有身份文牒的，一旦在燕山境内发现，立刻收押入狱，直到查清楚身份为止。从今以后，所有修行者的安全，归燕山保护，只要在燕山境内，那么燕山将会用尽全力斩除所有威胁你们生存的危险！”
<br/>    
<br/>        “若是想要外出，可以雇佣燕山新成立的燕山镖师保护自身的安全。”
<br/>    
<br/>        “燕山境内，所有人，每年都要上交十分之一的收入作为税收。”
<br/>    
<br/>        “为了更好的保障诸位的安全，燕山将会成立燕山钱庄，诸位可以将自身的灵石储存在燕山钱庄之中，根据身份文牒随时取出，随时储存。”
<br/>    
<br/>        众多妖王议论纷纷，关于陈明定下的规矩，他们可以感受到其中的严密性，陈明是想要彻底掌控整个燕山境内，掌控到每一个修行者。而且彻底剥夺了他们蓄养兵马的权力，从今以后，在燕山，之后燕山可以拥有妖军。
<br/>    
<br/>        这一点，是他们最不能接受的。
<br/>    
<br/>        至于后面的几条，他们都可以接受，毕竟他们没有了妖军之后，自身的安全将会是一个很大的问题，燕山将会保护他们。为了防止有人见财起意，更是让他们可以将所有的灵石全部储存在燕山之中，不收取分文保管费。对于十分之一的税收他们倒是没有什么反应，这个税收，对于之前妖月王庭来说，已经很低了。
<br/>    
<br/>        “山主，我们绝不会反叛的，我们只是想留下妖军保护自己！”
<br/>    
<br/>        “我们可以接受裁减妖军，但是要我们彻底放弃，这不可能！”
<br/>    
<br/>        “山主，还是莫要做的太狠了，若是没有妖军，我们如何掌控这偌大的领地？”
<br/>    
<br/>        陈明冷哼一声，然后道：“你们要知道，我现在并不是在和你们商量！我只是告诉你们规矩！另外，现在选择同意的妖王，每一位道种境妖王都可以拥有一处燕山这样的炼丹房！神府境三处！你们可以派遣麾下的大妖前来燕山，燕山将会对他们进行训练，直到他们会炼制丹药之后，就会送还给你们，任由你们自由支配！”
<br/>    
<br/>        严格的规矩定下来，自然要给点好处，不过在这些妖王的耳中，这岂止是一点好处啊，简直就是大大的好处啊！
<br/>    
<br/>        燕山的炼丹房他们已经见识过了，简直就是鬼斧神工，那些炼丹的，同样是妖族，那些妖族都学得会，他们麾下那么多大妖，一定也学得会！
<br/>    
<br/>        一天一万灵石，那一年就是三百多万灵石，有妖王看向拜月，问道：“拜月道兄，不知道那叛贼妖月，一年能收获多少灵石？”
<br/>    
<br/>        拜月想了想，心中也是有些惊恐，然后道：“一年不过也就几十万灵石罢了。”
<br/>    
<br/>        众多妖王听到这句话，简直心驰神往，不就是没有了妖军嘛，燕山已经承诺保护他们了，一年数百万灵石，还雇佣不起燕山的镖师不成？而且他们也有了灵石，有了修行来源，本来养这些妖军，就是为了巩固自己的统治，但是现在，燕山帮他们照看。
<br/>    
<br/>        以前养着这些家伙，一年才几万灵石，自己修行都不够，现在好了吧，不用养这些家伙，自己每年上百万的灵石！既然有了修行的灵石，那还要妖军干嘛！
<br/>    
<br/>        不过就是那个灵兵收缴的规矩，让他们还是有些难受。
<br/>    
<br/>        陈明也是看出了他们的顾虑，继续道：“若是在燕山境内，表现得很好的妖王，我会允许拥有灵兵，身份文牒上会有特殊标志。”
<br/>    
<br/>        燕山的规矩，虽然剥夺了他们很多权力，但是却是给了他们更多获取灵石的机会，这个时候，还有谁会反对，拜月第一个道：“山主英名，宽宏大量，我愿意接受山主的条件！”
<br/>    
<br/>        陈明对拜月露出一抹满意的神色，然后道：“待会儿，拜月留下来，我有事情吩咐你。”
<br/>    
<br/>        拜月大喜，他就知道第一个同意的会有好处！
<br/>    
<br/>        众多妖王也是争前恐后的答应了下来，向着陈明表明自己同意了！
<br/>    
<br/>        陈明露出一抹笑容，伸手道：“好好好，接下来的事情，就去找白狐和黑蟒吧。”
<br/>    
<br/>        在其他的妖王离开之后，拜月朝着陈明行礼道：“不知道山主留下我有何事？”
<br/>    
<br/>        陈明笑着道：“你刚刚表现的很不错，我决定了，你可以拥有四处炼丹房，如何？”
<br/>    
<br/>        拜月大喜过望，这可是所有人之中独一份的，“多谢山主！”
