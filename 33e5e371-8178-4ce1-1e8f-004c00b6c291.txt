　　慕清霄自然知道贪多嚼不烂的道理，但他也有自信，几年内将两门剑法参悟。
<br/>　　空明剑与太虚剑无一不是天下间一等一的剑诀，乃是执剑长老紫胤真人的成名绝学，威力自然是不用说。
<br/>　　而且，想要修炼两门剑诀，必须对剑道有所领悟。
<br/>　　若是普通弟子来修炼，估计无从下手，但慕清霄起步不同，他对剑道上的领悟已经很深，修炼两门剑法根本算不上困难。
<br/>　　不仅仅是因为更高深的剑法，他更期待在第四层，自己能获得多少掠夺积分。
<br/>　　“不劳凝虚真人费心，若非晚辈有自信，否则，也不会想去第四层。”
<br/>　　“......老夫自然清楚你资质非凡，也不是想要阻止你。”
<br/>　　说着，凝虚真人叹了口气，微微摇头，道：“但想要去第四层，必须先将空明剑与太虚剑参透，这是掌门立下的规矩，我们做长老的自然也要遵守。”
<br/>　　闻言，慕清霄剑眉一挑，道：“难道前辈也没有去过第四层？”
<br/>　　“没有，妙法堂第四层，百年来只有掌门与执剑长老上去过，至于其他长老，都没有资格上去。”
<br/>　　凝虚真人眼中闪过一丝遗憾之色，道：“我在剑道上的天赋一般，空明剑与太虚剑只是领悟的皮毛而已，自然也没有上去的资格。”
<br/>　　慕清霄微微点头，凝虚真人所说，与第一层弟子口中的信息有些偏差。
<br/>　　作为凝丹长老，妙法堂前三层都对其开放，但想要上第四层，必须参悟空明剑与太虚剑才能前往。
<br/>　　哪怕他是凝丹长老，也必须遵守掌门立下的规矩。
<br/>　　瞥了眼通往第四层的阶梯，慕清霄收回目光，道：“既然如此，那便只能先将两门剑法参悟后，到时候在来打搅前辈。”
<br/>　　暗道可惜，但他也没办法，既然有规矩他也只好遵守，只好参悟两门剑法后，到时候再来一趟。
<br/>　　挥了挥手，凝虚真人一脸无奈道：“老夫也就比你年长而已，辈分我们都是长老辈，不需要如此客气。”
<br/>　　“告辞。”
<br/>　　说罢，慕清霄就迫不及待的离开妙法堂，准备回自己的宫殿修炼。
<br/>　　望着慕清霄离去的背影，凝虚真人捋了捋胡子，道：“后生可畏啊，说不定在不久的将来，天墉城又有一尊剑仙横空出世。”
<br/>　　...
<br/>　　...
<br/>　　作为天墉城最年轻的凝丹长老，一路上遇到慕清霄的弟子，纷纷恭敬的朝他打招呼行礼。
<br/>　　慕清霄也不倨傲，对于打招呼的弟子，均是报以和煦的微笑，令得弟子们对他更加的尊敬。
<br/>　　“陵越，见过慕长老。”
<br/>　　行走在道路上，一道颇为熟悉的声音传入耳中，慕清霄一愣，旋即抬起头来。
<br/>　　映入眼中的是一名十岁左右的少年，身穿一袭白紫相间的长袍，有些长的黑发束在脑后，看起来气宇轩昂。
<br/>　　陵越在看到慕清霄后，快步走了上来，满脸笑容道：“前些天，多谢慕长老相助。”
<br/>　　“小事，不足挂齿。”
<br/>　　望着面前的少年，眼眸深处的那一抹坚定长存，慕清霄微微一笑，对他的印象颇为不错。
<br/>　　闲聊几句后，慕清霄就转身离去，陵越只是一个十岁的少年，两人本就没有任何的话题可言。
<br/>　　十分钟后，回到住处，慕清霄就来到悬崖最边缘的峭壁上，道：“也是时候突破炼虚境了，不过在这之前，需要炼制一枚丹药。”
<br/>　　说罢，一尊药鼎就出现在悬崖边上，盘坐而下，一缕黑色火焰自他掌心窜起，旋即灌入药鼎中。
<br/>　　眨眼间，黑色神火便在药鼎中翻滚起来。
<br/>　　望着药鼎中翻滚的火焰，不知何时，慕清霄手中已经出现两个玉瓶，从中倒出几株药材与一枚紫色的灵果。
<br/>　　待火势差不多，屈指一弹，几株药材就被他同时送进药鼎中，顿时，神火涌上将灵药包裹起来，开始焚烧与提炼。
<br/>　　在斗破位面的时候，慕清霄就在为突破练虚境做准备，他接下来要炼制的丹药，名‘紫灵破障丹’。
<br/>　　紫灵破障丹，七品丹药，能够打破化神期与练虚期之间的壁障，甚至能提升服用者的修为。
<br/>　　七品丹药珍贵至极，整个天墉城，除了他以外，也就只有还虚真人能炼制。
<br/>　　灵药投入药鼎中，慕清霄也不再迟疑，开始操控着神火提炼灵药中的精纯能量，去除杂质。
<br/>　　若是让还虚真人看到这一幕，估计会惊得连下巴都掉在地上，同时提炼四五种灵药，反正他是做不到。
<br/>　　要知道炼丹的提炼过程，必须集中精神，哪怕有一丝的不慎，都有可能毁了灵药。
<br/>　　而慕清霄对神火的控制本就源于血脉，以至于娴熟的炼丹技术，早就超出了七品炼丹师的范围。
<br/>　　待灵药提炼完成，几种能量开始融合，慕清霄则微微闭上双眼，将神火调制到一定程度后，观察着药鼎中的一举一动。
<br/>　　随着时间的推移，转眼间两个时辰就过去了。
<br/>　　陡然，慕清霄睁开双眼，双手结印，灵魂力量控制着庞大的能量，道：“凝。”
<br/>　　半刻钟后，只见他手轻轻一挥，神火就熄灭，鼎盖开启，伴随着紫色光晕透射而出，旋即一股浓郁的丹香扑面而来。
<br/>　　望着药鼎中，龙眼大小，圆润的紫色灵丹，慕清霄颇为满意，手一卷紫色灵丹就飞射而出，悬浮在掌心之上。
<br/>　　“紫灵破障丹，七品丹药，接下来就可以尝试着突破练虚境了。”
<br/>　　说罢，慕清霄就将紫灵破障丹装进玉瓶，同时也将万兽鼎收了起来。
<br/>　　看了看周围，虽然灵气充沛，但这里是天墉城，如果在这里突破，绝对会引起所有人的注意，这可不是慕清霄所希望的。
<br/>　　“看来得下山一趟才行，等突破练虚期在回来，反正也不需要多久。”
<br/>　　想到此处，慕清霄看了一眼宫殿，周围空间一阵扭曲，身影便自峭壁上消失不见。
<br/>　　此时，宫殿内小兰正在修炼，耳中忽然响起一道声音，将她从修炼中惊醒。
<br/>　　“本座偶然顿悟，找到了突破的契机，需要下山一趟，少则三五天，多则半个月，你自行在宫中修炼，切勿太过依赖丹药。”
<br/>　　望着殿外，小兰幡然醒悟，恭敬行礼：“多谢真人指点。”
