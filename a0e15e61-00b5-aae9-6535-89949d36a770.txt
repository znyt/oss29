        “你拿你自己的生命威胁的，不是吗？”童筱颖一直都是这么认为的。
<br/>
<br/>        季思妍微微摇了摇头，目光清冷地瞥向童筱颖：“你们同床共枕三年，你居然一点都了解他。”
<br/>
<br/>        “你这话什么意思？”童筱颖惊愕。
<br/>
<br/>        季思妍咧嘴一笑：“我跟阿景说，只要他跟你把离婚证领了，我就把我在季氏集团那百分之二十五的股份转给他。我还跟他说，我哥现在是彻底废了。隔三差五地就往监狱里蹲，季氏集团，我爸是不可能交到我哥手中的。所以，那段时间，他为了讨好我爸。在我家，真的是大献各种殷勤。就连我爸把他带出去一起谈商务，他的机敏才智，都让我爸很是欣赏。”
<br/>
<br/>        “昼景一直都是怀才不遇，这点我知道。”童筱颖还以为她季思妍要说出个什么惊天的理由，原来不过是这种。
<br/>
<br/>        “你当时为什么要那么爽快地就答应跟他离婚呢？你应该像之前对待我那样，不仅帮他争夺尚轩的抚养权，继续把我逼入绝境，赶出澳洲才对。更何况，我要自杀是我的事，跟你童筱颖有何干系呢？”季思妍忽然间冷嘲热讽起来。
<br/>
<br/>        童筱颖却不以为然：“你明明有抑郁症，需要去看心理医生，却比任何人都会玩弄别人的心理与感情不是吗？你把尚轩托付给我照顾，把一个单身母亲的那种不容易与无奈在我面前诠释得淋淋尽致，还跟我说，你早就不对昼景有奢望了，只希望我和昼景生个孩子去救尚轩，只盼望自己儿子能健康长大，你让我对尚轩产生怜悯之心。然后，你又自杀给我。好像是我夺了你儿子，又逼得你去自杀的一样。明明犯错有罪当第三者破坏别人婚姻的人是你，最终，到头来，反倒是你让我成了罪魁祸首。季思妍，你很厉害啊！我没你那样的心机，也做不出你那样的心机。”
<br/>
<br/>        “没想到，你看得可真透彻啊！我还以为，你会是个傻女人了！”季思妍笑了笑。
<br/>
<br/>        童筱颖淡定地反问：“那么，面对这样的你，还有一心想要跟我离婚的昼景，我选择放手，你觉得，我做得对吗？”
<br/>
<br/>        “对！你做得很对！你说他是不是很坏啊？一边可以把心掏出来给你，一边又可以对你阳奉阴违。阿景不止一次跟我强调，他爱的女人是你童筱颖，可是，他却又可以继续跟其他女人上床。童筱颖，你倒是跟我说说，现在是你可悲，还是我可悲？”季思妍闭了闭眼睛，泪水滑过脸颊，自我嘲笑道。
<br/>
<br/>        “我的‘可悲’已经翻篇了，但你还没有。”童筱颖此刻已经心如止水了。
<br/>
<br/>        季思妍缓缓睁开双眼，看向童筱颖，突然间嘴角微扬地问道：“那你还爱阿景吗？”
<br/>
<br/>        “不爱了。”童筱颖回答得干脆利落，更准确地来说，是她已经当她曾经爱过的那个男人死了。
<br/>
<br/>        对，小卫已经死在了她的心里，活在了她的回忆里。
<br/>
<br/>        “阿景，你听到了吗？童小姐说她已经不爱你了啊！你还在对她痴心妄想着什么呢？”季思妍的目光落向了童筱颖的身后，忽然间得意洋洋地说道。
<br/>
<br/>        童筱颖下意识地回过头去，才看到不知何时出现在病房门口的傅昼景。
<br/>
<br/>        呵！
<br/>
<br/>        这个女人，还真会给她下套。
<br/>
<br/>        童筱颖无所谓地站起身来，傅昼景也随之走了进来。
<br/>
<br/>        她与他擦肩而过，手腕却被他给倏然捉住。
<br/>
<br/>        下一秒，她来不及反应，就被他连拖带拽地拉出了病房。
<br/>
<br/>        医院的安全通道里，傅昼景一把将童筱颖给甩到了墙壁上。
<br/>
<br/>        童筱颖吃痛地揉了揉肩膀，不打算跟他多说一句话，欲要绕道离开，结果又被他给抓住双肩，直接按到了墙壁上。
<br/>
<br/>        “我和你说一个大哥的秘密，怎么样？”傅昼景意味深长地咧嘴道。
<br/>
<br/>        童筱颖闭了闭眼睛，淡定道：“不想听。”
<br/>
<br/>        “你是不是觉得，大哥他家庭背景好，长得好，钱又多，人很好？。”傅昼景勾了勾唇。
<br/>
<br/>        “无聊。”童筱颖白了傅昼景一眼，想要挪开他的手，却被她紧紧地按着动弹不得。
<br/>
<br/>        “我跟大哥说，你的那张膜是补的。”傅昼景厚颜无耻地接着说，“大哥一定觉得，你其实是个虚伪的女人。”
<br/>
<br/>        顿时，童筱颖脸色极其难看地看着傅昼景。
<br/>
<br/>        “言归正传吧！比起你的事情，大哥的那个秘密，足矣让你心里大吃一惊了。”傅昼景接着说道。
<br/>
<br/>        童筱颖见傅昼景不依不饶，打算摘掉耳朵上的助听器，却被傅昼景伸手给抓住了手腕阻止道：“别这样啊！大哥的秘密，你这个枕边人，岂能不知道？”
