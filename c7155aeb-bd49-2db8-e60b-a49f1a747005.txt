　　第274章
<br/>　　嘟，充满干劲的少年哦，你将从这里起步，打造最强山寨，投身到乱世的洪流中，给世界留下属于你的传说。
<br/>　　又是这种中二风台词，还是熟悉的配方，还是熟悉的味道。
<br/>　　嘟，接到任务：翻修山寨。必须建筑：聚义厅，码头。
<br/>　　嘟，接到任务：完成十次打劫，目前任务进度1/10 。
<br/>　　嘟，接到任务：完成十次山寨防守，目前任务进度0/10 。
<br/>　　程大雷收到一连串的系统提示，就像每次新建一个小号进入游戏，总要去给武器铺送酒，去给药铺的掌柜采药，给绸缎庄的老板娘送首饰……然后你总会知道，原来绸缎床的老板娘和武器铺的铁匠有一段奸情。
<br/>　　这就是所谓的新手任务，难度不高，奖励也常常是粗制滥造的铁剑，白板布甲之类的。
<br/>　　不过，有些任务是新手专属的，甚至会奖励属性点之类的稀缺资源。
<br/>　　比如，第一次打劫……
<br/>　　嘟，第一次打劫任务完成，奖励抽奖次数1 。
<br/>　　嘟，山寨人口达到100，奖励二级山寨建设图纸一份。
<br/>　　嘟，击杀野生怪物白蛇王，奖励藏宝图一份。
<br/>　　嘟，成功将公主护送回山寨，抢劫压寨夫人第一阶段任务完成，获得山寨技能：建筑精通。
<br/>　　建筑精通：山寨所有建筑消耗时间减半，所有建筑附加属性提升10%。
<br/>　　又是一连串提示，程大雷撇撇嘴，脸上没有任何惊讶的表情。
<br/>　　我就知道嚒，已经是删号重练，难道还要傻乎乎听绸缎庄的老板娘念叨那些陈年往事，按部就班的去做任务。
<br/>　　当然是又快又稳，多干好省的把任务奖励拿到手。
<br/>　　“大当家，这条大蛇怎么处理？”徐神机道。
<br/>　　程大雷回过神来，想起大蛇的尸体还没有处理，蛇尸也不能一直放在这里，这却是当下最该处理的事。
<br/>　　“还能怎么办，就地埋了吧。”程大雷。
<br/>　　“怪蛇长这么大，蛇肉肯定很补的，这样埋了岂不浪费了。”
<br/>　　“反正也不能吃，我们也没人敢吃。”程大雷说着，忽然想起什么：“我们不敢吃，未必没人敢吃，不如我们用盐腌起来，卖给城里的富户，这些富户都是很需要补都。”
<br/>　　“大当家，我们没盐。”
<br/>　　大家来到蛤蟆岛，目前连个住的地方都没有，缺的不仅是盐，柴米油盐酱醋茶等生活用品一概都少。
<br/>　　程大雷让刘悲列一份清单，看看目前都缺什么，缺什么就去找什么，能抢的就抢什么，抢不到就去买。
<br/>　　接下来大家清理废墟，埋锅造饭，从水里捕上鱼来充饥，一切都是草创，还得过几天苦日子。
<br/>　　在这个过程中，程大雷将系统奖励的藏宝图从废墟中找了出来，程大雷惊喜的发现，藏宝地点就在岛上。
<br/>　　目前看来，第一桶金已经搞到手，只是还不知道宝藏里究竟有什么。
<br/>　　“大当家，你快来看，看我发现了什么。”徐神机。
<br/>　　“慌慌张张做什么，每临大事有静气懂么。”
<br/>　　程大雷说着，随徐神机往外走，看建筑的模样，应该是山寨原先的地牢。程大雷走进去后吓了一跳，只见地牢内是累累白骨，粗略看上去至少有数百具。
<br/>　　这些定是莫鸣米在这里时祸害的，还不知道这小子造了多少孽。程大雷撇撇嘴：“把他们的尸骸清理出来，在岛上埋了吧，也让他们入土为安。”
<br/>　　累累白骨如山，便在白日也有阴寒气息，不过程大雷的手下，都是刀头舔血过来的，活人他们都不怕，就更遑论是死人了。
<br/>　　地牢内的白骨被清理出来，在岛上草草葬埋。程大雷看着地牢内的环境，忽然想起什么，把怀中的藏宝图拿出来比对了一下。这下他已经可以确信，藏宝图标注的位置就在地牢内。
<br/>　　把人叫进来，开始向下挖掘，只听咚咚声响，大家挖到一层木板。将木板撬开后，地面上出现一个地窖的入口。
<br/>　　地窖内是不计其数的铜板，价值连城的金银珠宝，更多的是保存得极好的兵器盔甲。
<br/>　　看这些兵器的数量，至少能武装一支千人队的精兵。
<br/>　　有了这些兵器和财报，蛤蟆寨算是有了第一笔金，即可以把山寨现有的弟兄武装起来，日后的吃穿用度也不用愁了。
<br/>　　“高飞豹，你明日带人去城里买一些盐巴粮米回来。”程大雷道：“还有一些修房用的工具，现在咱们有钱了，也不用去抢了，省些力气。”
<br/>　　“大当家，你怎么知道地牢埋着东西？”
<br/>　　“哈……”程大雷压低生意：“是有神仙给我托梦。”
<br/>　　“嘿，也不知道谁这么蠢，把宝物埋在地牢下面。”
<br/>　　“嘿，管他呢，吃饭吃饭……”
<br/>　　在愉快活泼的气氛中，太阳落入了西山，程大雷来到蛤蟆岛的第一天就如此过去。
<br/>　　大河之水天上来，穿山越岭，浩浩荡荡流经琴川。琴川横搁在西北平原，就像是一座瑶琴。
<br/>　　整个琴川的地形是易守难攻，八百里琴川有五座关隘，而琴川关则是其中最重要的。
<br/>　　一道河流从山中流出来，逆流而上，是蜿蜒凶险的曲折道路，再往前走，地形则突然开阔起来，一道巍峨雄关就矗立在两山之间。
<br/>　　这里就是琴川关。
<br/>　　由高处往下看，只见地上蠕动着蚂蚁状的东西，视线拉近后发现，他们不是蚂蚁，而是一个个人。只是一个个都瘦得皮包骨头，身上褴褛的衣服只能勉强遮羞，他们身上竹篓里背着沉重的铁矿石，腰被压成弓字形。
<br/>　　有身着布甲的监工挥舞着鞭子，恶狠狠的抽在这些苦力身上。
<br/>　　有些人倒下后，便再也站不起来，有监工指挥人过来，将他们的尸体丢进山沟中。
<br/>　　一个略有些消瘦的中年人从房里出来，脸上表情有些憔悴。
<br/>　　“大哥，怎么了，昨晚没睡好么？”有人问。
<br/>　　“我也不知道怎么回事，刚才做了一个梦，梦见我埋在蛇岛的东西被人挖走了。”
<br/>　　“怎么可能，蛇岛有蛇王在，谁能上去。大哥，梦都是反的，说不定马上就有人来给您送钱了。”
<br/>　　“呃，有道理，有道理。”男人面色稍霁：“不过还是派人去看看，要不我总不放心。”
<br/>　　
<br/>(本章完)
