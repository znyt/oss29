　　第579章
<br/>　　程大雷装神弄鬼，当时的确唬住了一些人。不过他还未『施法』，心里委实有些犹豫，自己动用『法术』召唤，人也不是说来就来的。
<br/>　　恰在这个时候，有一伙人弃船上岸，程大雷远远看了一眼，不看还好，一看委实吓了一跳。
<br/>　　这人他还当真认识。
<br/>　　倒也不好说认识，确实是打过交道，甚至同桌饮过酒。
<br/>　　姓耿名星河，在江湖上有个匪号，唤作红胡子。
<br/>　　红胡子坐镇骷髅岛，手下有千余海盗，为祸一片海域。而红胡子本人，则伪装成来自京城的富商，在扬州千舟城坑蒙拐骗。当初程大雷还只是个青牛山小山贼时，曾与他打过交道，关系，不是特别好。
<br/>　　红胡子慌慌张张而来，确实是慌不择路才来到这座白沙岛。可抬头一看，却吓了一跳。
<br/>　　本以为是座无人的荒岛，哪里来得这么多人。
<br/>　　岛上要杀死程大雷的东海高手也是一楞：咦，程大雷的帮手。
<br/>　　红胡子身后也有追兵，几艘大船劈波斩浪而来。相比这群乌合之众组成的海盗，这群人明显有纪律许多。
<br/>　　船头一人持枪而立，身后大氅迎风，口中喝道：“来人，骷髅岛的贼人一个也不许逃了。”
<br/>　　程大雷也将他认了出来，此人不是旁个，正是李行哉。
<br/>　　李行哉得到扬州漕帮相助后，打劫海盗起家，不说屡战屡胜其实也差不了太多。此刻，他已经不是初到落凤城的那个落魄王爷，手下已经有三五千人马。有一定实力后，李行哉便将注意打到骷髅岛上。
<br/>　　谁不知道红胡子乃是海上的巨盗，这么多年，还不知抢了多少人。如果把他吞下，自己又可以暴富一笔。
<br/>　　打来打去，就一直打到此地，谁知道红胡子别处不去，恰恰逃到白沙岛上。
<br/>　　程大雷当然不知道李行哉如何来的，可在如此境地见到旧相识，程大雷颇有种老乡见老乡，两眼泪汪汪的感觉。
<br/>　　“瞧见没，瞧见没。”程大雷气愤得大吼：“这就是本当家请兵天降的本事。”
<br/>　　船头的李行哉愣了愣，心道：这无耻的声音，怎么隐隐有些熟悉。
<br/>　　“李行哉救我！”程大雷扯着大吼：“你妹妹可在我手上，敢不来救，你们以后就别见面了。”
<br/>　　李行哉一激灵：果然是此贼，难得啊，程大雷还有求自己的时候。
<br/>　　他立刻开始下令，命令简单直接：上岸，救人。
<br/>　　夫子和孟玄青都是一惊：程大雷果然会妖术。
<br/>　　二人看程大雷的眼神难免有一抹震惊，心中均道：此人气数不小呐。
<br/>　　燕不归可不管你气数不气数的，挥舞着重剑杀来。杀一个是一个，杀两个够本。
<br/>　　程大雷本就不怕他，更何况现在。刚才他不过是条鲶鱼，扑腾着想要弄出点水花来，然而现在，他已经有了鲤鱼跃龙门，反败为胜的机会。
<br/>　　孟玄青与夫子也同时杀了过来，孟玄青一把老骨头，早已将生死置之度外，而夫子虽然还贪生，但他火爆脾气，本就是不怕死的。
<br/>　　程大雷的剑越来越犀利，他心中明白，只要撑过这一刻，等李行哉的人过来，那么胜局就是自己的。
<br/>　　李行哉的人马已杀上了岛，穷途末路的红胡子海盗，自然挡不足他们的冲锋。几轮箭雨，便送他们见了阎王。唯有红胡子一人，仗着身上武艺，倒是逃得一条性命，却也是分外狼狈。
<br/>　　“吾乃落叶城逍遥王，你们是哪里来到乱臣贼子，速速报上名来。”
<br/>　　李行哉手持大枪，以枪杵地，倒也是威风赫赫。
<br/>　　东海一地的江湖豪强，看到他们身上的盔甲便失去了战意。民不与官道，还是乘早开溜的要紧。而书院的弟子，更是自幼读圣贤书，所信奉的是为国为民做事，他们的信仰可不是当反贼。
<br/>　　所以，李行哉迅速的稳住了形势，唯一要做的一件事，便是诛杀红胡子。
<br/>　　这确实也暴露了李行哉目前的软肋。在得到漕帮相助后，他洗劫海上的海盗，很是发了一笔财。一旦有了财，想要招兵买马并不是难事。但千军易得一将难求，他缺少冲锋陷阵的猛将，戚继光虽然有帅才，但是并无将才。
<br/>　　如今，也只能发动人海战术，先灭了红胡子要紧。
<br/>　　“程当家，不要紧么？”李行哉远远看了一眼。
<br/>　　程大雷没有回答他，目前的确撑得住，但却也没有说话的力气。他和李莫愁独斗孟玄青三人，渐渐占了上风。
<br/>　　红胡子也是慌不择路，此刻竟向山头冲来，误打误撞，闯入了五人的战团。可这五人，都是世间少有的绝世高手。每人至少有一手看家本事，李莫愁的冰魄神针，燕不归的重剑术，孟玄青的毒术，夫子的书生意气剑，而程大雷的一手快剑，也是可以在整个天下排得上名气的。
<br/>　　他闯入战团，几乎和送死差不多，只能说他走错路了。
<br/>　　五人的兵器齐刷刷的落在他身上，红胡子连一声惨叫都来不及发出，性命便交待在了此地。
<br/>　　他怀里骨碌碌滚出一样物事，哧哧冒着火花，众人都是一激灵。程大雷猛然醒悟这是什么：霹雳雷。
<br/>　　这红胡子身为海盗，在海上抢劫来往客商，免不了有异域客商携带一些稀奇古怪的物事。这枚霹雳炸雷便是其中之一。
<br/>　　原来这耿星河不是慌不择路，而是想玩同归于尽。可你想要同归于尽，该去找李行哉啊，和我又有什么关系。
<br/>　　程大雷却是忘了，耿星河的弟弟就是死在他手中，耿星河也是将程大雷认了出来。
<br/>　　“小心啊。”
<br/>　　程大雷一脚将霹雳雷踢向空中，拽着李婉儿往后撤。这时候，李莫愁手中一蓬银针出手，射向夫子。夫子的竹中剑刺向燕不归，燕不归重剑砍向李莫愁，而孟玄青也攻向李莫愁。
<br/>　　轰！
<br/>　　霹雳雷在天空中四散炸开，铁弹横飞，程大雷因为知道这火器的威力，没有遭到波及。
<br/>　　而其他四人却没有程大雷这样的好运气，等爆炸的余波散去，程大雷看着眼前这一幕，突然呆住。
<br/>(本章完)
