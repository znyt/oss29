        拍摄期间，知名女演员亲自点名感谢洛羽的关照，夸他是个超级暖心的小哥，若不是他挺身相救，被划伤手臂的那个人就是她。
<br/>
<br/>        就连剧中其他两位大咖级别的男演员也纷纷在微博上亮照与洛羽“称兄道弟”，还大赞洛羽演技不错，与他对戏很容易进入状态。
<br/>
<br/>        洛羽在网上的好感度突然飙升，而且连着一个星期的粉丝涨幅度都排在了第一位，连带着《联盟》这部电影也跟着未播先火。
<br/>
<br/>        而后援群里的粉丝也一个个为洛羽感到骄傲。
<br/>
<br/>        连星儿为此自然也高兴，高兴的同时，洛羽主动跟她联系了。
<br/>
<br/>        星儿，三天后，你有空吗？我那天要去滨江城赶一个通告，工作完后，可否跟你见个面？——小纽扣。
<br/>
<br/>        有空有空，随时都有空！——连星儿。
<br/>
<br/>        那我们三天后随时联系。——小纽扣。
<br/>
<br/>        看到这里，连星儿抱着洛羽的公仔，兴奋不已地在床上打滚。
<br/>
<br/>        三天后，连星儿一大清早起来就缠着童筱颖，要她教她做曲奇饼干。
<br/>
<br/>        童筱颖倒是手把手地教了连星儿，同时也好奇地问道：“你这饼干是做给你爹地吃的？”
<br/>
<br/>        “当然有爹地的份！”连星儿手里拿着一把粉色刮刀，一边和面，一边甜甜地笑道。
<br/>
<br/>        童筱颖忍俊不禁地说：“我看你是有心上人了吧！是不是那个在北城照顾你的那个小哥哥？”
<br/>
<br/>        “才不是！”连星儿立马否认，话说回来，她回家的这些天里，明司寒并未主动联系过她，当然，她也不会娶主动联系明司寒。
<br/>
<br/>        因为，她感觉自己跟他没什么话题好聊。
<br/>
<br/>        “难道是洛羽？”童筱颖不禁狐疑。
<br/>
<br/>        连星儿更加只笑不语，而且笑容还特别地灿烂。
<br/>
<br/>        童筱颖脸上的笑容渐敛。
<br/>
<br/>        洛羽是明星，一般来说，一个明星不会对一个普通的粉丝上心。
<br/>
<br/>        那次，她和女儿通电话，听女儿说和洛羽同住一家酒店，她就觉得有些奇怪了。
<br/>
<br/>        明星隐私行踪，是不会透露给粉丝的吧！
<br/>
<br/>        “你今天是要去见谁？”童筱颖若有所思地问。
<br/>
<br/>        连星儿敛了敛脸上的笑容，一本正经地回答：“见我的一个好朋友。”
<br/>
<br/>        “星儿。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “我觉得你爹地说得对，洛羽他并不适合你。”童筱颖有些敏感地说。
<br/>
<br/>        连星儿脸上的笑容顿时全无，没好气地应了声：“我只是洛羽的粉丝，对洛羽的喜欢，只是出于粉丝对偶像的那种喜欢。”
<br/>
<br/>        童筱颖见自己多说无益，而且女儿连星儿还一脸不乐意她的好心劝告，只好就此闭了嘴。
<br/>
<br/>        曲奇饼干做完后，连星儿还特意挑了一个最好看的饼干盒，将放凉好的曲奇装好，又精心打扮了一番才出的门。
<br/>
<br/>        童筱颖站在露台上，看着连星儿所乘坐的私家车离开院门口后，私底下，给傅夜沉的保镖队队长打了通电话。
<br/>
<br/>        “星儿已经出门了，这次你多派些人手跟着，除了保护星儿的周全以外，你们还要阻止那些暗中偷拍的‘狗仔’。”
<br/>
<br/>        “……”
<br/>
<br/>        “对，我怀疑星儿是跟洛羽在约会，洛羽是明星，一点风吹草动，必定会惊动媒体记者。我不希望星儿被媒体曝光出来。”
<br/>
<br/>        “……”
<br/>
<br/>        “你不用跟大哥汇报，这件事情也不用跟大哥说，星儿感情上的事情，我们强行插手，只会适得其反。”
<br/>
<br/>        童筱颖挂了电话后，看着逐渐消失在视野里的车尾，不禁微微皱起了眉头。
<br/>
<br/>        连星儿如约赶到了洛羽所说的见面地点，滨江城里一个地处比较偏僻的咖啡馆。
<br/>
<br/>        咖啡馆被各种绿色植被与花卉点缀，走进去有点像是进了花店的错觉，有几个顾客就直接坐在了店外的木质露台上喝咖啡。
<br/>
<br/>        服务员将连星儿领到了咖啡管内一个不起眼的角落的雅座里。
<br/>
<br/>        连星儿看到桌子对面坐着的人，差点没认出是谁。
<br/>
<br/>        “洛羽？！”连星儿试探性地问。
<br/>
<br/>        坐在对面的人，这才摘下自己脸上的墨镜和口罩，但头顶仍旧戴着鸭舌帽遮挡了大半张脸。
<br/>
<br/>        “你平常都是这样出门的呀？”连星儿忍俊不禁道。
<br/>
<br/>        洛羽苦笑着点了点头，打趣着说：“以前，我其实犯不着如此，只是最近这段时间，不知怎么着就有种一夜爆红的错觉。”
<br/>
<br/>        “这不是好事吗？”连星儿微笑着说。
<br/>
<br/>        确实是好事，他现在也算得上是通告不断的小明星了。
<br/>
<br/>        “我给你点了卡布奇诺。”洛羽说着，服务员就把连星儿的咖啡给端了上来。
<br/>
<br/>        其实，她并不大喜欢喝咖啡，但她也没因此把自己的喜好摆到台面上来，让洛羽尴尬。
