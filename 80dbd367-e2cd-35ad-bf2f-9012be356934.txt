　　第682章
<br/>　　程大雷、刘悲、阿喜、吴用四人聚在凉州城的大厅上，商议凉州城现在的情况。
<br/>　　另外还有一头徐神机，但最近他发言的次数越来越少了。程大雷为他立下高人的人设，他也觉得，作为一个高人，不遇大事不值得自己发言。
<br/>　　但每次开会，还缺他不可，毕竟，他可是蛤蟆城的第一谋士。
<br/>　　这次会议重中之重，还是商议该以如何态度面对武威浩甲金城三座城。
<br/>　　程大雷道：“吴军师，你先介绍一下他们的情况？”
<br/>　　吴用点点头站起，道：“浩甲城的城主名叫钱通玉，背后是京州钱氏，他算是崔相的远方亲戚，与相府的纠葛颇深。武威城的城主名叫白空集，与宋伯康是结义兄弟，另外金城的城主名叫孙别山，他倒是寒门子弟，但也拜在相府门下。”
<br/>　　程大雷咧咧嘴，算起来都是相府的人，那么将他们拉拢过来的可能性几乎为零。日后迟早还是有刀兵相向的一天，只是程大雷希望这一天来得稍微晚一些。
<br/>　　“递给他们的信送过去没有，他们什么态度？”
<br/>　　“遵照大当家吩咐，信已经送到。我们的人也平安离开，还是他们亲自送出来的。只是他们的态度，现在也是摸不透，也说不清楚他们是要打还是要和。”
<br/>　　程大雷面向阿喜：“查得什么消息么？”
<br/>　　阿喜站起身：“属下惭愧，三番两次派人潜入，都没得到确切消息。属下着力去做，定在最短时间内，查到可靠的消息。”
<br/>　　程大雷点点头：“好好去办，查不到什么也不必自责，或许，他们根本没有态度。”
<br/>　　这件事不解决，终究是悬在程大雷头上的一把刀。真落下也未必可怕，关键是它一直不肯落下。
<br/>　　目前要做的，第一是稳住目前的局面，二就是查明其他三家的态度，程大雷也算知道该如何应对。
<br/>　　“禀报大当家，属下还有一事禀报。”吴用开口道。
<br/>　　“什么事？”程大雷本以为今天会议已经算是结束，谁知道吴用又突然站起来说话。
<br/>　　吴用清了清嗓子，道：“如今大当家已经占了凉州城，取凉州王而代之，大当家的称号是不是也要改一改，再将自己当做山贼，是否有些自降身份？”
<br/>　　“啧？”程大雷倒是没想到这一点，问：“你如何想法？”
<br/>　　吴用肯定是已思虑良久，他开口道：“该选个良辰吉日，大当家直接称王，名号可以再议，但需要有符合一路诸侯王身份的存在，不应再想从前一样随便。”
<br/>　　程大雷手指敲着桌面，脑海中不停思索。如吴用所说，自己现在称一路诸侯王是的确够格的，所谓名不正则言不顺，言不顺则事不成。有了名份，做起事情也顺手起来，这也不仅仅是为自己考虑，也为手下弟兄们考虑。拼死拼活打天下，他们自然也想获得些什么。
<br/>　　这是好处，当然也不是没有坏处。称王之后，就是摆明了造反，长安朝廷就不会放过自己了。而现在还没到坐享胜利果实的时候，程大雷也担心将他们的身份抬上去，他们便失去了奋进之心。
<br/>　　思索良久，程大雷摇摇头道：“目前凉州未平，浩甲、武威、金城是卧榻之贼，他们不除，谈什么称王。这件事等平掉三家势力后再议。”
<br/>　　吴用有些失望，但也没有再劝说程大雷。
<br/>　　今次的会议就此散去，程大雷晃了晃肩膀，除了疲惫之外，还有几分快意。他心中难免也幻想一下，自己成一路诸侯王后是怎样情况。身边美女如云，莺歌燕舞，也算是人间乐事。
<br/>　　当然，这也仅仅是幻想一下而已。现在帝国的局面很乱，明显不到享受的时候。
<br/>　　刚散会，程大雷打算去城中转转，看有什么事情自己顺便解决了。
<br/>　　正这时候，高力士急匆匆而来，道：“大当家，属下有一件事禀报。”
<br/>　　程大雷与高力士见面的机会并不多，对宦官这种存在也很难有好感。不过，对方也算自家兄弟，程大雷也不会特别对待。
<br/>　　“嗯，出了什么事？”
<br/>　　如今，这种事程大雷已经习惯了，每天几十件事找上门，假若那天没有事才会令程大雷觉得奇怪。
<br/>　　高力士弓着腰，道：“大当家，那些女眷如何处理？”
<br/>　　“什么女眷？”
<br/>　　“便是凉州文武官员家中的丫鬟、歌姬、乐班还有他们的夫人小姐，这些人该如何处理，还得请大当家定夺？”
<br/>　　程大雷这几天忙得不可开交，他倒是把这件事忘了。那白玉美人可也被自己擒住，之后，程大雷还没同她照过面呢。
<br/>　　城破之时，文武官员逃走不少，有些腿脚快的，就带上了老婆孩子，腿脚慢的，就把老婆孩子舍下了。
<br/>　　这也是没办法的事，毕竟自己性命要紧，老婆可以再娶，女儿可以再生，有什么会比性命重要。
<br/>　　正好今日稍微空闲些，程大雷一撩袍袖，道：“走，带我去看看。”
<br/>　　高力士在前面引路，程大雷跨出城主府，一路也看着两边建筑。令程大雷高兴的时，街上已经有了行人，有些商铺饭馆开始营业。
<br/>　　这证明凉州城的人已经接受程大雷的存在，开始适应新的生活。
<br/>　　第一，程大雷下了严令，不许侵扰百姓，蛤蟆城的兵入城之后，虽然免不了有几个办下恶事，但都被程大雷用雷霆手段严惩。
<br/>　　所以凉州百姓觉得，这程大雷为人还算可以。
<br/>　　第二，是之前程大雷四处散播谣言起到作用，万一程大雷当真是天命之子呢，这也是说不好的事情。
<br/>　　人总是要活着，无论头顶是谁，他们总要活下去。
<br/>　　这算是程大雷今天遇见的第一件喜事，经历这场风波后，凉州城正在慢慢复苏。而程大雷觉得，自己比宋伯康强一些，日后，凉州城会越来越好的。
<br/>　　嘟，恭喜占领凉州城，获得抽奖次数3 。
<br/>　　嘟，获得城主称号，奖励江山图一副。
<br/>　　嘟，接受任务：消灭凉州其他势力。
<br/>　　随着一连串的系统提示，直到现在，程大雷算是正式占领凉州，成了此地的主人。
<br/>(本章完)
