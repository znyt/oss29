　　叶少阳随后想到，这两个土匪被自己放过，未必就是好事，或许结果更遭——今晚上洞房花烛夜，那个人形煞肯定会展开行动，到时候山寨上能逃出去几个人，还真不知道，也许这两个家伙就是其中之一，这就看他们的命了。
<br/>　　等他们走后，叶少阳继续上山，一路走到山寨附近，仍然悄悄绕到之前那个位置，爬上围栏朝里面看，立刻就看到那间“大瓦房”房门大开，里面张灯结彩的，从他这个角度，正好能看到一长排的桌子，从房间的最里面一直延伸出来，排列到房门附近，桌上已经摆了一些凉菜，有土匪们在大厅里走来走去得忙活着。
<br/>　　叶少阳等了很久很久，一直到天完全黑透了，总算看到有两个人从寨门方向打着火把走近，手里提着几个包袱，一个身上挂着一长串鞭炮。
<br/>　　叶少阳猜测这两个就是之前下山的两个，采购好东西之后又回来了。
<br/>　　这两人把鞭炮摆在大厅门外的地上，进屋去交任务，然后一行人布置起来，在大厅的横匾上挂上红花，叶少阳想到白天相同的位置还挂着白花，当时这道门还出过棺材，这才办完丧事不到几个小时，居然又办起了喜事，这些土匪也真是可以，不觉得瘆的慌。
<br/>　　在靠近最里侧的一张桌子前面，像模像样地摆了一对红烛。然后那个大块头的大当家鬼见愁，胸前缠绕着红花，大咧咧地坐上去，左侧的侧面坐着那个穿锦衣戴帽子的，叶少阳猜测应该就是那个二当家铁算盘了。
<br/>　　山寨的人不少，大厅里面坐不下，在外面空地上也摆了几十张桌子，上百个土匪从寨子后面的一排排的建筑物里走出来，寻找着座位。
<br/>　　看着这一幕，叶少阳不由的想起了自己在大学里放学时候、所有人一起去食堂吃饭的情景，还真是挺像。而且他觉得这些土匪之中，大多数八成也是几个人住一个房间，跟大学宿舍似的，因为光是这么看上去，山寨上的建筑物根本不够几百号人一人住一间。
<br/>　　这些人入席之后，就开始上热菜，叶少阳看着这些上菜的家伙，也是土匪打扮，估计是找了些土匪临时充当厨子伙夫。
<br/>　　开席之间，大伙都站起来，拿起面前的坛子，把桌子上的酒都倒满，然后铁算盘开口说话，因为距离太远，叶少阳完全听不清说的什么，想来也八成是充当主持之类的角色，果然等他说完，众土匪们一起起哄起来，然后鬼见愁对身边一个伺候的小土匪说了什么，这小土匪离席出去，绕到后面那排建筑物前面，因为那边没灯，叶少阳看不清楚在干什么，好像是敲门之类的。
<br/>　　过了一会，就看见几道人影走到了大厅这边，是两个女子，手中扶着一个身穿旗袍的姑娘，脑袋用红盖头遮住，缓缓走过来。
<br/>　　叶少阳隔着老远，一眼就看出了她身上的煞气，人形煞，到了夜晚，她身上的煞气果然已经出现了。不过她没有忙着动手，叶少阳猜测，她估计是想等一个合适的机会，不想造成太大的骚动，不然逃走的人会相对的多，吸血也吸不过瘾。
<br/>　　那两个扶着她的姑娘，叶少阳怀疑，估计也是哪个当家的娶来的媳妇，充作化妆师和伴娘。
<br/>　　这几人走进大厅之后，那些土匪们立刻就起哄。两个姑娘扶着新娘子坐到鬼见愁的边上去，然后在众多土匪的簇拥起哄之下，开始拜天地之类的。
<br/>　　叶少阳冷笑着看着这一幕，心想这些土匪也是挺会玩的，还真以为找了个貌美如花的压寨夫人，现在闹得欢腾，等到真出事了，这里现在推杯换盏的人里，不知道还能有几人存活。
<br/>　　叶少阳百无聊赖，开始观察这些土匪，突然间发现，坐在鬼见愁左手边的是铁算盘，右手边坐着的那一排，却好像是穿着军装，而且最靠前的一个，似乎地位很高，鬼见愁不时跟他说话，互相敬酒。
<br/>　　叶少阳很好奇，土匪窝里怎么还有军人，转念一想，这年头兵荒马乱，也就可能是逃兵之类的，没地方去了，所以落草为寇。
<br/>　　猛然间，他似乎想到了什么，不会……真的这么巧吧？
<br/>　　叶少阳睁大了眼睛，朝那几个穿军装的望过去，但是因为距离太远，根本就看不清楚长相。
<br/>　　看着这些人大口吃肉，大快朵颐，自己只能在这趴着等着，虽然吃过饭了，叶少阳肚子还是有点咕咕叫起来。好在鬼见愁并没有呆多久，跟那女尸拜天地之后，陪着兄弟们喝了一会酒，就急不可耐地抱着新娘子入洞房去了。
<br/>　　两人的洞房，就在大厅后面的那一排房间的头一个。
<br/>　　叶少阳看着鬼见愁抱着新娘子、一摇一晃地走进房间，就知道这个家伙凶多吉少了，不过他一点也不可怜这个手上沾满鲜血的土匪头子，他现在关心的是，找一个最恰当的时机，前去除掉那个人形煞，现在这情况，自己如果跳进去抓鬼，不用说一定会被这些土匪打成筛子。
<br/>　　只有等待。
<br/>　　又坚持了半个小时左右，这些土匪大部分都喝的醉了，大部分还在喝，有的已经东倒西歪地往自己的卧房走去，还有一些跑去偏僻的地方小便。
<br/>　　叶少阳感觉差不多了，于是翻过围栏，跳进了寨子里，潜伏在靠山野这边的林子里，等了一会，有一个家伙哼着歌过来撒尿，叶少阳冲上去，就像电视里经常演的那样，照着后脖颈就是一掌，直接干晕，然后拔了外衣披在自己身上，再把这个醉鬼扔进草丛里去。自己也在里面等了许久，感觉外面没什么动静了，出去看的时候，只有少数几桌人还在吃喝划拳，其余的大部分都散了，也有一些人趴在桌子上就睡着了，有的在桌子下面吐。
<br/>　　叶少阳悄无声息地朝之前鬼见愁抱着鬼媳妇去的那座建筑物走过去，想着那鬼尸也不知道会不会跟鬼见愁啪啪啪，然后再吸血，不过鬼见愁必然是死定了。
