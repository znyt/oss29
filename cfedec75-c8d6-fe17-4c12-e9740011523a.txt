　　这些煞气在她体内蛰伏，缓慢地涌动着。
<br/>　　有煞气，也只能证明她是尸煞，并不能确定是哪一种形态的尸煞。
<br/>　　叶少阳想了一下，打开自己鼓囊囊的背包，摸出了十八神针的针盒。
<br/>　　张道长和青冥道长两人在边上看着他把针盒打开，看到里头大大小小的金灿灿银闪闪的细针，不知道是什么名堂，感到十分新奇。
<br/>　　叶少阳摸出一根金针，一只手捏开了女尸的嘴巴，把针头刺进了她的舌根处，用两个指头撑着嘴，不一会工夫，从女尸口中流出了一股绿色的粘液，顺着下巴流出来。
<br/>　　叶少阳把金针从舌头上拔下来，原本金色的针头，也被绿色的液体所染，叶少阳点燃一张灵符，烘烤针头，液体被蒸发之后，发出一股异香，令张道长和青冥道长闻到之后，都是心中一颤，感觉不妙，警惕地朝后面退了几步。
<br/>　　叶少阳把金针弄干净，收起来，抬头迎着二人疑惑的目光，说道：“这是人形煞，人死后不僵，面皮如常人，幸亏你用定尸符贴住她，体内煞气没法生发，不然灵符一除，煞气外泄，立刻就会成煞，比一般的僵尸要猛的多，不惧阳光，浑身散发香气，闻者迷醉，想跑都跑不动。”
<br/>　　青冥道长听他这么说，惊得倒吸一口冷气，看着躺在地上的女尸，说道：“叶天师，这什么人形煞是怎么形成的？”
<br/>　　“我也是第一次遇到，之前在书上看到过，好像说是要无伤而死……”叶少阳挠着头皮，“好像还有什么条件，几件事凑巧在一起，才能形成人形煞，但是别的原因我忘了。”
<br/>　　叶少阳很有点不好意思，不过这件事也不能怪他，灵异方面的知识浩如云海，就算是他也不可能完全记住，不过最重要是，这人形煞并不算是特别难对付的邪物，对这种一般性的邪物，他也不可能记得住每一种形成原因，用金针一刺，就能判断出是人形煞，已经很不容易。
<br/>　　“这种人形煞，最大的特点就是身上很香，让人迷醉，同时形态如人，如果有法师操控祭炼，可成为傀儡……这具尸体生前是什么人，怎么来的？”
<br/>　　青冥道长有些委屈地说道：“说起这个，还真是怪的很，这个女尸是我们一个钩子从山下永安县城里找来的，当时是一个男子找来，声称这女子是他小妾，因发急病而死，想带回老家安葬，但是兵荒马乱，很多地方马车不通。而且路途遥远，回去就烂了，所以找到钩子，给了一笔挺大的报酬。
<br/>　　干我们这行，有三杀不赶，自杀不赶，凶杀不赶，官杀不赶。想必叶天师也是听说过的。”
<br/>　　叶少阳点点头，这种说法的确是有：自杀的人，魂魄容易流连人间，凶杀的怨气重，所谓官杀，就是犯了死罪被砍头或者绞刑，这样的死者戾气重，总之这三种人都是最容易发生尸变的，符咒不容易压得住，因此赶尸的一般情况下不会赶这三种尸体。
<br/>　　青冥道长接着说道：“钩子当时也看了，这具女尸身上无伤，没任何问题，所以就接了。后来我们发现尸体有问题，查验发现，尸体全身上下没一处伤口，也没有发病的模样，不知这是为何？”
<br/>　　叶少阳道：“我之前也说了，人形煞必然是无伤而死，无伤而死的办法有很多，例如‘贴天面’就可以做到。”
<br/>　　所谓贴天面，就是把人绑好，然后用羊皮纸洗水，贴在人的脸上，羊皮纸吸水收紧，随着人吸气不断急促，会越来越紧，最后把人活活憋死，身上不会有任何伤口。
<br/>　　叶少阳早先回家调查婴煞的时候，曾遇到叶小烁的母亲三娘，就是遭遇过，不过她被用的是黄裱纸，要贴十几张人才能憋死。
<br/>　　“照这么说，这还是个有钱人家的小妾。”叶少阳朝女尸身上看去，见她身穿一件深红色的修身旗袍，两条雪白光洁的大腿露在外面，虽然是死人，但因为外表栩栩如生的缘故，看上去还是很诱人，再加上两团鼓鼓的胸脯……叶少阳忍不住多瞟了几眼。
<br/>　　青冥道长说道：“我们看她是没有外伤，叶天师也可以脱了她的衣服，好生检验一番，看是否有遗漏。”
<br/>　　脱了衣服检查一边……叶少阳偷瞄了一眼女尸的身材，暗自咽了下口水，说道：“既然你们都查验了，也不至于连外伤都看不出来。而且这个也不重要了，反正人尸体都在这了，你们也不可能带她去报官。”
<br/>　　张道长道：“俗话说死无对证，报官也没用，再说现在都打仗，衙门谁理会这个。”
<br/>　　叶少阳道：“话虽如此，但这尸体必然有问题，极有可能是被人所杀。”想了想，又问青冥道长：“找你们赶尸的那个人，有没有什么特别交代？”
<br/>　　青冥道长想了想，说道：“也没什么特别的，给的报酬是一般赶尸的十倍，让我们一定要把尸体送到他说的那个地方，唯一的要求就是，一定不要让尸体任何损毁。”
<br/>　　“不要尸体损毁？”叶少阳皱眉问道。
<br/>　　“对，他说时候那边有家属接收尸体的时候，会再给我们一笔钱。”
<br/>　　叶少阳想了一下说道：“如果你信任我，这单生意交给我，我是看张道兄的面子来帮忙的，我不收你一分钱报酬。到时候我送尸体过去，看看是什么人来接收。”
<br/>　　张道长皱眉道：“叶老弟的意思是？”
<br/>　　“我怀疑是有人杀人用来邪修，或者有别的什么原因，总之我要调查一下。”虽然这不是自己的世界，虽然自己现在法力只剩下一成半，但叶少阳还是有一种人间天师的责任心，遇到这种事情，还是想要调查清楚。
<br/>　　青冥道长对叶少阳拱了拱手，笑道：“叶天师展示了这么多的手段，若我还是不服，那可真是小肚鸡肠了。叶天师，青冥子服了！”
<br/>　　叶少阳听他说到最后，突然怔住了。
