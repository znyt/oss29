　　称心的府邸之中吴琦正在向他汇报着城中的情况，包括来了些什么人，都是什么身分，还有那结异族都有什么人，分属哪些国家，在国内的地位是什么样的。
<br/>　　最后把所有的情况都说完了，才谨慎的说道：“大人，这一次来的异族太多了，我们的力量根本不足以盯着他们，您看要不要向上百请示一下，再调一批人过来？”
<br/>　　“不必。”称心摇了摇头：“那些异族多数都是掩人耳目的，如果你把注意力全部放在他们身上，估计这次就真的栽定了。”
<br/>　　“可我们总不能放任不管啊，那些异族这次来了这么多，显然也是不怀好意。”吴琦有些担心的再次说道。
<br/>　　“来者不善，善者不来啊！”称心感慨的笑了笑：“这次我们的对手显然不是一般的人物，如果你千万不能有一点大意，否则保管你怎么死的都不知道。”
<br/>　　“大人，小人不明白，不就是一些异族么？还有，那个刘洪、唐义，只要把他们抓起来一问什么事情都清楚了啊。”吴琦还是坚持着自己的观点，认为把那两个已经暴露的家伙抓起来才是最靠谱的事情。
<br/>　　但显然他并不能改变称心的主意，这位‘红衣’大人绝对是位有主见的人物，不会被自己的手下所左右，在听了他类似于牢骚的表达之后摇了摇头：“我记得上次已经跟你说过，暴露出来的奸细便已经不是奸细，他们只是一些诱饵，你完全没有必要把注意力集中在他们的身上。”
<br/>　　“可是除了他们两个，我们还能去监视谁呢？科学院里人那么多，我们根本没有那么多的人手一个一个守着他们啊。”吴琦有些郁闷的说道。
<br/>　　他是真的不理解，为什么都这个时候了，这位‘红衣’大人还是这样不紧不慢，难道这位‘红衣’大人只是浪得虚名？实际上根本就没有那个能力？
<br/>　　“谁说我们一定要去找他们？难道你觉得大唐真的只有一股势力在惦记着科学院里的东西？如果你真的这么想，未免有些太天真了，我会向黑子建议另外派人过来，将你替换掉。”
<br/>　　“大人……”吴琦露出一个难看的笑容：“大人这是说的哪里话，小人对大人忠心耿耿……”
<br/>　　“行了，你不必在这里对我表忠心，这件事情我已经另外有了安排，你只要按照我的吩咐去做就好。”称心摆了摆手，示意吴琦不必再说。
<br/>　　而就在吴琦有些尴尬的不知说些什么才好的时候，外面突然风风火火闯进一个人来：“‘红衣’大人，吴大人，不好了，出大事儿了！”
<br/>　　“什么事？慢慢说！慌慌张张成何提统。”吴琦不等称心开口，便一把将来人扯住。
<br/>　　不过，来人显然被发生的事情吓坏了，根本安静不下来，急吼吼的说道：“大人，出大事儿了，唐，唐大人在来咸阳的路上被人袭击，随身携带的图纸全部被劫！”
<br/>　　“什么？”吴琦的脸色瞬间就变了，扯着来人的手猛一用力，将那人整个提了起来：“你可知道你刚刚在说什么吗？！”
<br/>　　“大人，小人不敢说谎，一切都是千真万确，眼下唐大人受了重伤，正在被送往医学院的路上。”来人脸色涨的通红，但却依旧没有改口。
<br/>　　“这，这可如何是好！”吴琦这下子可是彻底乱了手脚，将自己的手下丢到一边，快步来到称心的面前：“大人，小人建议立刻将消息送回长安，另外和情报科申请人手，加派力量全力捉拿贼人。”
<br/>　　称心的面色也有些凝重，似乎同样处于震惊当中。
<br/>　　千算万算独独忘了对方会不会孤注一掷，眼下唐善识被重伤，他所携带的资料全都被人劫走，这已经不是称心所能控制的局面了。
<br/>　　要知道，唐善识那可是大唐科学院的最高负责人，他随着带着的资料又岂能是简单的？
<br/>　　另外还有一点就是唐善识来回往返于长安与咸阳身边身然不可能没有人手保护，可就算是这样依旧让敌人得手了，由此可见敌人的实力似乎不小。
<br/>　　只是现在让称心有些奇怪的是，唐善识往来于长安和咸阳并没有指定的路径与时间，敌人到底是如何掌握他的动向，并如此准确的下手袭击的呢。
<br/>　　“大人，大人！”吴琦叫了称心数声，终于将他从沉思中唤醒，在看了几个手下一眼之后叹了口气说道：“吴琦，就按你说的方法办法，从长安那边调人，另个派人去城门口、车站等地方守着，严查所有非咸阳本地人口。”
<br/>　　“是，小人这就去办。”吴琦得了命令，如飞般跑了出去。
<br/>　　而称心则是看着那个刚刚来报信的家伙继续问道：“现场还有没有伤亡？敌人有没有留下什么有用的线索？”
<br/>　　“回大人，现场被打扫的很干净，什么都查不出来。唐大人的护卫四人重伤，六人轻伤昏迷，还有两人失踪。”
<br/>　　“失踪？”称心的眉头紧紧皱了起来：“周围可都查过了？”
<br/>　　“查过了，方圆五里之内都查遍了，没有那两人的踪迹。”来人努力回忆了一下，最终摇头说道。
<br/>　　“那你速速派人去查一下这两个人的身份与背景，重点查一下这两人最近在与什么人接触。”
<br/>　　“是，小人这就去办。”来人此时已经猜出了称心是什么意思，连声答应着迅速离开，只留下称心一个人在房间中皱眉苦思。
<br/>　　到底是谁有这么大的胆子，竟然敢袭击大唐科学院的负责人？
<br/>　　又到底是谁能这么大的能力，竟然能够策反那些忠心耿耿的卫士？
<br/>　　眼下一个不明势力已经够特么让人挠头的了，现在又出现了另一个，这难道就叫福不双至祸不单行？
<br/>　　又或者这次的袭击与派来刘洪、唐义两人的势力是同一个？这个势力已经意识到自己的存在，知道无法通过‘和平’的手段拿到自己需要的东西，所以才会孤注一掷？不断推演着一切的称心只觉得自己的脑子都快要炸开一般。
