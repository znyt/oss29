　　心动不如行动，有了决定之后，李承乾立刻返回自己的书房，同时下令将苏玫、孔雯、程琳还有小武等几个大小老婆叫了过来。
<br/>　　几个败家娘们儿刚刚被长乐一顿教训，各自回去没多长时间，又被李承乾叫到书房，一个个神色尴尬的很，扭扭捏捏有些不些不知所措的样子。
<br/>　　“好了，别一个个愁眉苦脸的了，让人叫你们过来不是给你们上课的。”李承乾看着苏玫等人忐忑的样子，多少也有些心疼，摆手示意她们各自坐下之后再次开口说说道：“我这次找你们来，若说与你们争后之事无关也不现实，不过在我看来你们争的方式错了。”
<br/>　　“是，臣妾等知错。”苏玫以原太子妃的身份起身说道。
<br/>　　“罢了，过去的事情就过去了，你们想必也知道，我是一个重感情的人，所以我不希望自己的家里闹的不可开交。”李承乾看了苏玫一眼。
<br/>　　几个女人全都低着头，显然还是有着各自的打算，口服心不服的态度。
<br/>　　李承乾看着眼前的几个女人，头不自觉得又开始大了起来，但事情总要解决，就算是头大也要解决，于是只能硬起心肠说道：“这一次短时间内朕不打算封后，宫里切官制按父皇制定的标准执行。”
<br/>　　言罢，也不等几女有所表示，便看向苏玫：“汝可为贵妃！”
<br/>　　“喏！”苏玫眼中闪过一丝黯然，但没有后位，得了贵妃似乎也不错，至少在没有皇后的时候，她还是最大的。
<br/>　　“孔雯！”李承乾点了孔雯的名子，但显然在某一个瞬间他有了那么一丝犹豫，不过这个时间很短，短到只是顿了一顿，然后便继续说道：“汝可为淑妃。”
<br/>　　“喏！”孔雯眼中闪过一抹喜意，同时那份忐忑也淡了一些。
<br/>　　“程琳，汝可为德妃。”
<br/>　　“喏！”
<br/>　　“武媚，汝可为贤妃。”
<br/>　　“喏！”
<br/>　　只一下子，李承乾便将四妃之位全都封了出去。
<br/>　　唐有四妃，一贵、二淑、三德、四贤，皆为一品，旨在辅佐皇后，而后位若有空，则四人共治后宫。
<br/>　　这四妃的地位虽高，但却不足以影响什么，所以皇帝一般自己就可以封赏，不似皇后一般，需要报到前朝议上一议，这也是为什么李承乾可以在书房就随便封赏的原因。
<br/>　　接着，李承乾又将目光看向脸上还带着病容的林晓晓，若有深意的说道：“晓晓为仪人吧，前段时间委屈你了。”
<br/>　　“妾身还好！谢陛下！”林晓晓起身施礼。
<br/>　　她是众女之中最没有根底的一个，有些细节上的事情自然不能不注意。
<br/>　　“好了，平身吧，这几天要注意休息，不要过于劳累。”李承乾脸上露出一丝笑容，伸手虚扶了一下，而后示意林晓晓坐回原位。
<br/>　　最后，李承乾又看向一直没有说话的杨雨馨，这丫头现在大着肚子，行动有些不便，坐在那里像是在想什么心事。
<br/>　　李承乾见她似乎在走神，便咳了一声，招呼道：“雨馨！”
<br/>　　“啊！陛下！”杨雨馨似是惊了一下，立刻起身看向李承乾。
<br/>　　“汝与晓晓一样，也是仪人。”李承乾并没有问杨雨馨在想什么，只是说了关于她的封赏。
<br/>　　“喏！谢陛下！”杨雨馨的地位不如四妃，加上她的哥哥地位也不是很高，所以同林晓晓一般，也是对李承乾施了一礼，只是大着肚子动作有些走样。
<br/>　　“雨馨要好好注意休息，这段时间正是关键时候，不要有太过激烈的动作。”
<br/>　　“是，谢陛下关心。”杨雨馨答应着，坐回了自己的位置。
<br/>　　六个败家娘们儿，该封的都封了，看着她们几个面和心不和的样子，李承乾在心里重重叹了口气，咂咂嘴说道：“这段时间我一直在想一个问题，那就是把你们全都锁在宫里到底对还是不对。”
<br/>　　‘唰’的一下，几个女人同时看向了李承乾，对他突然说出这样的话有些不解。
<br/>　　不过李承乾却并没有任何停顿，只是继续说道：“所以我决定每人给你们派一个差事，根据你们的喜好和擅长的东西来安排，只希望你们以后不要无聊到把所有的心思都用在自家姐妹的斗气上。”
<br/>　　“臣妾等有错。”苏玫见李承乾再次提出她们几个前几天斗气的事情，连忙起身。
<br/>　　“朕说过，那些事情不要提了，你们听朕把话说完。”说话一再被打断，就是泥人也有三分土性，更何况是李承乾。
<br/>　　几个败家娘们儿见李承乾换了自称改用‘朕’也都收敛了许多，全都闭口不言。
<br/>　　有我的时候可以算是家事，但用到朕那便是公事，这个时候如果乱说话，很有可能直接就被丢进冷宫里去了，所以几个老娘们儿此时连个屁都不敢放。
<br/>　　“朕给你们三天的考虑时间，你们可以考虑一下自己最喜欢做的事情，注意，不是让你们想什么刺绣之类的手工，而是让你们负责大方面。”
<br/>　　“比如武贤妃善长经济，那么朕会将一州的经济交给她来打理，孔淑妃善长文学，朕便交一州的教育交给她打理，明白么？”
<br/>　　明白，当然明白！只不过苏玫等人完全没有想过李承乾竟然会做出这样的决定，要知道以前从来都是后宫不得干涉政事的，他的这个决定几乎是给了后宫干政一个合理的理由。
<br/>　　可是，话说回来，几个女人在听到这个消息之后还是很开心的，必竟有事情可以做比天天无所事事要强上不少。
<br/>　　而且大家都有事情做，正好可以互相比比看，看看谁的能力强些，谁的能力又弱一些，这可比在宫里互相算计，今天我给你打小报告，别天我给你碗里放苍蝇强多了。
<br/>　　想到这里，几个败家老娘们儿全都高兴了起来。
<br/>　　至于说李承乾做出这个决定需要在朝堂上与那些大臣打多少口水仗，这已经不是她们能关心的事情了，而且她们几个甚至决定，给家里人通个信，让家里人帮帮忙出点力气，把这件事情做成。
