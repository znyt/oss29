　　退守孤岛的太上皇李二这几天烦躁的很，那些土著就好像死不完一样，一波接着一波的向岛上发起进攻，海滩上的尸体已经堆的跟小山一样高，可他们还是悍不畏死的向岛上不断的进攻。
<br/>　　贞观炮的炮弹早就已经打光了，火箭弹也在两天前打出了最后一发，如果不是还有不少的步枪子弹，估计他们现在已经开始陷入肉搏。
<br/>　　但是现在步枪子弹也已经没有多少了，那些不断向岛上冲来的敌人在接连不断的攻击中已经学会了很多事情，比如如何成散兵线进攻，如何趴在地上躲避射来的子弹。
<br/>　　所以现在虽然每天都要消耗掉大量的子弹，但是能消灭的敌人却没有多少。
<br/>　　另外一方面，那些李二临走是丢在海上的钢铁战舰虽然已经没有了动力，也没有了武装，但还是被那些大食人看中了。
<br/>　　他们虽然无法操作战船，但却不耽误他们把战船拖走，用利抢来的运输舰为工具，在风帆的助力下，他们可以轻轻松松的将钢铁战舰拖出去老远，甚至拖回到大食，而太上皇李二陛下却只能眼睁睁的看着，没有一点办法。
<br/>　　不过就算是这样，李世民却并没有绝望，毕竟秦琼跟尉迟恭还在身边，有这两员虎将在，就算是打到弹尽粮绝，伟大的帝国主义头子李二陛下也不会有丝毫的畏惧。
<br/>　　只是在没有人的时候，老李会咬牙切齿的骂上几句：该死的土猴子云云。
<br/>　　长孙皇后这段时间也憔悴了许多，岛上的生活毕竟不比战舰上安逸，更不要说前面还在打仗，只是看到眼睛里面已经熬出血丝的李二，她不得不强打起精神，装成没事儿人一样。
<br/>　　李世民自然不会看不出老婆憔悴的样子，这一天在打退了土著白天的进攻之后，他第一次早早的回到了长孙皇后的身边，陪着她一起，看夕阳落入大海。
<br/>　　待到夕阳没入海平面，夜幕彻底笼罩一切之后，长孙皇后将整个身体靠入李世民的怀中，脸上露出一丝疲态，喃喃问道：“二哥，你说乾儿现在在干什么？”
<br/>　　李世民顺势将长孙皇后揽入怀中，哼了一声说道：“除了调兵遣将，他还能干什么！自家老子整整三个月没有消息，如果他脑子没有被城门夹过，早就应该派出援兵了。”
<br/>　　“万一，万一他没有呢？”长孙皇后问道。
<br/>　　李世民突然笑了笑，在长孙皇后的胳膊上拍了拍：“不可能，别人不了解那小子，朕还能不了解？如果他老子被人欺负了他都能不管，‘睚眦必报’这个外号只怕就要送人喽。”
<br/>　　长孙皇后听完李世民的话一下子站直了身体，略有些苍白的脸上绽放出一抹笑容：“呵呵……，难怪二哥如此有信心，原来竟然打着这样的主意。”
<br/>　　四下无人，当着自己老婆的面，李世民自然不用再维护皇帝的尊严，啧了一声说道：“打着这样的主意有什么不好，朕的儿子争气嘛，如果朕不是想等着那小子把援兵派过来，好好给那些大食人一个教训，这会儿早就已经回到大唐了。”
<br/>　　长孙皇后白了自家老头子一眼，哼了一声说道：“原来二哥是早有算计！亏得妾身这段时间还担心的要命。”
<br/>　　李世民讪讪一笑：“观音婢，其实你还是太小看这次朕带出来的这批人了，他们的番号可是‘海狼’，怎么可能被这么一点点的敌人压制成这个样子。如果朕愿意，马上就可以组织一次反击，将那些土著和那些大食打散。可是打散他们容易，想要追上去却难了，就算是能追上，我们的弹药已经消耗的差不多了，根本没有可能打一场像样的战争。”
<br/>　　长孙皇后到底是跟着李世民一起天南地北杀出来的，听他说到这里如果再不明白老头子想要干什么，那可真是白白做了几十年的夫妻。
<br/>　　敢情是李世民在宫里待的寂寞了，想要玩儿一把大的，所以才会赌上一切，一不向二儿子求援，二不想杀出重围，只等着李承乾自动自觉得派出援兵。
<br/>　　这样一来只要等援兵一到，李世民立刻就可以组织反击，等到将敌人击溃之后，他就可以顺利的接收这一批人，到那个时候，老李的手下兵精粮足，顺势杀敌人一个回马枪，直接杀到大食国，再过一把征战沙场的瘾头。
<br/>　　李世民见长乐皇后不说话，只是看着他发呆，不由再次讪笑说道：“观音婢，朕前段时间也是在宫里憋的很了，戎马倥偬一辈子，结果到头来却只能在那方圆二、三里转悠，着实让人烦闷的紧了。”
<br/>　　“二哥不用再说了，妾身都明白！”长孙皇后伸手轻轻掩住李世民的嘴：“妾身只是想要让二哥知道，不管二哥您要做什么，无垢都会陪着你！”
<br/>　　夜色下，李世民与长孙皇后对视着，良久才轻伸猿臂，将其再次揽入怀中：“观音婢……”
<br/>　　李世民只刚刚叫出长孙皇后的乳名，远处天海连接处的夜空下，突然闪过一道暗红色的光，接着，第二道，第三道……很快那暗红色的光芒便映红了天边。
<br/>　　那是什么？流星么？不，不会！李世民先是愣了一下，但很快他就反应过来那是什么，旋即发出一阵爽朗的大笑：“来了，果然还是来了！哈哈哈……，朕的儿子派了援兵过来，大食的崽子们，你们等着朕的报复吧！”
<br/>　　长孙皇后先是被李世民的状态吓了一跳，但很快就反应过来他说的是什么，双眼同样眨起一丝光芒：“二哥，你是说，来的是乾儿派出来的援兵？”
<br/>　　“可不就是么，除了我大唐舰队，谁还能在海上打出如此气势！”李世民哈哈大笑着，对远处的黑暗喝道：“叔宝，敬德，速速整军备战，明日一早随朕杀敌！”
<br/>　　“是！臣等遵旨！”不远处的黑暗中，秦琼和尉迟恭略带着兴奋的声音传来，显然这两位杀神这段时候也过的很是压抑。
