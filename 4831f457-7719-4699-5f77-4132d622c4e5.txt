　　“老师，我没有开玩笑，说的是实话，还有老师，你先洗个澡吧，我们一群男的，大半个月不洗澡倒也无所谓，但是女生的话还是要稍微的爱干净点。”

<br/>　　林峰花了五颗晶石，就把一切生活必备品，诸如牙缸牙刷牙膏毛巾刮胡刀剃须膏之类的东西搞定了。
<br/>　　这些东西基本上不怎么值钱，放在超市内也没有人拿，毕竟这年头谁还去顾着刷牙洗脸，能吃饱饭活下去就不错了。
<br/>　　但是林峰有洁癖，他受不了，当然，最主要的是这点晶石压根不算什么，若是林峰也吃不饱饭，那么他也和其余人一样的心态。
<br/>　　把物资带回去后。
<br/>　　林峰敲了敲苏云的门。
<br/>　　“老师，你衣服脱了没。”
<br/>　　“怎么了。”苏云的声音在屋内响起。
<br/>　　说着就门开了。
<br/>　　“给你买了些东西，比如说浴袍毛巾还有拖鞋洗发露沐浴露爽肤水之类的。”林峰把东西递了出去。
<br/>　　“谢谢你了。”苏云愣了一下。
<br/>　　“居然连护发素都有。林峰似乎很贴心啊。”
<br/>　　“嗯，那老师你洗白白哦。”林峰说着就回屋里去了。
<br/>　　苏云关上门很快屋内就传来了稀稀落落的水声。
<br/>　　林峰把一些生活物资给李广他们发放了后自己也躺在屋内。
<br/>　　不知不觉就这样睡着了。

<br/>　　曹龙的赌场内。
