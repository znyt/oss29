　　“楚风眠，北邙宫是我们北邙学院的圣地，决不允许被你带走。”
<br/>　　北邙学院院长看向楚风眠，缓缓的开口道。

<br/>　　楚风眠轻描淡写的开口道。

<br/>　　北邙学院院长目光闪烁，似乎是在考虑这楚风眠话的真假。

<br/>　　剑皇殿的力量，就算是仅仅只有两三成，也不是他们可以抗衡的。

<br/>　　一声爆喝之间。
<br/>　　突然间，这北邙学院的院长，凝聚灵力，突然出手，他大袖一挥，一股灵力便是震爆而出，当即跟这剑芒，轰击在了一起。

<br/>　　这北邙学院的院长，实力果然是要远远超越夜皇等人，他这一出手，直接便是足以跟剑皇殿的力量，不相上下。
<br/>　　最终这两股力量，一同散开，在楚风眠的背后的剑芒，便是消失不见，而那北邙学院的院长，也被逼退了数步。

<br/>　　甚至有可能，是已经踏上了最后一重，第九重，不朽之境的恐怖人物。
<br/>　　北邙学院中，居然隐藏有着如此可怕的强者。

<br/>　　不过这北邙学院的院长，想要对付楚风眠，也不容易，剑皇殿虽然无法击败他，但是保护楚风眠，还是足以的，而且若是真的跟楚风眠撕破脸皮的话，也就意味着在给他们北邙学院惹来一个实力强大的敌人。

<br/>　　现在楚风眠有着北邙宫所守护，就连这北邙学院的院长出手，都只能够是平分秋色，他在想要杀，楚风眠根本是一件不可能的事情。
<br/>　　除非是真的有像是北邙学院院长这样的强者，不顾一切的出手，才有可能轰杀楚风眠，但是那样，也要付出极大的代价。
