　　第588章
<br/>　　小白狼和程大雷突然重逢，这几日一直处于喜悦中。这一点，白马山的山贼都感受到了，毕竟，平日见小白狼和颜悦色的同一个人说话可不容易。
<br/>　　她上下打量着张三斤，眼中藏着困惑。
<br/>　　怕小白狼不明白，张三斤耐心解释道：“如今，各家都在追杀程大雷，我听说有青州城的官兵，黑虎寨的土匪，还有徐州书院的人，若他们知道程大雷藏在这里，白马山非得有灭顶之灾。”
<br/>　　小白狼听罢呵呵笑了两声，倒是笑得张三斤一头雾水。
<br/>　　“首先说清楚，他不是藏在这里，只是在这里养几天病。”小白狼道：“还有，这种事不用你操心，我自有考虑。”
<br/>　　“大当家……”
<br/>　　张三斤还准备说什么，小白狼已没有听下去的兴趣。她开口打断对方的话：“三斤，你跟在我身边，毕竟委屈你了。如果你愿意，我可以给你指一条前程？”
<br/>　　“什么前程？”张三斤奇怪道。
<br/>　　小白狼指了指远处程大雷蹲着的方向：“那位是谁，你心里也知道。多少人想做他的手下，如你以前是不会有机会的。当然，今时不同往日，我说句话还是管用的，你能跟着他，当是你几辈子修来的福分。”
<br/>　　张三斤不得不顺着小白狼手指的方向看去，之间程大雷裹着一件大氅窝在墙角，头发胡子乱糟糟的，整个人从上到下没有半点精神。
<br/>　　关于程大雷的名声，张三斤也不是没有听说过。满天下都在传，传得是越来越邪乎。可这也容易激起人的逆反心思，再加上小白狼一味吹捧程大雷，免不得张三斤心里就会想：他真有那么大本事么？
<br/>　　与小白狼别开之后，张三斤晃悠到程大雷面前，仔仔细细打量他一遍，然后才道：“程当家，身子好些了么？”
<br/>　　程大雷晒太阳晒得蔫蔫的，抬起头打了声哈欠：“还好吧，你是有事？”
<br/>　　张三斤哈哈笑了一阵，蹲在程大雷面前，道：“只听说程大雷一身本事，恨天无把，恨地无环，今天才知道闻名不如见面，见面更胜闻名。”
<br/>　　连番几场大战，程大雷也不会太妄自菲薄，道：“没那么邪乎，也就还过得去吧。”
<br/>　　“那个，能否请程当家指教两招，我以后也好受用。”
<br/>　　啧，原来是拜师学艺的，一般程大雷是不太愿意拦这差事。但对方既然是小白狼的手下，程大雷也不可太保守。
<br/>　　“你是用什么兵器的？”
<br/>　　“使一杆四十七斤铁棍。”
<br/>　　“嗯，锤棍之将不可力敌，看来是有把子力气了，马战强些，还是步战强些？”
<br/>　　“马上步下倒是都有几套把式。”
<br/>　　“嗯嗯，不错。”程大雷伸了个懒腰：“把你的兵器取来，耍一套把式给我看看，我帮你过过眼。”
<br/>　　张三斤愣了愣，这才大概明白程大雷的意思：他是真准备指点自己啊。
<br/>　　他嘿嘿笑了一声：“光说不练假把式，光练不说傻把式，不如干脆咱们过过手？”
<br/>　　程大雷看着他满怀期待的眼神，大概也是明白了他的意思：这小子是不服啊。
<br/>　　他同样哈哈笑了一声，站起来拍了拍张三斤的肩膀，转身准备回房。
<br/>　　“程当家，该不会是不敢了吧？”张三斤语出挑衅，什么天下第一贼，传言估计也只是传言，他手底下并没什么本事。
<br/>　　好死不死，要来拉程大雷的肩膀，张三斤正是要在白马山上将他拆穿。
<br/>　　程大雷回身抓住了他的胳膊，只是一抖肩，张三斤就感觉一股巨力传来，身体好像已不是自己的一般，啪地一声倒在地上。
<br/>　　“你刚才说什么？”程大雷皱起眉头，居高临下的问道。
<br/>　　“程当家，我……”
<br/>　　张三斤脸涨得通红，还准备再说什么时，程大雷已踱着步回了屋里。
<br/>　　他在原地又恼又怕，周围人看着这一幕，确实也搞不清楚怎么会事，就见张三斤已倒在地上。
<br/>　　心中当真是气愤不已，过会儿又被知道这件事的小白狼叫到一边，狠狠训了一番。
<br/>　　男儿争强好胜，本也没有什么。可张三斤的好胜心用在程大雷身上，就委实有些不自量力了。
<br/>　　张三斤不听这话则以，听到这话，心中一股无名火起。自己跟在小白狼身边，勤勤恳恳，就算没有功劳，也有苦劳。
<br/>　　怎么事到临头，她反而帮一个外人说话。
<br/>　　小白狼走后，张三斤这口气咽不下去，待在房间内左思右想，忽然一发狠，不跟你这个女人混了。
<br/>　　现在到处都有人在追杀程大雷，将他交到青州城，交给刘黑虎，交给书院，都能换一份不小的前程。男子汉大丈夫，宁为鸡头，不做凤尾，干脆不跟这个女人混了。
<br/>　　想通这点之后，他连夜下了白马山，也没同任何人说，骑着一匹快马去了。
<br/>　　于此同时，程大雷也正在房间中收拾东西，他倒是想在这里多住几天，可时间实在拖不起。眼下，身子已好得差不多，自己也该做准备离开了。
<br/>　　这个时候，小白狼走进屋里，看到程大雷将头发散开，正一条一条的打成缕。
<br/>　　“准备走了？”小白狼接过程大雷手中的活计，立在他身后帮他梳头。
<br/>　　“嗯呐。”程大雷道了一声：“总是要走的，不能一直误在这里。你怎样，还不打算同我一起走？”
<br/>　　小白狼叹了口气：“我一走了之容易，可山寨的弟兄怎么办，从这里到凉州，要将所有人都带上，路上可是不好走。”
<br/>　　程大雷点点头，小白狼说得也是实情。一路穿州过县，想要赶到凉州蛤蟆城，委实不是一件容易事。
<br/>　　“既然你愿留在这里，那就先等等，日后，说不得我会打到这边来。”
<br/>　　二人说了一阵话，天色一黑，程大雷收拾行李，小白狼又送过来两套干净衣服，让他路上穿。
<br/>　　程大雷收拾好了一些，打算明天一早就走，继续出发向草原。
<br/>　　夜里的白马山静悄悄，张三斤引了一队人马悄悄过来。
<br/>　　“他铁钉在你山上么？”一个黑脸汉子问。
<br/>　　“刘当家，我还能骗你不成，我帮你探路，今天一定能抓了他。”
<br/>(本章完)
