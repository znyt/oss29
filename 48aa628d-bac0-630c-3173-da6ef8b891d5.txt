　　阵法之道，按照等级来划分的话，一共分为十个等阶。
<br/>　　最高的十阶禁制，足以将渡劫帝君困住！
<br/>　　眼前的这道禁制，虽然还远远没有达到十阶禁制的程度，却也差不多已经有六阶的样子。
<br/>　　寻常的化神天君，若是被这种级别的禁制困住，没有个一年半载的时间，根本无法从中逃脱！
<br/>　　若非叶尘前世，连十阶禁制都曾破解过，以他这一世的修为，根本没有任何丝毫的可能，可以走到现在这一步。
<br/>　　时间在一点点的流失，用了将近半个小时，叶尘总算在禁制最薄弱的一角，硬生生打开了一条裂缝。
<br/>　　唰！
<br/>　　叶尘从裂缝中闪身而过，成功破解了第九道禁制。
<br/>　　此时，叶尘已经是满头大汗，破解禁制，需要用强大的神念力量，快速的推算预演各种可能，即便是叶尘如今的神念力量，已经不弱于那些返虚真君，在这么短的时间内，破解如此高难度的禁制阵法，还是让感受到了不小的压力！
<br/>　　“只剩下最后一道了，还有一个小时不到的时间，拼一下！！”
<br/>　　叶尘一咬牙，继续朝最一道禁制冲了过去。
<br/>　　根据他前世进入迷罗星海中，所总结出来的经验，如果他真的能够打通关，之前规则信息中所说的奖励，肯定不会差！
<br/>　　要知道，当初的那件逆天神器月光宝鉴，便是他通过这种方式得来的！
<br/>　　当然了，月光宝鉴是他在第八重星域得到的绝世神器，眼下仅仅只是第二重星域，所谓的奖励，肯定远远比不上月光宝鉴，但也值得一拼。
<br/>　　时间在一点一点的流逝。
<br/>　　这第十道禁制的难度，竟然比叶尘想象的还要更加困难的多！
<br/>　　半个小时过后，叶尘依旧未能找到第十道禁制的漏洞。
<br/>　　“不行！看来只能动用那一招了！”
<br/>　　叶尘双眸微微一凝，似乎是下定了某种决心。
<br/>　　在他前世的海量记忆中，其中有一门，专门用来寻找阵法漏洞的秘术，只不过操作起来，难度极高，而且对神魂力量的要求，也十分的苛刻。
<br/>　　以叶尘如今的精神修为，应该勉强可以施展，但是对神念会有一定的损伤，所以不到万不得已，叶尘还真不想动用这一招。
<br/>　　但是如今都已经走到了这一步，若是就此放弃，他说什么也不甘心。
<br/>　　打定主意之后，叶尘深吸了一口气，缓缓闭上双眼，将状态调整到最佳，同时将所有神念力量，全部汇聚在一处，通过某种秘法，开始暗暗运转起来，
<br/>　　片刻后，
<br/>　　霍！
<br/>　　叶尘猛然张开双眼，同时一声低喝，
<br/>　　“天道之眼！！”
<br/>　　轰！
<br/>　　叶尘的双眸之中，顿时射出两道白光，没入前方的那道禁制之中。
<br/>　　大约过去了近十分钟，
<br/>　　“找到了！在这里！”
<br/>　　在天道之眼的加持之下，叶尘总算在一个极为隐蔽之处，找到了眼前这道禁制的薄弱之处。
<br/>　　嗖嗖嗖！
<br/>　　随着叶尘双手在空中一阵虚点，这道固若金汤的禁制阵法，终于开始出现一点点的漏洞，然后在叶尘的不懈努力之下，开始慢慢变大……
<br/>　　“给我，开！！”
<br/>　　随着叶尘一道低喝，
<br/>　　刺啦！！
<br/>　　这第十道禁制，终于被叶尘硬生生撕开了一条裂缝！！
<br/>　　而此时，离两个时辰的限制，也仅仅只有十来分钟的时间了。
<br/>　　“呼！”
<br/>　　叶尘松了一口长气，直接闪身从裂缝中穿梭而过。
<br/>　　这里是一个古朴而空荡的大殿，而在大殿中央的位置，有一个圆形的柱台，在圆形柱台之上，则放着一个古朴的木盒子。
<br/>　　“莫非，这就是那所谓的通关奖励吗？”
<br/>　　叶尘深吸了一口气，走到那圆形柱台之前，然后将那古朴木盒缓缓打开。
<br/>　　轰隆！
<br/>　　木盒才刚刚打开，伴随着整个宫殿一阵晃动，一道璀璨的白色光芒，自那木盒之中爆射而出！
<br/>　　叶尘微微一惊，定睛一看，这竟是一面镜子。
<br/>　　而与此同时，
<br/>　　叶尘所在的这颗星球，突然一道璀璨的光芒自星球上爆射而出，直冲星斗之外。
<br/>　　远在迷罗星海之外，公孙无我和云苍生等巨擘，看到这一幕，也顿时全都一惊，
<br/>　　“这应该是法宝出世的征兆吧？”
<br/>　　“看这气息，至少也应该是一件仙器级别的法宝！”
<br/>　　“没想到这么快，就已经有人得到了仙器法宝！看来这一次的闯关者，果然比一百年前要强上不少！”
<br/>　　“也不知道是那位天骄，能有如此实力！”
<br/>　　“能在这么短的时间，拿到仙器法宝，恐怕也只有龙族的那个小子，或者就是绝情居的那个小丫头！”
<br/>　　……
<br/>　　众大佬一阵议论纷纷，显然哪个也没有想到，拿到仙器法宝之人，居然会是叶尘。
<br/>　　忽然，
<br/>　　其中一名大佬，冷不防的来了一句，
<br/>　　“你们说，这一次，会有人成功吗？”
<br/>　　此人这话一出，几位修真界的巨擘们，顿时纷纷露出一抹严肃之色，一下子都沉默了。
<br/>　　过了好一会，方才有人幽幽开口道：
<br/>　　“十万年了！除了我们八大仙门之外，恐怕已经没有人知道，这迷罗星海，乃是当年一统了整个修真界，百万年以来，唯一一位渡过天劫的昊天大帝，留下的传承之路！”
<br/>　　“可惜，十万年过去了，无数的天骄进进出出，却始终没人能够得到昊天大帝的传承……”
<br/>　　“若是有人能够得到昊天大帝的传承，说不定就有可能知道，昊天大帝成功渡过天劫的秘密！”
<br/>　　此话一出，几位站在修真界最巅峰的巨头们，全都纷纷露出一抹浓浓的炙热之色。
<br/>　　……
<br/>　　而此时，
<br/>　　叶尘已经将那把古朴的镜子握在手中，将自己的精血滴落在上面，开始将其炼化，
<br/>　　“昊天镜，中品仙器，精神防御类法器，可抵挡八成以上的精神攻击……”
<br/>　　将新的这件仙器法宝炼化之后，搞清楚了它的用途，叶尘顿时大喜，
<br/>　　这昊天镜的功能，虽然很简单，可是却太有用了！
<br/>　　要知道，精神类防御法宝，而且还是仙器级别的精神类防御法宝，可是极为稀有的存在。
<br/>　　譬如以如今叶尘的修为，若是碰上八大仙门宗主一级的顶级强者，对方仅仅只需要一道精神攻击，就足以让他魂飞魄散！
<br/>　　可是如果一旦有了昊天镜，可以抵挡八成以上的精神攻击，叶尘便等于有了一个保命的底牌，不至于会被对方一道念头直接秒杀。
