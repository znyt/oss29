        童筱颖不禁恼火地冲着傅夜沉大吼道：“凶手是何明旭，警察局的人都没有去把他抓起来，你为什么要告诉我这案子已经结了？”
<br/>
<br/>        “警察查了何明旭，他有不在场的证明！”傅夜沉冷冷地回答。
<br/>
<br/>        童筱颖却不以为然：“他买凶杀人！压根就不需要那些什么不在场证明吧！”
<br/>
<br/>        “警方已经对何明旭进行过调查，没有任何证据证明他是买凶杀人。反倒是你，筱颖，你可不可以安分点，不要再去管这件事情！”傅夜沉疾言厉色地回怼。
<br/>
<br/>        马智杰的尸检报告里，他最后的死因，不是拳打脚踢，而是被一根铁棒状的东西所伤。
<br/>
<br/>        那个东西，警方在案发现场已经找到了。
<br/>
<br/>        就是天台大铁门上的那根长及半米的铁门栓。
<br/>
<br/>        而那铁门栓上，只有筱颖一个人的指纹。
<br/>
<br/>        法医解剖了死者，还鉴定出，使用铁门栓给马智杰最后致命的那几击，通过力度角度等等数据上去判断，是一个女人所为。
<br/>
<br/>        而那个女人的身高体重，全都与她童筱颖相符合。
<br/>
<br/>        又加之，马智杰死之前紧紧地抓住她童筱颖的手不放，所以……
<br/>
<br/>        傅夜沉能做的，就是把这件事情压下来。
<br/>
<br/>        很明显，筱颖是被人栽赃陷害的。
<br/>
<br/>        如果事先有人躲在了天台上，特意等着筱颖上去。
<br/>
<br/>        筱颖要想进入天台，双手自然会去握天台大铁门上的门栓，将铁门拉开。
<br/>
<br/>        这样，自然而然地能解释，那把铁门栓上为什么会留下筱颖的指纹。
<br/>
<br/>        而且，他也点名道姓地让警察局里的人去查了那个何明旭，但是什么结果也没查到。
<br/>
<br/>        目前，警方所掌握的证据，全都来自于她童筱颖。
<br/>
<br/>        指纹、凶器、嫌疑人的身高体重性别年龄，全都与筱颖相符，唯独只有杀人动机，警方没有掌握。
<br/>
<br/>        傅夜沉心里知道这些就足够了，他不可能把警局里的事情，一五一十地全告诉给童筱颖听。
<br/>
<br/>        其实，这个案件并未结案，因为他保着她，说她是冤枉的，那么警局那边就必须揪出这件命案的真凶。
<br/>
<br/>        傅夜沉不希望童筱颖被这个案子所缠住，被警察局里的人拘留审来审去。
<br/>
<br/>        他是为了她好，哪怕她误解他，他也不能说。
<br/>
<br/>        童筱颖算是看明白了，这件事情，不管是哪一方，似乎都只是想草草结案，息事宁人。
<br/>
<br/>        她再待在这里，一个人跟大哥傅夜沉争辩下去也没有任何意义。
<br/>
<br/>        “既然，我和大哥聊不到一块，那么我先走了。”童筱颖站起身来，打算只身离开。
<br/>
<br/>        傅夜沉连忙起身，伸手抓住她的臂弯，质问道：“你要去哪儿？”
<br/>
<br/>        “我还能去哪儿？当然是回滨江城啊！”童筱颖冷冷一笑。
<br/>
<br/>        傅夜沉这才放开了童筱颖的手。
<br/>
<br/>        她这段时间，留在临海城，只不过是为了等马智杰那个案子的结果，现在她已经等到了，心里却已经另有一番打算。
<br/>
<br/>        既然，警察局的人没有任何证据去把何明旭抓起来。
<br/>
<br/>        那么，她童筱颖就去想办法找证据。
<br/>
<br/>        马智杰的死，一定跟何明旭有关！
<br/>
<br/>        童筱颖心里的直觉，是这么坚定地告诉自己的。
<br/>
<br/>        在回滨江城之前，童筱颖去买了一套职业装，配上了一副没有度数的眼镜，将头发盘起，然后在打印店里打印了几份合同，又去银行里取了钱，装成是卖保险的，只身去了马智杰的家。
<br/>
<br/>        开门接待她的人，是马智杰的爸爸，而马智杰的妈妈因为伤心过度卧床不起。
<br/>
<br/>        童筱颖被马智杰的父亲招待了在客厅的沙发上坐下，而她在电视柜上，看到了马智杰的黑白照。
<br/>
<br/>        那一瞬间，她差点没忍住，哭出声。
<br/>
<br/>        童筱颖只觉自己的鼻子酸痛难耐，眼睛瞬间就红了，声音哽咽地说道：“马叔叔，请节哀顺变。因为您儿子生前，在我们公司买了保险。他现在意外身亡，我们公司将会对他的家人进行一定数额的补偿。在这之前，我得跟您做个简单的笔录，了解一下关于您儿子的事情。”
<br/>
<br/>        “嗯，有什么事情，你尽管问吧？”马智杰的父亲一边抹着眼泪，一边问道。
<br/>
<br/>        童筱颖深吸了口气，调整了心态，打开了录音笔，接着问道：“马叔叔，你儿子马智杰的案子，警察那边有没有跟你们交底？”
<br/>
<br/>        “警察只说，是我家小杰欠了高利贷，被几个混混给打死了。”
<br/>
<br/>        “你儿子还只是一个大学生，他为什么要去借高利贷？”
<br/>
<br/>        “这种事情，我也很纳闷啊！”马智杰的父亲唉声叹气起来。
