　　第20章
<br/>　　李狗剩离开蛤蟆寨的时候，身后大儿子牵着一匹马，二儿子也牵着一匹马，他脚步踏在地上，觉得晕晕乎乎，一切都是那么梦幻。
<br/>　　“爹，咱们不是在做梦吧，山贼真的让我们牵走两匹马？”李大。
<br/>　　“什么山贼，那是神仙，是神仙可怜我们呐！”李狗剩一声长呼。
<br/>　　李大李二点点头，割一天麦子，四块菜饼就换两匹马，不是神仙，不能解释。
<br/>　　“那神仙……也割麦子？”李三道，年轻人对这个世界还是有怀疑精神的。
<br/>　　“神仙怎么不割麦子，你小毛孩懂什么，神仙不仅割麦子，神仙还顿顿都吃葱油烙饼呢！”
<br/>　　嘶！
<br/>　　三兄弟倒吸一口凉气，果然是神仙呐。
<br/>　　当李狗剩回到李家庄的时候，两匹马在院子里一拴，顿时全村人都恨不得围过来看。
<br/>　　“啧啧，这是三岁口的好马啊，等我家耕地的时候，你可得让我家使使！”
<br/>　　“瞧瞧这大青马，没有二十两银子根本下不来，老李行啊，富不露白，你藏得好深。”
<br/>　　“老李，你们家老大还没成亲吧，我说一直给你撮合一门亲事，你看我，这一忙起来就忘了。”
<br/>　　“啧啧。”李狗剩端着一大碗水，得意洋洋道：“这是山神爷爷可怜我们，我老李一辈子没做过亏心事，这就是福报啊。”
<br/>　　李狗剩架不住虚荣心作祟，便把今天的经历说了一遍，这牛足够他在村里吹半年，不过他一口咬定是神仙因为他善良所赏赐的。这话还真有人信，李狗剩家穷归穷，却是十里八乡有名的厚道，家里三个儿子，也从不欺负谁，卖给别人的柴也从不往里面灌水。
<br/>　　话说李家庄有个财主名叫李壮，家里有上百亩田，富得流油。富虽富，却是为富不仁，铁公鸡一毛不拔。在附近村庄放驴打滚的账，把人逼得拆房子卖女儿的事情也不是没有。
<br/>　　他听到这话，难免就动了心思，李狗剩有三个儿子，我也有啊，神仙让他牵四匹马，他竟然只牵两匹，这不是蠢是什么！如果是自己遇见这种事，肯定要他四五十匹……
<br/>　　啧啧，正是狼行千里吃肉，狗行千里吃屎。
<br/>　　想到此处，李壮连夜将自己三个儿子叫到身边……
<br/>　　……
<br/>　　幸亏今天有李狗剩四人帮忙，蛤蟆寨才能将麦子收完，现在一捆捆麦子整齐的码在校场上，散发出清新的香气。之后还要通过摔打，石磙碾压才能见到真正的麦粒。不过反正现在山寨有马……
<br/>　　提起马，程大雷就忍不住头疼，再这个样子下去，圈里的马就只能饿死了。
<br/>　　林少羽是晚上回来的，而且身边还带了一个人。
<br/>　　“大当家，有人想入伙？”
<br/>　　“谁呀？”程大雷下意识抬起头，看到对方模样时，忽然一楞：“怎么是你？”
<br/>　　这人不是旁个，正是黄三元。
<br/>　　说来黄三元最近的日子，过得可是分外痛苦。那天他九死一生，算是逃回黑石城，却发现韩虎踞已死，薛半川正对韩虎踞生前的亲信进行清理。黄三元也不知道自己算不算亲信，不过估计薛半川不会放过他，便悄悄裹了一些细软，准备逃到落叶城。刚巧不巧，被今天下山打劫的林少羽碰到。熟人见熟人两眼泪汪汪，林少羽便道：既然你没地方去，不如直接在蛤蟆寨落草！
<br/>　　听明白事情经过，程大雷问：“你的意思是，薛半川杀了韩虎踞，他自己坐了城主？”
<br/>　　“肯定是他，可是他在城里宣传是你派人杀的韩虎踞，他还宣扬要杀你为韩虎踞报仇。”
<br/>　　“我屮，蛤蟆寨一共五个人怎么派人去杀他！”
<br/>　　“可没人知道你们只有五个人。”黄三元目光转了一圈：“我也没想到你们真的就这几个人，可薛半川在城里宣传，蛤蟆寨有一千山寨，人人杀人不眨眼，为首的蛤蟆大王更是凶残，每顿饭都要吃人肉，现在他还在城里张贴榜文，悬赏你的人头。”
<br/>　　程大雷眼睛一亮：“赏金多少？”
<br/>　　“蛤蟆大王的人头值五十两，玉面书生徐神机的人头值三十五两，还有一个黑脸大汉……”黄三元看了秦蛮一眼：“你的人头是三十两！”
<br/>　　“魂淡！”薛半川嫁祸程大雷他不生气，可这次却是勃然大怒：“这薛半川竟然如此小看于我，拿区区五十两悬赏我！”
<br/>　　这的确是程大雷不了解这个世界的货币制度，大武建国时发行龙虎币，也就是铜板。一千铜板能兑一两银子，约等于程大雷前世的一千块钱。而这个时代，普通农户一年的生活费也不过三四两银子。
<br/>　　“军师，高飞虎的赏金多少？”
<br/>　　“好像是一百八十两。”徐神机不知道程大雷为何这么问。
<br/>　　“我一定会超过他的。”程大雷坚定道。
<br/>　　看着程大雷意气风发，自信满满的样子，徐神机几个全部无语，你还当这是什么好事啊！
<br/>　　“大当家，黄管家真心投靠，你看不如就让他加入我们山寨。”林少羽道。
<br/>　　程大雷目光落在黄三元身上，脑海中查看他的信息。
<br/>　　姓名：黄三元（小有名气的管家）
<br/>　　年龄：43
<br/>　　技能：无
<br/>　　隐藏属性：无。
<br/>　　黄三元其实也正打量着周围的情景，看众人对程大雷说话的样子，似乎他也没有传言中那么可怕。应该……不会吃人肉吧？
<br/>　　“俺们蛤蟆寨好不好？”
<br/>　　“这个……好。”
<br/>　　“哪里好！”
<br/>　　“呃……”黄三元气得吐血，我客套客套就完了，你这样刨根问底很伤感情的好不好。
<br/>　　“你这种态度，明显是对我们山寨看不起呐。莫看我们山寨人数少，却是人人都有以一敌百的实力。想要加入我们蛤蟆寨可不简单，这样吧，现在你已经有了林少侠做介绍人，自己再写一份五千字的申请书，我看过之后，再考虑要不要收留你。”
<br/>　　徐神机点点头：“阿雷真的很严格。”
<br/>　　“五千字！”黄三元大呼：“大当家你不如直接要了我的命！”
<br/>　　程大雷明显是对这个时代的文字有误解，在这个之乎者也，咬文嚼字的时代，五千字的篇幅有多大，他根本没有任何概念……道德经全文才五千字。
<br/>　　“你这样让我很难办喔，你好好想想，怎么表示自己的诚心，而你又可以为山寨做什么？”程大雷淡然道。
<br/>　　黄三元想了想，忽然道：“大当家，我身上有一份藏宝图，不知这个可否表示我的诚意？”
<br/>
