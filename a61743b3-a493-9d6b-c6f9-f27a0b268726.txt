　　良久良久，李承乾估计时间应该已经是中午了吧，上面老头子终于开口了。
<br/>　　只听一声轻咳过后，老头子在上面说道：“罢了，今日就这样吧，李淳风，回去算一个吉日，准备登基大典！”
<br/>　　“啊？！”李承乾蒙头蒙脑的抬起头，眼中全是迷茫。
<br/>　　什么情况啊这是？雷声大雨点小，这是在玩儿声东击西？又或者是想要一网打尽？
<br/>　　不对啊！这事儿不对劲儿啊！老头子这是要闹哪样？
<br/>　　刚刚不是还要打要杀的么？怎么转眼又准备登基大典，这是要干啥？玩自己一次还不够，还打算再玩儿次？
<br/>　　神经高度紧张之下，李承乾根本反应不过来老头子到底是想要干什么，只是条件反射的觉得似乎今天可以侥幸逃过一劫。
<br/>　　而就在李承乾琢磨老头子是啥意思的时候，已经快要走出大殿的李二突然回头说道：“太子，跟朕来！”
<br/>　　来就来呗，反正已经这样了，估计不是打板子就是流放，反正有老娘在，砍头什么的应该不会，最多也就是圈禁到死吧，李承乾一边想着硬着头皮上前，跟在老头子的后面。
<br/>　　只是经过李泰身边的时候，看到丫那张胖脸显得十分怪异，既有高兴有郁闷，让人看着就想揍上一拳。
<br/>　　可怜的娃，竟然在想好事儿，以为老子不能当皇帝就能轮到自己，真是马不知脸长，难道他就不知道，就算是老子死了，皇位也轮不到他么。
<br/>　　李承乾如是想着，从李泰的身边走过，不过此时他也反应过来了，李泰似乎真的不知道皇位轮不到他，因为这毕竟是很久以后才会发生的事情。
<br/>　　……
<br/>　　李承乾不知道为什么到了这个时候脑子里还会转这些乱七八念头，不过管它呢，反正只要不死就成，相信以自己这条三寸不烂之舌，怎么也能忽悠一段时间吧。
<br/>　　“你在想什么？为何到这个时候还在出神，莫非真以为朕不会把你怎么样？”正走着，前面的老头子突然说道。
<br/>　　“啊？没，没有！”李承乾连忙否认。
<br/>　　“作为太子，马上就要登基的太子，警觉性竟然如此之差，朕很想知道，你的心里到底在想些什和东西！”行至无人处，老头子停下脚步，回头直视李承乾。
<br/>　　“……”李承乾有些无语的看着老头子，舔了舔嘴唇，大脑一片空白。
<br/>　　今天的老头子太反常了，反常到让人无法理解他的行为。
<br/>　　而且重要的是，要杀要剐您给个痛快话不行么？等死的感觉真的很难熬好吧。
<br/>　　“是不是觉得想不通？想不明白？”李二似乎看出了李承乾的凝惑，指了指不远处的一处凉亭示意他过去。
<br/>　　“父皇，您这是……”李承乾是真的想不通，他不明白为什么老头子已经在大殿上布置了先手，结果却还要把他拖到这里来谈心。
<br/>　　而且这大冷天的，找个房间去谈不行么，为什么要在这个破地方谈呢。
<br/>　　要知道，刚刚他可是没少出汗，里面的衣服早就透了，现在被寒风一吹，冷的直打哆嗦。
<br/>　　“刚刚大殿上的一切只不过是朕给你上的最后一课。”李二坐到刚刚被内侍铺上垫子的石凳，看着李承乾语不惊人死休的说道。
<br/>　　李承乾能说什么，敢说什么，他只是觉得自己早晚有一天会被老头子玩死。
<br/>　　还最后一课，这最后一课也太吓人了吧！如果自己的胆子小一些，怕不是现在已经吓死了好么。
<br/>　　李二的目光越过李承乾，看向远处的宣政殿：“朕记得你以前曾经说过，任何事情，不到最后功成的一刻，绝不要掉以轻心，因为在你觉得百分百分成功的时候，往往已经进入了别人的陷井。”
<br/>　　“是的！”李承乾点点头，现在的他隐隐已经明白了老头子的意思。
<br/>　　“所以这一次朕在用一个实际的例子告诉你，想要成为一个皇帝，那就永远不要相信任何人，包括自己的亲人！你可明白？”
<br/>　　“儿臣……不懂！”这一次李承乾没有点头，反而皱紧了眉头说道：“父皇，如果连自己的亲人都不能相信，当不当皇帝还有什么意思？儿臣的意思是说，如果连亲情都没有，岂不是活成了……”
<br/>　　“孤家寡人。”老头子接过的李承乾的话：“所以你现在应该明白，为什么历代皇帝都会称自己为寡人或者孤了吧！”
<br/>　　“不，父皇，儿臣不同意您的观点。”这是李承乾第一次在李二面前如此坚决的说出一个不字，而且说完之后也不等老头子开口便继续说道：“皇位虽然重要，但是如果没有亲情，没有友情，没有一切的话，即便是当了皇帝又如何？这不是儿臣想要的。”
<br/>　　老头子似乎很惊讶李承乾竟然会这样说，所以听完他的话之后沉默了很久才缓缓开口问道：“你是说，为了亲情，你可以不要这个皇位？”
<br/>　　“是的！”李承乾坚定的点了点头。
<br/>　　“为什么？难道皇位不好么？”
<br/>　　“父皇，皇位好，皇帝谁都想当，可是人不能把自己活成一个独夫，请原谅儿臣的用词，可是在儿臣看来，那种没有人分享乐趣，没有人可以敞开心扉的交谈，实在是无法接受，儿臣，儿臣……”李承乾说到最后突然有些词穷。
<br/>　　他刚刚说的这些并不是想在李二的面前表现自己，也不是因为他知道老头子用这样的方式考验过李泰。
<br/>　　他其实真的是不想因为一个皇位把自己弄的孤零零的，每天坐在那个空荡荡的书房里面，无人与其分享成功。
<br/>　　所以他觉得很有必要把自己的想法对老头子说清楚，最后老头子决定把皇位传给他也好，不传给他也罢，一切都交给命运来决定好了。
<br/>　　而此时的李二其实也是心有感触，这么多年皇帝当下来，谁都不能相信的感觉的确让他觉得有些失落，可是李承乾这小子从小就一肚子坏水，他的话真的可以相信么？
<br/>　　想到这里，李二皱了皱眉头，对李承乾问道：“那么，如果朕将皇位传给你，你要如何对待你的几个兄弟？”
