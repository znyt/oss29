        拨开这些衣物后，下方还有一个平板电脑。
<br/>
<br/>        童筱颖拿起平板电脑，刚按亮屏幕，屏幕上就开始自动播放着一男一女在大床上的活动。
<br/>
<br/>        直到这一刻，她才恍然大悟地明白过来，冷夫人所指的做妻子的义务，以及说大哥当和尚是怎么一回事了。
<br/>
<br/>        第一次看这种视频，她没觉得羞涩，反而有种好奇的心理。
<br/>
<br/>        可结果，视频还未看完，她便跑到卫生间里去吐了。
<br/>
<br/>        为什么她会觉得这种画面让她觉得很反胃、很恶心？
<br/>
<br/>        童筱颖站在洗漱台前，给自己洗了把脸，抬头看着镜子里的自己时，脑海里忽然闪过一些莫名其妙的画面。
<br/>
<br/>        好像曾几何时，她把自己脱光了躺床上，结果那个男人却跑去了别的女人那里。
<br/>
<br/>        还有很多乱七八糟的画面，在脑海里像走马灯一样转动着。
<br/>
<br/>        这一瞬间，她的心里有些乱，也有点儿发慌。
<br/>
<br/>        “嘎吱”一声，外边忽然传来开门的声音。
<br/>
<br/>        童筱颖这才反应过来，但为迟已晚。
<br/>
<br/>        傅夜沉已经进来了，而且还看到了桌上的那些衣物，更重要的一点是，平板电脑上的那段视频还在继续播放。
<br/>
<br/>        “你……”他身形一顿，喉结滑动了一下，才接着说道，“怎么在看这个？”
<br/>
<br/>        童筱颖疾步走过去，连忙将那堆暧昧的衣物和平板电脑一起塞回盒子里并盖好，然后冲着傅夜沉咧嘴支吾道：“我、我在学习、学习，呵呵——对，学习。”
<br/>
<br/>        她真不知道自己在胡说八道些什么。
<br/>
<br/>        自从她在实验室里醒过来后，很多东西确实是裴姨临时教会给她的，但是裴姨没教过她这方面的技能。
<br/>
<br/>        她也不知道生孩子是需要这样做的，原本她还以为，生孩子是像在实验室里看到的那样。
<br/>
<br/>        取了男女双方的精子和卵子，经过基因优化筛选，在培养皿中合成受精卵，再移植到母体即可。
<br/>
<br/>        更何况，冷夫人还一直在跟她提找个女人代孕的事情。
<br/>
<br/>        “那你身体在抖什么？脸色也不大好？”傅夜沉微微凝眉。
<br/>
<br/>        童筱颖将目光移到一旁，拿起礼盒藏在了自己的身后，讷讷地回答道：“大概也许是因为……见到你……有些不大好意思吧！”
<br/>
<br/>        傅夜沉眉心舒展，踱步朝童筱颖走近，大手向她伸了过去后，将她背在身后的礼盒拿了过来，语气暧昧透磁地说道：“今晚穿给我看。”
<br/>
<br/>        不容置喙的语气让童筱颖“噌”地一下，抬起头来。
<br/>
<br/>        她有些毛骨悚然地看着傅夜沉，顿时莫名其妙地起了一身的鸡皮疙瘩。
<br/>
<br/>        他来真的啊？
<br/>
<br/>        “那个……等、等我学会了，再穿给你看。”童筱颖又将傅夜沉手中的礼盒给夺了回来。
<br/>
<br/>        傅夜沉意味深长地笑着，伸手将童筱颖揽入怀中。
<br/>
<br/>        他强劲有力的臂膀，透过衬衫的布料，能感受到他那矫健的肌肉。
<br/>
<br/>        男人的荷尔蒙气息，就像巫女施展的魔法，将她浑身的血液烧得沸腾起来。
<br/>
<br/>        “筱颖，我为你禁欲了大半年。”连一次他自己动手的机会也没给过，哪怕是落以柔脱光了去诱惑他，他都不为所动，因为他只为祭奠她，清心寡欲，哪怕一辈子禁欲，他都能做到。
<br/>
<br/>        傅夜沉的声音，突然间好听得不能再好听，带着几分诱哄，温热的呼吸，喷洒在了童筱颖的脸上。
<br/>
<br/>        童筱颖的神情有些复杂地看着傅夜沉的俊脸，此刻心里更是乱成了一团。
<br/>
<br/>        为什么只要是他主动的，她就会有种本能的抗拒心理？
<br/>
<br/>        反过来，她若是自己主动的，便会表现得很自然。
<br/>
<br/>        “我不要！”心里的感受未被大脑的主观所支配，这三个字，童筱颖脱口而出，拒绝得干脆。
<br/>
<br/>        傅夜沉身形微颤，心口又开始一阵隐隐作痛。
<br/>
<br/>        他眸光黯然地凝视着她，片刻后都没有任何下文。
<br/>
<br/>        童筱颖就这样怔愣地盯着他，捉摸不透他此刻的心情。
<br/>
<br/>        过了半会儿，傅夜沉突然放开了童筱颖，语气清冷地说道：“你早点休息，我出去走走。透透气。”
<br/>
<br/>        “好。”童筱颖应了声，抿了抿唇，接着说道，“你早点儿回来。”
<br/>
<br/>        傅夜沉勉强扯出一抹笑意，揉了揉童筱颖的头顶：“晚安。”
<br/>
<br/>        他说完，便出去了。
<br/>
<br/>        房间里，只留下她童筱颖一个人。
<br/>
<br/>        她随后去泡了澡，但他仍旧没有回来。
<br/>
<br/>        童筱颖洗完澡后，坐在床边拿着毛巾擦头发的时候，无意间又看到了那个被她随手放到桌上的礼盒。
<br/>
<br/>        要不……试试看？
<br/>
<br/>        脑海里响起了另一个自己的声音。
<br/>
<br/>        童筱颖鬼使神差地拿起礼盒，再次打开了礼盒的盖子。
